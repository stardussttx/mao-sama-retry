瑪妲穆，前往溫泉旅館

一一一一一一一一一一一一一

（這又是從未見過的建築物呢……）

。

當蘭波慌張地離開村莊，瑪妲穆就在男性兔人的帶領下，來到了溫泉旅館的大門前。這是一座不可思議到難以言喻的建築物。

建築周遭到處都種植著前所未聞的植物，形成一個從未感受過，洋溢著靜謐氣氛的空間。

。

（主題看來應該是「和諧」與「治癒」吧……）

。

瑪妲穆與妹妹不同，對於藝術和美術並沒有多大的興趣。然而她的審美觀也跟妹妹一樣頗為優秀。

優秀到一看見溫泉旅館，便立馬看出建築物的本質。不過她的興趣是主要是自身的「美麗」與「健康」。即便是姐妹，兩人的目標也不盡相同。

。

（總覺得光是眺望著也富有風情，感覺真不錯呢……）

。

遠離忙碌的喧囂。

在鄉下小村裡。

度過不可思議的時間。

。

若傾耳聆聽，

便得以聽聞隨風傳播的鈴聲。

。

（呵呵，真奇怪呢……就好像是回到了小孩子的時期一樣……）

。

這是為什麼呢――？

。

瑪妲穆回想起自己至今從未回想過的孩童時期。當她還處於什麼都不懂的年紀時，她就已經跑到稍遠的地方，展開僅是走在陌生街道上的「大冒險」。

並不只是瑪妲穆，無論是誰都絕對擁有這麼遙遠的回憶。

。

在寧靜的鈴聲回響中，瑪妲穆置身於遙遠的回憶裡。

。

小時候她跟妹妹關系十分不錯，兩人經常在宅邸裡探險，偶爾偷偷溜出來跑到街道上，結果慘遭大人們的嚴厲訓斥。

還有過擅自切開田地裡的西瓜，兩人一起吃掉這種事……。

。

「這音色十分不錯呢……」

。

為何聽見這道聲音就會聯想起久遠的記憶片斷呢？

瑪妲穆即便會感到不可思議也不足為奇。

。

「――這東西在我們國家稱之為「風鈴」」

。

「風丶玲……」

。

魔王彷彿像是回應這道聲音一般現身。

接著，恭敬地伸出手――引導瑪妲穆走向旅館裡面。

。

「重申一次，歡迎來到――「我的世界」」

。

那絕對是――帶有魔性的言辭。或許是夾帶強力的魔法吧，這句話久久殘留於瑪妲穆的耳際不肯離去。

■□■□

「「歡迎光臨，尊貴的來賓！」」

。

兩人堂堂正正地走在身穿男士禮服，並排著的男性兔人們兩側之中。當站在建築物前面的瞬間，透明的玻璃門便自動開啟，瑪妲穆的身軀也不禁因嚇一跳而抖動。

。

即便環顧四周也沒有人影。若往建築物裡面望去，映入眼簾的女性兔人身穿煽情到令人大吃一驚的服裝。

。

當然，穿上兔女郎裝的便是裘恩和沫沫。在田原熱心教育出來的結果，給出了「嘛，短時間應該是沒問題的吧」這種GO暗號。

這「驚人的服裝」就連看著的瑪妲穆也完全面紅耳赤。

。

（表面上靜謐的空間和這種反差到底是怎麼回事……）

。

反差過於甚遠，以至於瑪妲穆不禁深深思考起來。

畢竟在瑪妲穆看來，這位魔王並不會平白無故地就準備這種服裝讓女性穿戴。不久之後，聰明的她得出一個結論。

那就是魔王無數次提到過的「美」和「幸福」這幾個詞語。

。

「也就是說這兩人正是「像徵」和「鉆研」對吧――？」

。

兔人就種族而言，無論是男性和女性多均為美型。

倘若再穿上如此煽情的衣服，其魅力也就更加龐大了吧。其全身丶其一切都可說是女人的武器。

。

正因為如此，她思忖著。

如果來到這裡，變得像這樣的兩人一樣美麗便是像徵。那或許可謂是活生生的例子也說不定。

。

而當所有女性來訪此地，看見她們應該就會奮發圖強，不忘刻苦鉆研。

正因為如此才要特意讓人穿上這種服裝安置在大門入口。

。

「姆――――」

。

面對瑪妲穆的詢問，魔王沉默著一段不算短的時間。

過了不久，猶如放棄一般開口說道。

。

「真令人困擾啊――無論何事似乎都無法瞞得過您」

。

魔王攤開雙手，罕見地做出浮誇的舉止。

見狀，瑪妲穆也莞爾一笑。可說是終於在這男人身上擺了一道。

然而，等待瑪妲穆踏入建築物的卻是驚愕的暴風。

。

（原來如此，所以才說要脫掉鞋子呢……）

。

木製地板磨得猶如鏡子一般丶。

實際上也確實是映照出瑪妲穆的身姿。

而比任何都令人吃驚的事情是――「用紙製成的門」。

。

「這……您還真是想出了荒唐無稽的事情呢……」

。

到底是有怎麼樣的構思才會構想出這種東西――？

紙上描繪著連觸摸都感到誠惶誠恐般精緻的「繪畫」。繪畫既有鶴，也有龜，更甚有華麗的蝴蝶等。

這些繪畫都是以鮮艷的極彩色所描繪。

。

在瑪妲穆看來，這是「一張繪畫」而並非一扇門。而且是散發出異國情調，頗具魅力的名畫。

。

「這種名畫居然用來當做門……您真的是……」

。

「這叫做「隔扇」――嘛，算是我的「興趣」吧」

。

「興丶興趣……」

。

瑪妲穆發出宛如呆然般的聲音。

應該具有萬金價值的名畫為何要用來做門？

對此瑪妲穆摸不著頭腦。在會被他人觸摸的基礎上，絕對會沾上汙垢，甚至還有可能弄破也說不定。想到這裡，瑪妲穆的腦海串過一陣電流。

。

「無論沾上多少汙垢，美麗的東西依然美麗――您是想這麼說吧？」

。

站在瑪妲穆的角度上也是十分同意的事情。

女性追求美麗的「過程」中，無論哪一種都需要令人淚下的努力。在世上的男性看來，全都是不禁後退一步兩步的俗物。

可正因為具備這種俗氣――女人才會熠熠生輝。

這也可說是瑪妲穆的某種信念。

。

「並不是用語言，而是展現出這種東西什麼的……這是你的溫柔嗎？又或者是針對沒注意到的女人的刁難呢？」

。

「姆――――」

。

魔王面對這個詢問，沉默一段絕不短的時間。

不久之後，猶如放棄一般開口說道。

。

「現在――我只能說「兩邊」都有吧」

。

「真是的，真的是麻煩的人呢」

。

瑪妲穆開心似的笑著。

這位乍看之下具備膽量，擁有極不可思議力量的男人，也有如此纖細且若無其事的一面。

在瑪妲穆看來，某種意義上就像是在試探女人的「智慧」，不過反過來也感覺到這個男人深處具有的深度。

。

行走在走廊上，所見之處滿是瓷器丶瓷盤以及用繩子掛起的繪畫（掛軸）等。在瑪妲穆看來，無論哪一個都是尤物。

。

「要是讓妹妹看到，肯定會發狂呢……這樣的美麗空間……」

。

「哈哈，那真是深感光榮啊」

。

當目瞪口呆的瑪妲穆來到溫泉的入口處時，在裡面的露娜就連忙走出來。其表情上滿是笑容。

。

「真是的，瑪妲穆！你還真是慢啦！好了，快點快點！」

。

「哎呀，今天你心情相當不錯呢」

。

露娜原本是不擅長應付擁有巨大影響力的瑪妲穆，不過今天不同。不管怎麼說，可以盡情地向這位可謂是女帝的人物「炫耀」。

。

「那麼，本人就此告辭――請慢慢享受」

。

以魔王這句話為開端，露娜高興地拉著瑪妲穆的手，帶到溫泉裡面。當兩人離開之後，走廊迎來漫長的沉默。

過了一會兒，魔王深深地嘆了口氣，嘀咕道。

。

「雖然適當地讓她們會面了……應該沒問題吧？」

。

魔王叼著香煙，一副工作結束的樣子走向大廳。

■□■□

。

「來，瑪妲穆。我們走吧！」

。

「呵呵，今天小露娜真有精神呢……」

。

脫掉衣服，打開門扇――

裡面展現出瑪妲穆從未見聞的光景。

。

首先光從周遭來看是個寬敞的空間。

裡面有各式各樣形體的浴池，白色的霧氣充斥著整個視野。地板則是漂亮地鋪上一層整齊的瓷磚，其精巧度令人無法想像到是魔法所為。

。

遠處則有像是以平坦的石板製成的道路。令人吃驚的是，道路的縫隙間塞滿了小缸石。

在瑪妲穆看來，目光所及之處宛如是異空間。

。

「這全部都是熱水嗎……真是難以相信的地方呢……」

。

「也有冷水的浴池哦？」

。

不對，我想說的事情不是這件事。盡管瑪妲穆如此想道，可姑且還是噤口不言。

如果不先讓露娜說明和引導，就會有許多不明白的事情。露娜按下某種像是按鈕的東西之後，就勢如破竹地噴出多股細流熱水，這令瑪妲穆大吃一驚。

。

「什丶什麼啊丶這是……到底是什麼魔石的使用方法……！？」

。

「這叫蓮蓬頭。好了，先把身體洗幹凈吧」

。

。

裘恩雙手拿著香皂和毛巾，手法高超地幫她洗背。這是考慮到瑪妲穆的巨體，為了洗背而重復練習。

多虧於此，沫沫的背部完全是變紅了，不過成果明顯呈現出來了。

。

對於瑪妲穆而言，自己的身體過於龐大的緣故，因此也習慣讓人洗背。不過這裡的香皂是特製的物品。遊戲會場裡一旦投擲便消失的消耗品香皂，跟溫泉旅館放置的最高級香皂在「編寫的設定」上是不同的。

。

「總覺得皮膚上堆積的汙垢都連根脫落地洗幹凈了呢……」

。

「是吧～？我要是沒有這個都沒辦法再活下去了」

。

露娜也仔細地洗幹凈自己的身體和臉面。

畢竟洗得越仔細，肌膚也就愈加白凈柔嫩洗，所以也是沒辦法。

最後裘恩把蓮蓬頭往瑪妲穆的全身上沖洗，將泡沫都全數洗凈。瑪妲穆也總覺得整塊肌膚都已經脫離，變成全新的樣子。

。

接下來是沫沫雙手拿著洗發水和護發素站在瑪妲穆身後，邊把頭發洗幹凈，邊進行頭部按摩。這也是以裘恩為對像反復練習而呈現出來的成果

。

多虧於此，連裘恩兔兒的毛都變得亂哄哄的。

沫沫默不作聲地動著手指。

因為田原對她說過「你就別給我說話」。

。

手指的動作彷彿在撫摸頭皮般，時而又像是在慰勞頭發般。

這種纏繞在頭發上的沙塵，堆積在頭部上的汙垢都洗凈的舒適感，令瑪妲穆都情不自禁地發出呻吟聲。

。

「啊浮……哈啊浮浮……」

。

這國家裡的女性受到太陽的日晒，肌膚與頭發都是相當受損。

而這些都似乎開始緩慢地再生起來。實際上，洗發水具備洗凈汙垢的效果，護發素則是有修復損傷的效果。

最後用蓮蓬頭沖洗，在整塊頭皮都清爽幹凈的時候，瑪妲穆不禁露出恍惚的表情。

。

「總覺得我這樣就已經很滿足了呢……」

。。

實際上對於瑪妲穆而言，光是這樣即便要花上幾枚金幣也覺得不錯，不過聽見這句話的露娜則是大聲喊道。

。

「等一下啦，瑪妲穆！你到底在說些什麼呀！　我的炫耀才剛剛……不是，我都還沒開始帶路呢！」

。

「小露娜還是老樣子「為所欲為」，是個「直率」的孩子呢……」

。

瑪妲穆不禁苦笑起來，不過在某種意義上也具有新鮮感。

對於在可謂是虛有其表的社交世界中，經常戴上幾塊假面，絕不展現出真心丶不給與任何言語上的機會，由此度過而來的瑪妲穆而言，像露娜這種存在就像是天然紀念物一樣。

。

「真是的，瑪妲穆！快點走吧！」

。

「好的好的……」
