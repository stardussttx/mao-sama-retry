4-14 第46話 瑪妲穆，吼叫道

一一一一一一一一一一一

「瑪妲穆，首先就是這個哦。這叫做炭酸溫泉來著」

。

「這溫泉感覺有些奇妙呢……」

。

瑪妲穆看著溫泉連綿不斷地冒出水泡，微微蹙起眉頭。然而，露娜倒是毫不猶豫地把全身都浸泡在裡面，浮現出似乎很舒適的表情。實在無法想像這是虛偽的表情。

終於連瑪妲穆也畏畏縮縮地把腳伸進溫泉裡，全身一鼓作氣地泡在溫泉裡面。

。

溫度絕不算熱。真要說的話差不多是溫和的程度，似乎可以長時間浸泡的樣子。

。

「瑪妲穆，你可不能動哦……因為泡沫會逐漸沾在身上」

。

露娜恍惚地閉上眼睛，向一旁的瑪妲穆搭話說。

盡管瑪妲穆暗自思忖著泡沫沾上身體是什麼意思，不過她立馬就明白了。泡沫真的沾在了肌膚上。

。

「這是……」

。

碳酸泉最具代表性的效果是擴展血液和改善血液循環。

具備良好的預防高血圧丶糖尿病丶血栓等癥狀的功能，也可以減緩這些癥狀影響。一旦改善血液循環，對於懼冷丶腰疼丶關節痛等也具有效果，因此可以說十分適合身體肥胖的瑪妲穆。

。

「真好呢……總覺得身體「解放」開來一樣……」

。

正因為瑪妲穆把自己前半生都全花在自己的身體上，所以才能立即感受到這些效果。

她花在自己身上的錢財與時間根本不是小數目。

在關於自己身體方面早已可以說是「神通力」了吧。

。

言歸正傳，她其實也不是因為喜歡這樣才會變胖。

由於家族的遺傳，如今已經逝世的雙親，包括妹妹，其一族體型全部都是猶如小山一般。

。

在她們仍是孩童時期，就為此遭受到周圍貴族的嗤笑丶鄙視。在社交界上歷經各種慘痛的回憶。這些幼小時期的經歷令瑪妲穆走向美麗的道路，而妹妹則是走向藝術的道路。

。

「啊，還有瑪妲穆……這溫泉對肌膚效果也很好喲」

。

露娜如此說著的同時，以可愛的舉止唰啦唰啦地把溫泉往臉上。

露娜所說並非虛言。碳酸溫泉具有作為化妝水滋潤皮膚的效果，以及有效預防皮膚乾燥問題。

聽完，瑪妲穆連忙把溫泉往臉上潑。

。

「這……乾燥的肌膚……」

。

當然，效果不可能立竿見影。僅僅是微乎其微而已吧。

不過，瑪妲穆很清楚。

這溫泉可以改善自己的肌膚這件事。

。

「來，瑪妲穆！差不多去下一個地方了！」

。

「誒……等丶等一下啦，小露娜……」

。

「快點快點！」

。

露娜二話不說地拉起瑪妲穆，帶到下一個溫泉處。

實際上若只限定於瑪妲穆，露娜或許也的確擔負得起向導的職責。如果放任瑪妲穆不管，估計會在碳酸溫泉裡泡個幾小時吧。

而且對於瑪妲穆能夠如此厚顏無恥的也只有露娜。

。

「接下來是香草浴哦。因為會有股香氣」

。

「香草呢……」

。

在那邊有五個即便容納瑪妲穆的身體也綽綽有餘的溫泉。

顏色五顏六色，溫泉表面上看起來蠻可愛，有種令人莫名聯想到「夜晚」的情色。

。

「我果然還是喜歡這個！」

。

露娜跳進寫有「粉色金」的浴池。

其他還有寫著「黃色線」「黑色興奮」「綠色森林」「深藍色」等等的溫泉，瑪妲穆也決定進入自己經常喜歡穿的顏色――綠色的溫泉。

。

綠色森林――這是給人一種森林早晨清爽般印像的溫泉。

。

順便一提，這種香草（？）浴有三十六個種類，不過這些種類都是每天進行變更。而決定權全都是基於隨機抽選，因此無論是誰都無法預料到究竟是什麼溫泉。

。

「這是「森林」吧……有股清爽的香氣呢」

。

令人可恨的事情是這浴池是躺著泡溫泉的設計。

頭部倚靠的地方是採用由金屬製作的類似於枕頭的東西，而枕頭內由於冷水循環的緣故，具有難以血沖上頭的特點。

舒適到若一不小心，有可能就這樣睡著的程度。

。

。

瑪妲穆看見身旁的露娜精神奕奕地說道，不禁也莞爾一笑。

回想起來，這位嬌小的聖女大人也改變許多了――她如此想著。

盡管從以前開始就是天真無邪，不過就好像不想讓任何靠近自己一樣，莫名有種無論對於誰都展露獠牙，威嚇別人的危險感。

。

「是嗎……懂得「戀愛」了呢，小露娜也」

。

「哈丶哈！？到丶到底在說什麼啦！」

。

「你要隱瞞也沒用喲？你喜歡他吧，對於那位「魔王大人」」

。

「別說這種傻話啦！為什麼我要喜歡那種人！」

。

看到那個反應，這次輪到瑪妲穆面露苦笑。

這已經不是戀愛新手的程度，而是態度簡單易懂到旁人一看都覺得害羞。

。

「小露娜，那個人可是強敵哦……？　如果不是相當出色的女性，他甚至都不會多看一眼吧」

。

在瑪妲穆看來，那位魔王怎麼想都不是「普通的存在」。

這設施也是如此。一瞬間就讓那個令人畏懼的惡魔信奉者集團屈服了，甚至傳聞中他是哼著鼻歌炸死中級惡魔。

即便說他是人類以外的「某種什麼」，瑪妲穆也不會驚訝吧。

而即便如此，對於瑪妲穆而言也絲毫不在意。

。

（那位魔王大人從我身上看到了某種巨大的價值……）

。若不是如此，也不會有這麼良好的待遇吧。

既然如此，只要提高自己的價值便可。畢竟利用關系對於瑪妲穆這種大貴族而言是理所當然的事情，不如說正合她意。

沒有什麼事情比利害一致更令人「放心」。

。

「那丶那那那傢伙的事情怎樣都好啦！我們去下一個地方吧！」

。

「等丶等一下啦，小露娜……至少在讓我泡一陣子」

。

「不行！不行就是不行！」

。

面乏紅暈的露娜又拉起瑪妲穆，精神奕奕地喊道「去下一個地方了！」引導起來。

接下來抵達的地方是桑拿浴――而且還是鹽性桑拿。

對於瑪妲穆而言，這設施也成為了命運般的邂逅。

■□■□

。

「這裡呢，叫做鹽性桑拿浴」

。

「鹽……」

。

聖光國的南方由高山所圍繞，並沒有面向海洋的地區。

因此鹽是進口品，絕不是便宜的貨物。

露娜打開門扇之後，相當高溫的熱氣則是噴灑而出，不過房間裡面更是不同尋常。

。

裡面擺置著幾張椅子和長椅子，中央處則是堆放著分量超乎想像的鹽。

令人驚奇的事情是地面上的鹽多到把腳裸都完全包裹的程度。

正所謂是一面白――而且是用鹽形成的白色。

。

「就好像很久以前在都市國家看見過的……「雪」一樣呢……」

。

盡管瑪妲穆回憶起遙遠的事情，不過那是從空中「免費」降落的東西，跟這些「白色」的意義不盡相同。

。

「來來，我來教你，所以坐下來吧」

。

「嗯丶嗯……」

。

露娜自滿地說道，猶如教師一樣的態度。

明明前些天露娜也剛剛受到悠的教學，不過表情卻猶如這方面的老手一般。

。

進行鹽性桑拿浴並不怎麼難。進入裡面之後只需等待流汗，一旦流汗就把鹽往肌膚上灑，再以撫摸的方式塗遍身體。等一會兒再沖洗幹凈。

不過效果對於女性而言全都是極品。

首先，如果塗抹在臉上，皮脂和**會逐漸排出體外，轉變為光滑的皮膚。雖然禁止過度使用，不過卻具備使用磨砂膏而形成的舊角質脫落丶肌膚顏色變亮的效果。

。

「昨天呢，使用完之後效果不錯」

。

「是嗎――」

盡管瑪妲穆的樣子有點變化，不過露娜依然不為所動，繼續進行說明。

真要提起來，臉只是附帶的效果，對於身體的效果果然才是重點所在吧。

通過把鹽塗抹在身體上――便具有把皮下脂肪排出體外的效果。

對於身體脂肪過厚的瑪妲穆而言，便是夢寐以求的東西。因為無論是什麼地方，一旦產生的脂肪都是難以消除。

。

迄今以來，不管瑪妲穆做過多少運動，嘗試過多少古今以來所有的魔法道具都全部徒勞無功。

然而，在重復著塗抹鹽使得身體流汗，把身體沖洗幹凈的過程中，瑪妲穆的表情逐漸發生變化。

。

這種步驟重復的意義所在――便只是加快新陳代謝，劇烈耗損皮下脂肪。

而且是可以針對自己想針對部位的優秀功能。

如果這只是單純的鹽分，單純的桑拿浴，效果應該會是因人而異吧，不可能會產生如此急劇的變化。

然而，就如同滑輪一樣，這也是「遊戲」裡的道具。

它會產生出遊戲設定好的「結果」――就如同一加一等於二一樣，使肌膚恢復美麗，讓皮下脂肪回到健康狀態。這並非不可能的事情。

畢竟這就是這設施裡設定好的內容。

那是猶如身在戰場的武人所發出的凶猛怒吼般，極為可怕的咆哮。她會咆哮也並非毫無道理――因為她得到實感了。

――――否，是聽見了。

她聽見了包裹自身的厚重脂肪發出的「悲鳴聲」。

而且是面對連抵抗都無法做到的敵人所發出的「臨終前的慘叫」。

至今以來從未聽聞過的悲鳴，從剛才開始就猶如要割破耳膜般大聲喊叫著！

。

「怎丶怎麼了！？生氣了嗎！？有丶有什麼覺得不滿意嗎……？」

「小露娜。我，決定了――」

「――從今天開始，我要住在這座村莊裡」

――那一夜　溫泉旅館「鬆之房」

房間裡面鋪著兩張羽毛被褥，露娜已經發出睡聲。

而瑪妲穆也在一旁安靜地仰望天花板。

不過如果習慣了，就有種難以言喻的趣味。或者不只是有股香氣的緣故，很大的原因在於有她喜歡的「顏色」也說不定。
