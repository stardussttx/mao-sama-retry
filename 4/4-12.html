4-12 第44話 瑪妲穆來訪

瑪妲穆來訪

一一一一一一一一一一一一一一一一

一輛豪華的馬車，停留在拉比村的大門處。

這倆馬車上刻有巴塔福萊依的紋章，並升起它的旗幟。原本應該不可能停留在這麼貧寒的村莊裡。

隨從逐個地從馬車上下來，恭敬地打開馬車的門扇。

。

是身為社交界的權威人士，深得貴族夫人熱烈支持，並將其統領而起的女帝。

。

她若說是白，無論何物都將是白，反之也亦是如此。

。

現在聖光國的勢力圈可說是分為非常細致。

。

作為一國應有的狀態，依舊是以三位聖女為中心的形式。

其麾下是聖堂教會以及聖堂騎士團，不過教會是培養聖女機構的同時，也是把具有魔法才能者集中於一處，起到類似於「學校」進行教育的職責，因此並不是政治性質的機構。

。

由於掌控著領地內用於『水』的魔石，因此資金雄厚。

聖堂騎士團的一部分人也是通過揮灑金錢來逐漸壯大勢力。

。

統領著國境附近被稱之為武鬥派的貴族，雖說資金不多，不過卻結成在戰場上可相互託付後背的獨特牽絆。聖堂騎士團大多數人也是他們的擁護者，在武力上可說是首屈一指。

。

有過被稱之為貴族中的貴族一事，根本不會有人會公然地跟他們吵架。因為那就像是在宣傳自己是野蠻人一樣。

。

這一派聚集的人全都是貴族夫人，其自尊心非常強，且權利龐大。甚至就連入贅的女婿都大多是不屑一顧。

那已經不是妻管嚴的程度，而是真的抬不起頭。

實際上，大多數人都是站在一旦遭受夫人討厭就會被趕出家門的立場上。

。

再下面就是許多一般階級的平民，以及貧民。

若還潛入地底，還有惡魔信奉者蠢蠢欲動著，基本就是這樣的狀態。乍看之下會以為這國家是以基於同個信仰而團結一致的國家，可其內部卻可以說是處於激烈分裂的狀態中。

。

「「歡迎光臨此地，瑪妲穆――」」

。

身穿男士禮服的魔王與田原把手置於胸前，優雅地行禮。

兩人都是技藝高超的演員，連瑪妲穆見狀也面露笑容。

她看人眼光極為嚴格，與那豪爽的外表相反，連細節部分都十分拘泥，在她眼中兩人的表現都是滿分。

。

（還是老樣子是個不錯的男人呢……）

。

在瑪妲穆看來，「魔王」是頗具魅力的男人。

而且是唯有一流的惡黨才具有――散發出一種獨特的「危險色氣」。若是在明白這一點的女人眼中，便是令人慾罷不能的男人。

如果接近他，根本不知道有多麼危險――那種戰慄感是語言所難以形容的。

。

同樣的，在魔王旁邊的男人――田原也是非常「危險的男人」。

言行舉止雖說恭敬恭維，可其伶俐的眼神令瑪妲穆心驚膽戰，彷彿連內臟的一舉一動都完全看穿一般。

無論對手是怎樣的存在都能毫不猶豫地殺死，且猶如若無其事一般當場離開。瑪妲穆的腦海里甚至都浮現出這種光景。

。

「這村子感覺真不錯呢……跟聽說的傳聞截然不同」

。

瑪妲穆環顧四周，微微發出驚嘆的聲音。

覆蓋村莊的柵欄都全部換成嶄新的，田地與房屋也佈置得絲毫不損景觀的雅緻。

應該說這不是冷清的村莊，而是「鄉下風味的村莊」吧。

對於已經習慣神都熱鬧街道的瑪妲穆而言，莫名感到新鮮感。

。

「不不，現在還只是剛剛發展而已。還有一個月應該就可以變得有模有樣了吧。對於夫人們而言，這地方會成為「天國」喔」

。

「也是呢――您看起來也不像是會腐敗到只會嘴上說說的男人。您所說的話語，以及提供的物品的確是具有「真實」」

。

聽聞此句，魔王想著「香皂有效了啊」。

實際上汙垢的清潔效果令人吃驚，全身感到神清氣爽。特別是這國家的沙塵較多的緣故，僅僅是在外面待一天全身就沾滿了。

。

「實乃榮幸――不過我可以保證，接下來要參觀的設施必定會讓您更加吃驚，以及為瑪妲穆的人生帶來「幸福」喔」

。

「啊啦啊啦，真是令人十分期待呢――」

。

現場的氣氛與兩人的笑容相反，彌漫著緊張的氛圍。從馬車上又有另外一位年老的男性搖搖晃晃地下車。

。

「謔，這裡就是瑪妲穆之前說過的村莊嗎」

。

「等一下老頭子，我說過在我叫你之前都給我等著吧！」

。

「啊哈哈！老人都是急性子啊……」

。

這是位腰上佩戴著一把氣派的劍的老貴族。

不過眼睛卻藉由藍色布料所包裹，釋放出異樣的氛圍。魔王看見老人的樣子，略微展現出「警戒」的舉動。

。

「瑪妲穆，那邊的貴人是――？」

。

「瑪妲穆還是老樣子說話真毒啊……再給我稍微尊敬一下老人！」

。

「閉嘴，老頭！」

。

關系是好還是差呢――瑪妲穆以輕率的口氣介紹蘭波。

。

聽到這些事情，魔王與田原都面露同情。兩人的視線僅一瞬間交匯在一起――無言地開始做出行動來應對此次突變狀況。

。

「那還真是慘痛的過往呢。不過我的部下中有位技術高超的醫生。如果是這位貴人的傷口――應該可以「順利地治好『』吧」

。

「喂丶喂……剛才的聲音！那是真的嗎！？」

。

蘭波伸出雙手逼近聲音發出的方向。

他的雙手迅速地揪住了田原，恭敬地說道。

。

「客人――若不介意，請先讓我帶您進去裡面」

。

「噢丶噢……拜託你了！真拜託你了！只要有一點點可能性，老朽也無所謂！」

。

「真是添麻煩的老頭子呢……非常抱歉，好像讓你們多費功夫了」

。

「這也沒什麼問題喔。那麼，田原――就拜託你帶路了」

。

魔王浮現出笑容，而田原則帶領瑪妲穆一行人前往野戰醫院。

魔王邊看著他們的背影，邊在通信中交代著某些事情。

■□■□

「這丶是……」

。

「――請往裡面走」

。

瑪妲穆看見野戰醫院之後大吃一驚，田原則恭敬地領導他們往裡面走。

當走進建築物裡面，瑪妲穆首先感覺到的是可謂絕妙的「清爽度」。

。

聖光國是炎熱的國家，瑪妲穆在地位上也可藉由冰之魔石與風之魔石消除炎熱。不過魔石波及到的影響範圍過於。

如果要讓整座建築物猶如冷庫一樣降低溫度，光是這樣消耗都十分巨大。

。

正因為瑪妲穆的體型如此，所以才十分懼怕炎熱。光是應付流汗都不知道費勁多少心機。

然而一呆在這建築物裡面，一瞬間便停止流汗了。

。

「還真是相當涼爽呢……到底使用了多少魔石呢」

。

。

即便瑪妲穆向田原的背影搭話說，也是類似於模稜兩可的回答。

原本這種答復應該會感到不悅才是，不過瑪妲穆並不會這麼想。不如說，她覺得這樣反而更令人覺得那個男人愈加神祕不是嗎。

對於處於像瑪妲穆這種立場上的人物而言，「不知道」的事情基本是已經不存在於這世界上。

。

（這種建築物既沒看見過也沒聽說過呢……）

。

瑪妲穆在參觀的過程中，時而忍住想要大喊的沖動。

瑪妲穆體驗過這世界上全部的珍稀物品丶富貴丶藝術丶華麗的社交――不過在這種「近代化的建築物」裡面行走卻是初次經驗。

。

「這邊就是診察室。請吧――」

。

田原以平時所想像不到的恭敬舉止打開門，引導著兩人。

。

「歡迎光臨，瑪妲穆。還有蘭波大人――」

。

在房間裡面的悠浮現出溫和的笑容，迎接兩人進入房間。

房間裡有各式各樣的器具和藥物，以及簡便的床鋪。還有測量血圧的用具丶體重測量器丶人體模型等等。

。

（簡直就像是來到了不同的世界呢……）

。

瑪妲穆率直地如此想著。

來到這裡之後，心境上都覺得不如干脆笑出來算了吧。她用力拍打蘭波的後背，讓他坐在椅子上。

。

「好了老頭，趕緊讓她看一下吧！」

。

「你這算是什麼女人……男子漢又不可能會逃跑！」

。

「真囉嗦，你這個老年痴呆！」

。

悠微笑地看著兩人的交談，接著伸出手取下遮住眼睛的布塊。

那裡是白色渾濁的眼球，以及腐爛成紅色的傷痕。

。

「抱歉，小姐……讓你看到這麼惡心的東西。這是讓紅斑的蛇（赤鏈蛇）咬到，它的毒液所致」

「原來如此」

。

「現在頂多也就勉強可以看見別人輪廓的程度啊……如果癥狀能稍微變好一點我也就十分感謝」

。

「請放心。治療這種癥狀的「藥液」在我們國家早已完成」

。

悠彷彿像是要令病人安心一般，以強而有力的語氣回應道。

蘭波即便對這番話感到半信半疑，不過聲音還是透露出喜悅之色。因為至今以來無論是什麼醫生看見這種病例，都是無計可施。

連魔法也難以治療紅斑蛇這種強烈的毒液。何況像眼球這麼復雜的器官要進行修復根本是不可能的事情。

。

「那麼，這邊請坐――」

。

悠讓蘭波坐在床鋪上，從藥品櫥櫃裡拿出小瓶的藥水。

這是為眼睛帶來清爽感――普通的眼藥水。

悠打開藥水瓶蓋子的同時，回想起剛才傳達給她的信息。

。

《治療時要做得像是使用藥品一樣。畢竟讓瑪妲穆看見「神之手」刺激性有點過頭了》

。

當悠用上眼藥水的瞬間，蘭波便發出奇妙的聲音。

這也是理所當然。

畢竟這種具有爽快感滲透於疲憊的眼睛裡的效果，根本不存在於這世界上。悠的大拇指觸碰到眼瞳，溫柔地揉動，猶如要讓藥水滲透進裡面一般。一瞬間，其大拇指變化成手術刀――深深地往眼球劃過。

蘭波頓時發出呻吟聲，不過當大拇指離開時，他的視野便迎來急劇的變化。

。

「噢……噢噢……！？　噢噢噢哦哦哦！？」

。

蘭波環顧左右兩邊，抬頭觀望天花板，接著又再次往左右方向扭扭頭。

扭曲丶歪曲丶曾經失去的視野已經回來了。

。

「能丶能看見……能看見了啊！老夫丶老夫的眼睛回復了啊浮浮浮浮浮！」

。

「老頭，你說的……是真的嗎！？」

。

「噢噢，瑪妲穆……這不是又變得更胖了嗎！到底是吃了什麼才會變成這種體型啊浮浮浮！？」

。

「吵死了，老不死的！」

。

瑪妲穆毫不猶豫地往蘭波的頭部全力敲打下去，蘭波也從床鋪上滾落下來。然而，躺在地面上的蘭波依然是止不住笑聲。

他抱著肚子大笑，連呼吸都感到痛苦般翻來覆去的。過了不久，其笑聲便轉變為哭聲。

。

「老夫的丶老夫的眼睛丶看得見了。看得見了啊……」

。

看到他這幅樣子，瑪妲穆也是一副難以言喻的表情。

畢竟蘭波和瑪妲穆之間的往來已經久到可說是孽緣，不過她還是第一次看到他哭泣的樣子。

不久，蘭波擦拭眼淚，姿勢端正地拿出腰上的劍。

。

「小姐，你是我恩人……這把劍就請你收下了吧」

。

蘭波所遞出的劍並沒有多餘的裝飾物，是把注重實用性的武器。常年跟他一同奔赴戰場，可說是靈魂化身。

不過，瑪妲穆見狀倒是猶如呆然般大聲喊道。

。

「你這**！這種東西就算送給年輕漂亮的小姐也不可能會開心的吧！你這樣也算是貴族的一份子嗎！？」

。

「這種東西是什麼意思啊！這把劍可是蘭波家代代相傳的家寶――」

。

「閉嘴！那邊的那位，給我把這老頭丟到馬車上去！」

。

這番話令田原一瞬間猶豫著。

。

「……趕緊給我回到安茨那邊去吧。那個老爺子也會為你感到高興的吧」

。

「對丶對了！覆茨殿下……此時此刻，蘭波要回來了噢噢噢哦哦！」

。

蘭波突然朝著外面跑出去，而田原也緊跟其上。

當兩人離開之後，只剩下診察室裡的瑪妲穆發出深沉的嘆息聲。

。

「對不起了呢，這麼吵鬧――這次賠禮就由我來補償吧」

。

「這樣……可以嗎？」

。

瑪妲穆從懷裡拿出一件物品。

那是一個小小的箱子――裡面放置著一枚拉姆達聖幣。根據場合，價值至少也是可以輕易超過百枚大金幣的貨幣。

就連初次看見聖幣的悠也從中感覺到這就像是某種神祕的尤物。

作為醫療費而言，可說是極為超過。

。

「沒事的啦，多虧了你，還給那兩個頑固的兩人賣了個大恩情。這呢，可是比起這枚拉姆達聖幣更具幾十倍的價值哦」

。

瑪妲穆妖艷地嗤笑道。

實際上，正因為她認為無論砸多少錢都毫不足惜才會拿出這筆錢吧。

因為絕對會有比起這金額更具價值的某種利益

。

「那麼，我就恭敬不如從命――我感覺似乎能夠跟瑪妲穆構建不錯的關系」

。

「啊啦，真巧呢，我也是哦」

。

兩人面露微笑，相互注視著對方。

這兩個女人在某方面具有相似的部分，而且也頗為可怕。
