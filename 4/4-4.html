最初的一步

一一一一一一一一一一一一一一一一

（為什麼，這小鬼會找上我……）

。

昨天零應該是胡說什麼「我有使命」，於是順利逃跑了才對。

恐怕那傢伙是覺得猶如風一樣離去非常帥氣吧。真是無藥可救的笨蛋。

嘛……雖說在裡面演了將近十年的是我。

。

（不不，不對，這不對吧！那是因為在遊戲中才做得出這種事吧！在現實中，在大庭廣眾之下怎麼可能扮演這樣的角色啊！）

。

即便是莫名地在自我糾葛的折磨期間，小鬼頭也一直凝視著這邊。

雖然無法順利說明，不過她散發出一種她自身目睹到事物並不是這邊的臉孔和身軀，而是宛如某種不同的東西一般不可思議的氛圍。

明明我接下來要忙得不可開交……適當地趕走吧。

。

「然後呢，有什麼事嗎――小鬼頭」

。

不小心說出口了，我如此輕率地想著，不過已經為時已晚。

那道仰望著這邊的赤紅眼光變得更加強而有力。

。

「――――果然是跟零一樣的說辭。同樣的顏色」

。

「顏色？？到底在說什麼啊你？」

。

我吸了口煙，慢慢吐出來。

或許是因為具有回復氣力的效果吧，頭腦逐漸冷靜清晰。這傢伙比起我想像中還要麻煩也說不定。

。

「……我可以看見萬物的顏色，人的感情或者靈魂」

。

「你是看相占卜的新人還是什麼嗎？給我去找其他人吧」

。

原本我打算就這樣離開，不過斗篷的尾端被抓住了。

而且是以極為可怕的力氣。

這小鬼，要是光看外表而輕視她的話搞不好會吃大虧。就連周遭飄散的氛圍感覺也跟人類有所不同。

話是這麼說，跟已經目睹幾次的惡魔又好像有點不同……。

。

「把零交出來，現在立刻」

。

「你從剛才開始到底在說些什麼啊？搞錯人了」

。

「交出來，在這裡。現在立刻，交出來」

。

「別給我連續用這種奇妙的說法！要是衛兵過來要怎麼辦啊！」

(颯:出して意思是拿出來，交出來，在某些場合下也可以翻譯成射出來，你懂的)

。

這不就像是把少女帶到衚衕裡面，讓她說些猥瑣的詞句嗎！

明明事業終於要開創了，在這種情況下卻建立起奇妙的評價誰受得了啊。

。

（可是這傢伙……）

。

抓住斗篷的手越來越用力，固執地不想放開。

到底是基於什麼理論呢，她似乎確信我就是零。

雖說如此，這種小鬼無論說什麼都不會有人相信吧。因為無論是外貌丶年齡以及聲音都是截然不同的存在。

。

「你從剛才開始是有什麼目的――？想要金錢嗎」

。

「零……不是，是您救了我」

。

救了你的不是我，當我想這麼說的時候便噤口不言。

我莫名地覺得越是向她解釋，越是會暴露破綻。從外表看感覺就像是無家可歸的少女，實際上也應該差不多吧。

那麼，比起矇混過關還不如誆騙她比較快。

。

「從外表看像是無家可歸的少女……你想說因為無處可去所以要我收養你？」

。

「……嗯，我想見見零」

。

看到她眼眶裡噙著點淚水低著頭的樣子，猶如心如刀絞。

盡管我說得倒是簡單，或許是有點草率了吧……。

雖然不知道這小鬼究竟多大，不過難得零救了她一命，要是餓死在街頭上也不怎麼好……在誆騙她的同時，順便巧妙利用她吧？

。

「是呢，想見零也可以――不過有幾個條件」

。

「條件？我什麼都會做」

。

盡管我想著別隨便說出什麼都會做這句話，不過由於是小孩子應該是沒其他意思吧。實際上即便除掉同情心，這傢伙或許是可以利用的人才。

她的異常力氣與能夠目視「色」這種不可思議的力量。

。

「服從我的命令，老實地為我做事的話讓你見一下也可以」

。

「做事是要做什麼事才好？只要殺死許多人就好了嗎？」

。

「才不是！」

。

無論是這傢伙還是那傢伙到底把我當做是什麼人了啊。

臉可能是長得有些恐怖也說不定，可我早就說過我是善良的人類了吧！

。

「最近無論有多少人手都還是不夠啊……」

。

說到這裡，姑且噤口不言。

拉比村的人數盡管無法明確，不過最多應該也就是300人左右的小村莊。而考慮到其中大部分都要進行農業工作，還是希望多些人手。

一旦來客增加，那就不能像現在這樣採取無防備的態勢，在警備意義上似乎也可利用這傢伙。

。

「如果你的力量以及顏色啥的為我所用的話，收養你可以喔，小鬼頭」

。

「我叫託瓏――才不是小鬼頭」

。

「小鬼就是小鬼吧，走吧」

。

我們向旅館踏出步伐。畢竟在這種衚衕裡長時間跟小鬼在一起有可能會引起奇妙的傳聞。

。

（話又說回來，周圍的女性比例越來越高了啊……我已經要受不了了）

。

下次召喚親信的話，還是召喚習慣照顧小孩子的那傢伙比較好吧。我也差不多想要年齡相近的同性夥伴了。

。

「喂，小鬼頭……差不多也該從斗篷上放開手了吧」

。

「嗯……」

。

小鬼――不對，託瓏老實的放開手。

或許是服從命令這一點起到作用了。然而，放開的手卻伸直握住我的右手。

。

「喂，你在做什麼？」

。。

「因為我不知道這座城鎮的事情。所以走散了會麻煩」

。

「你是笨蛋嗎！都這種歲數了還能這樣走路嗎」

。

我單身抱起小鬼，快步地走向旅館。

這張惡人面孔跟小孩子牽著手走路什麼的可不是鬧著玩的。

我才不是爸爸喔！

。

「跟零一樣的味道……」

。

盡管小鬼似乎嘀咕著什麼，不過我無視她又加快步伐。

我感覺到從周圍而來的視線，不過應該比起牽著手走路要好得多吧。

其中也有吹著口哨，叫喊著昨天真是幫大忙了！的傢伙，不過我盡數無視，終於抵達了旅館。

。

（為什麼只是回個旅館就要這麼辛苦啊……）

。

■□■□

。

「那麼，所有人都做好準備了吧？那我們出發吧行！」

。

回到旅館，我大聲地拍著雙手。

我還必須得盡快回到拉比村，做好許多準備才行。意料之外的變故以及情況未免太多了吧。

。

「長官，那孩子到底是……？」

。

「哇啊！非常漂亮的頭發呢！」

。

悠對於「九內伯鬥」帶人回來應該不會感覺到多麼不可思議。因為實際在設定上親信們都全部是由九內勸誘而來。

雖然其中豈止是勸誘中，根本是強行拉攏過來的……嘛，這件事就不提了。

惡還是老樣子，純粹是在為託瓏的可愛而眼神精神奕奕的。

。

「你丶你啊……那孩子不是混進了「魔」嗎」

。

露娜難得表情僵硬，凝視著託瓏。

混進魔是指什麼事啊……？

就像幻想世界中經常提到的暗精靈這種嗎。

雖然在我看來完全就跟人類沒兩樣，不過正因為是聖女，所以說不定會有某種辨認的能力吧。

。

「她對於今後的拉比村是必要的。睜一隻眼閉一隻眼吧」

。

「聖女對於魔人……別說些不可能的事情啦！」

。

「那座村莊如果發展起來，就可以令上面的兩位姐姐刮目相看喔」

。

「嗚……這……」

。

正如意料之中，露娜對於她姐姐抱有強烈的對抗心。

只要針對這一點，應該可以解決吧。

。

「第一，立於國家定點的存在指的是善惡一律接受的人物。像你這樣無論什麼事都要黑白分明可是一步也前進不了喔」

。

「什麼嘛，說得這麼偉大……」

「――這也就代表我就這麼期待你」

。

「嗚哇……我丶我知道了啦！但是，如果做了什麼壞事我可不會饒恕你哦！」

。

露娜用手指指著託瓏，託瓏也點著頭。

畢竟她已經無家可歸，接下來只要詳細說明細節問題應該就沒事了吧。

而且還有零這個「糖果」。

嘛，雖然那傢伙與其說是糖果，不如說根本不像是甜品……。

。

《悠，全移動可以使用了嗎？》

。

《是的，長官。我有種受到限制的一種機能已經解放的感覺》

。

《這就足夠了。前往拉比村吧》

。

全移動並不是委員會的門面，也是玩家誰都可以使用的指令。只要擁有氣力，即便是在遊戲初期也可以直接使用，代表悠也順利解放機能了吧。

。

（不過問題是誰的機能與指令受到限制，而誰的限制又解放了……）

。

這件事再怎麼考慮，能夠依據的線索也實在過少。

這方面疑問的根源會藉由調查熾天使一事而一清二楚，我如此期望……。

。

「那麼，抓住我。我們要前往拉比村」

。

面對我這番話，惡和露娜歪著頭。可由於已經幾次看見我施展魔法，所以即使浮現出問號也乖乖地照我說去做。

只不過有一個疑問，為什麼連悠也抓住我的手。

。

《悠，你自己就可以抵達吧》

。

《消耗30點氣力就解決的問題非要消耗60點氣力，這是無意義的事情，長官》

。

《嘛……雖然或許是這樣》

。

悠穩當地確保我的手腕，猶如數學老師般以教導的語氣向我說明。

與其他小孩子不同，正因為悠外貌過於端正才令人困擾啊……各種意義上。

。

「那麼，出發吧――《全移動：拉比村》」

。

視野一瞬間切換過來，接著就已經抵達令人懷唸的拉比村。

所有人似乎都平安無事的樣子。

盡管露娜和惡在為一瞬間的轉移而吵嚷著，不過我的注意力已經漂向村裡拼命幹活的兔人那邊。

。

兔人們晃動著兔兒，在農田裡拼命幹活中。

他們滿身是汗，衣服也沾滿泥巴。

。

明明如此，為何莫名會覺得那副模樣顯得美麗呢――

。

或許是因為貧窮的緣故吧，連小孩子的兔人都來到田地裡幹活。明明是想要玩樂的年紀，可從情況來看並沒有這種從容。

即便水已經用不盡，問題也並不會就此解決吧。

村民的減少，荒廢的土地，為數不多的雨。

以及聖光國對於亞人的強烈偏見，盡管面對兔人的偏見比較少一點。

。

「悠，雖說有些快，還是立刻設置吧」

。

「是的，長官。接下來請教給我來辦」

。

應該是注意到這邊了吧，兔人們揮著手跑過來這邊。由於滑輪以及託付肥料這兩件事，對於我們產生信賴關繫了吧。

然而――他們的表情上滿是疲憊之色，就連穿著的衣服也是破爛不堪。

。

（給我看好了……無論是什麼我都會轉變給你們看，讓這座村子降下前所未有的黃金量）
