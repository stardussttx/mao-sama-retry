4-10 第42話 逐漸改變的村莊

逐漸改變的村莊

一一一一一一一一一一一一一一一一

――幾天後　拉比村　「澡堂」

。

「啊浮～心情愉快～」

。

「肌膚滑溜溜的～」

。

兔人們做完早晨的農地工作之後，便在澡堂裡泡澡。

考慮到前些日子連給田地澆水都是問題，現在已經是非同一般的改變。畢竟只要按一按就可以流出無限的冷水和熱水。

。

每一塊美麗得出奇般的鏡子，分別豎立在擺放著許多椅子的面前，在其高處的物體（噴頭）中，勢如破竹地噴灑出熱水。當頭部沐浴其中時，體驗到的舒適感是無與倫比。

。

在此之上，連只有貴族才會擁有的香皂都擺放得宛如山一般，而且洗發水以及護發素也猶如理所當然般放置於此處。

雖說兔人們並沒有理解到香皂丶洗發水以及護發素的作用，而沒有去使用這些產品，不過遲早會使用起來吧。

。

「嗚薩薩！要在冷泉池消毒！」

（颯:之前翻譯的冷水浴全部改為冷泉浴，字典欺騙我，特意查了一下，似乎是20度左右的溫泉）

。

一群小孩子精神奕奕地在冷泉池玩耍，這也是經常在澡堂裡看到的光景。

澡堂這邊是二十四小時免費為兔人們開放，所以無論是在白天或是夜晚都會有許多人過來。

。

順便一提，地板和椅子雖然也需要清潔，不過熱水和冷水都是自動循環，所以現在是時常性保持著幹凈的結構。

澡堂的衛生打掃範圍雖說很少，不過溫泉旅館那邊就不是這樣了。

因此田原今天也熱心地在指導兔人們的打掃方法。

。

「你們聽好了？溫泉的區域要用這些刷子和刷帚擦乾凈。清潔劑也別省著用哦？」

。

田原展現自己手上的樣品，擦拭著地板與墻壁。

既然有瓷磚，那麼在一些地方也會有碎石鉆進縫隙間。而每一處洗法跟使用的刷子也各有不同。

何況在溫泉中是赤腳走動，汙垢也比較容易堆積。田原教他們清掃各個區域，而兔人們對此也坦率服從。

。

建造出這種無與倫比的設施，而且每天還會有面包以及蔬菜送到這裡。兔人們想著要做點什麼回報這份恩情，因而以認真的態度對待工作。

。

當清掃的指導告一段落，無暇休息的田原便走到大廳。

在那裡有穿著兔女郎裝的瓊恩和沫沫。

（颯:誒？瓊恩原來是女生？）

。

「兔女郎和溫泉旅館嗎……起初我還以為會怎麼樣，不過意外能行呢。畢竟是真正的兔人啊」

。

「這丶這件衣服……好令人害羞！……piun」

。

「你們是變態wusa」

。

兩人邊遮掩著胸口，邊向田原投以包含憎恨的眼光，不過他的表情卻幾乎沒有變化。

因為對於他而言，除了妹妹以外的女人「都不算是女人」。

。

「這種東西就給我習慣啦，習慣。不然就來點殺必死，搖搖屁股吧」

。

「絕對不要piun！」

。

「wuza！……不對，wusa！」

。

「比起這種事，還是復習一下迎接客人的姿勢。要深深地鞠躬，低頭三秒。別忘了還要面帶微笑，以及吐字清晰大聲」

。

田原努力地做著工作，而關鍵的魔王――

。

不知為何裸著上半身呆在醫院的診察室裡，接受來自悠的安檢。

魔王坐在椅子上，以宛如鋼鐵所制的後背面向悠的方向。而悠則把聽診器放在那「鋼鐵」之上，面露恍惚神情。

。

「雖然我覺得我身體是健康的……」

。

「不，考慮到有什麼萬一，還是應該定期進行檢查吧。因為長官要是發生了什麼事情就糟糕了」

。

「嘛，說的也是……」

。

背向悠的魔王並不知道悠浮現出怎樣的表情。

畢竟光聽表面的言辭，悠所說的事情是十分正當的事情。悠拿開聽診器，用那猶如白瓷般的手指直接觸碰後背。

。

她的呼吸越來越激烈，其臉上也泛起紅潮。而接受診斷的魔王臉色卻越來越蒼白。

紅與白的反差。

。

「差丶差不多可以了吧……？身體不好的時候再拜託你吧」

。

「不行，長官。身體檢查還沒有結束」

。

悠的手指從後背摸到胸膛，她的胸部也緊貼在後背上。事到如今，這已經不是身體檢查也不是什麼診斷，而是逐漸在變化成某種東西。

。

「非常健壯的胸膛……即堅硬，又雄偉……」

。

「我丶我接下來還有工作，是時候走了――下次有機會就拜託你了」

。

「啊浮，長官……」

。

盡管悠對他投以依依不捨的目光，不過魔王拿起衣服站起來，慌慌張張地走出診察室。

雖然他額頭上流著冷汗，不過該說給我爆炸吧，還是應該同情他呢，真是不知如何選擇的光景。

■□■□

。

「卡其福萊依那隻女狐貍！居然令我臉上抹黑！」

。

。

「像那樣的物品，只有我這種真正的貴族所擁有才是與之相應！居然被那隻滿身脂肪的肉塊……什麼國家最厲害的收集家啊！」

。

。

因為這些物品偶爾會發揮出遠比金錢還要更強的向心力。

因此，具備實力的貴族都會盡量收集美術品以及藝術品，並將其展現出來誇耀。當然，其中甚至也包括稀有的裝備。

。

「居然是四十二枚大金幣……少丶少丶少開玩笑了啊浮浮浮浮浮浮！」

。

終於憤怒到忍無可忍了嗎，多納雙手都砸在餐桌上。沖擊使得餐桌上幾盤料理都掉落在地面上，不過他瞅也不瞅一眼。

。

近年來的競拍形式都是像以前一樣宣告價位，通過互相爭奪的方式進行。這種情況下，本人的競拍價大多數會超出計劃之中的價格。所以在「相互之間都在理性範圍內收場幹」這種暗地裡的默認下，進行了規矩的改變。

。

那就是每個人都觀察物品――把寫有價位的紙條放進投票箱之中。

。

如此一來，就不會支付超出自己假定以上的金額。按照以前的形式，趁著興頭競拍下來還好，不過一旦冷靜下來就會發現金額十分可怕，之後就是引發取消交易的騷動，這種事頻繁發生。

。

若採取這種新形式進行競拍，就不會發生這種糾紛――

當初是這樣預想的。

。

然而這種形式也會因本身性質發生激烈的「心理戰」。畢竟無論在何時――機會僅此一次。如果無論如何都想要得到，那就需要龐大的金額。

。

然而根本不知道周圍人究竟對此定下多少價值。他們會覺得明明周圍的人並沒有定下龐大的價值，只有自己定下龐大的價值，虧損會相當巨大。

。

「老爺提出的金額，我覺得沒有任何問題――」

。

在多納身後，有位站姿端正的男人。他觀望著情形對多納搭話說。

這男人是多納在北方所僱傭，名叫「亞茲魯」（Azure）。

。

他在那一帶是作為實力高強的暗殺者而知名。其容貌非常端正，端正到不像是男人。膚色也猶如陶器般潔白。

雖說多納是作為暗殺者而僱傭的他，不過由於他相當聰慧，外表也不錯，因此平時都是讓他做著管家的工作。

。

「比起老爺的四十枚金幣還要高……」

。

當初多納對於八音盒是定下三十八枚大金幣，是個荒唐無稽的價格。

即是十分充足的數字，也是猶如擊潰螞蟻般擊敗他人的金額。

不過，考慮到卡其福萊依的存在。多納還慎重地再加上兩枚大金幣，定下總共四十枚大金幣這種鐵壁般的數字。

若是無視文化與價值的區別，那筆金額就相當於八千萬日元。

。

然而，作為勝者而呼喚的卻是卡其福萊依的名字――

。

光臨競拍會場上的大量貴族們，先是為她大聲宣告的四十二枚大金幣這筆金額感到驚愕，接著便因卡其福萊依的膽量而掌聲四起。

。

「居然是四十二枚大金幣……瑪妲穆一直以來都讓吾等感到震驚啊！」

「讓那位婦人提出這種程度的價格……這是多麼幸福的魔法道具呀」

「瑪妲穆才是貴族中的貴族！」

「為瑪妲穆卡其福萊依的勝利乾杯！」

。

那陣掌聲丶那份畏敬丶以及一瞬間便席捲全場的尊敬的龍卷風――

那就是丶那才正是作為貴族的「力量」。從多納多納的角度來看，就像是在眾人眼中完全丟失臉面的狀況。

。

在那種場合的「心理戰」中敗北，作為一位貴族應該是致命性的吧。

。

「可惡……可惡！亞茲魯！就不能搶回來嗎！」

。

「那件物品已經被眾人所知，是卡其福萊依大人的所有物。雖然是可以奪走，不過是下下策」

。

「那你說要怎麼辦！要我就這樣忍氣吞聲嗎！」

。

「不如跟那位帶來八音盒的人物商量一下？」

。

就亞茲魯個人而言，他是打算避免主人發生奇妙的暴走，委婉地進行誘導。雖然他沒有一絲忠誠心，不過主人一旦沒落，自己衣食住行也會消失不見，所以也是理所當然的行動。

。

「哼，如果做得到也用不著這麼辛苦……那個男人不知道什麼時候變成了「第三位」的監護人立場」

。

「這件事……」

。

亞茲魯當然也聽聞過那個男人的傳聞，只是沒想到居然是帶來八音盒的人物。他在城裡聽說的傳聞中，那個男人是被稱之為「魔王」的可怕人物，跟「親切」到會帶來奏響那種不可思議音色的物品，印像可是相差甚遠。

。

「連「吾之妻」的霍瓦伊特也說嚴禁跟那個男人接觸。雖然我只是點頭同意那傢伙是個***」

。

「……是這樣啊」

。

厚臉無恥地胡說吾之妻的樣子，一瞬間令亞茲魯蹙起眉頭。

雖然他不是這國家出身，不過那位高潔的女性會成為這種男人的妻子之類的，他連想都不願意想。

。

「嘛，也好……我遲早會把那個男人擁有的物品都全部奪走。到時候，卡其福萊依的表情究竟會扭曲成什麼樣呢，真令人期待不是嗎」

。

「是的――」

。

總而言之，避免主人暴走這件事令亞茲魯放下心頭大石，不過同時，他對於那位被稱之為魔王的人物又更加警戒。

他覺得，那人會不會因敘述者而猶如千變萬化般發生變化呢。

有人說他是大富豪，也有人說他是魔王，還有人說他是來自於海洋另一邊的貴人等等。

。

（從主人的樣子來看，遲早會變成敵對狀態吧――）盡管多納的宅邸裡進行著這些交談，不過拉比村卻沒有這些。
