各自的夜晚

一一一一一一一一一一一一一一一一

「柵欄得改變一下吧。對面的區域也進行擴大……」

。

夜晚，田原單手拿著筆記本在村裡巡視。

盡管周遭幾乎沒有一絲明亮，不過由於這男人是超一流的狙擊手，因此夜視能力極不尋常。

他幾乎是以與白天時毫無區別的言行舉止，在筆記本上寫著什麼。

。

「不如干脆把那邊的山「削掉」吧？」

。

雖然偶爾會說些荒唐無稽的事情，不過如果是他應該做得到吧。

田原的能力是全方位進行攻擊，可攻擊一旦集中於一處，豈止是戰車，就連山都甚至絕對可以摧毀掉。

。

真是相當熱衷工作呢――

。

田原回頭一看，站在那邊的是面露妖艷笑容的悠。

她在月夜的照耀下，顯得美不勝收。然而，如果是遊戲的玩家在這種黑暗之中遇見到她，應該會嚇一跳吧。

會說，至少讓我死在明亮點的地方――

。

「嘛，畢竟長官殿下這麼期待我啊……」

。

田原一副愁眉苦臉的表情回答道，樣子上像是無法接受這種事情。

對，他是最喜歡懶惰以及自甘墮落的男人。縱使搞錯了，他也不是會主動行動起來，主動工作的人。

。

「你這傢伙才是，最近相當老實不是嗎？　如果是平時的你，這附近的「人口」即便已經減少也不奇怪吧」

。

「這說法真過分呢。我一直都是尊重長官的意志而進行行動的哦？」

。

對，就是這點――田原如此想著。

迄今以來，距離那份「意志」最近的便是悠。她與「靜」不同，即便發狂也具有判斷能力。

。

像無意義的虐殺丶會簡單暴露蛛絲馬跡的罪行丶以及見人就徹底殺死等武斷行為，她都不會去做。

。

深深地，靜靜地潛伏著――

當回過神來，地面上已經變成「無法挽回的地獄光景」，她就是這種類型。當她出來表面時已經「為時已晚」，成為一位極為棘手的「殺人狂魔「。

。

「然後呢，長官殿下到底在想些什麼？正因為是你這傢伙，應該會帶來幾個像是答案的回答吧？」

。

不惜做好所有的提前準備，時而砸下鉅款，謀劃任何僅有的惡謀，最終必定會成就已身的「目的」。

即便身上沾有多少鮮血丶哀嘆以及惡名。

基於這種意義上，在田原看來九內伯鬥與悠極為相似。

。

不過田原對此其實也並沒有抱有惡意，也沒有厭惡感。因為如果不這樣做，根本無法在那個世界上生存下來。

如果不陷害對手，或者殺死對手，那麼自己應該會完蛋吧。

田原也是從這種「人生」中走過來的。

。

「老實說，連我也沒能完全掌握――」

。

「嘿~……這點倒是意料之外啊」

。

田原邊往香煙點上火，邊若無其事地窺視悠的表情。

沒在說謊――田原如此認為。

其實也並沒有什麼根據，只是田原有種獨特的直覺，他對此深信不疑而已。

。

「雖然長官殿下說過想要走上與大帝國相反的道路來著」

。

「是呢，他還說過原先的很「無聊」」

。

「悠，你這傢伙覺得這樣就可以了嗎？」

。

田原思忖著――這種狀況對於悠而言，才是真正的「無聊」不是嗎。

如果是以前的悠，在這種滿是「實驗動物」的世界裡應該是盡情的蹂躪，使其屈服的道路才是比較接近她的期望不是嗎。

。

「――田原，我呢，接觸到「世界的斷層」了」

。

「你又給我說些莫名其妙的事情……」

。

田原邊胡亂地繞繞頭發，邊吐了口煙。

悠偶爾會有些難以理解，會說些離奇古怪的事情。這也是相隔一線的科學家之中常見的莫名其妙的表達措辭或者哲學之類的。

。

「而且，我在長官裡面感覺到「神的存在」――」

。

「啊哈哈哈！那是什麼鬼…又是那個嗎，類似於你這傢伙的口癖一樣，人體的神祕丶進化之類東西嗎？」

。

「不，不是哦。隨著時間的經過，這想法也越來越強」

。

――你也微微感覺到不是嗎？

。

悠留下這句話之後，一聲不響地往醫院的方向走過去。

那身白衣在面前幾步都甚至猶如墨水遍佈的黑暗之中搖晃著。田原呆然地看著這幅光景，不久後便在隨身攜帶的煙灰缸上把煙熄滅，又開始再次在村裡巡視。

。

《在長官裡面感覺到「神的存在」――》

。

這句話莫名殘留在腦海中，他想著這件事。

另一邊，明明是魔王卻被稱之為神的男人――

■□■□

「嘛，也不可能連頻道都有吧……」

。

堂堂正正地佔據溫泉旅館最高級的房間，邊睡在羽毛被褥上，邊翻弄著遙控器。

光看他這副樣子，就像是在出差地點隨意放鬆的上班族。

即便搞錯了也不像是會被他人稱之為神的樣子。

。

「房間裡面也沒有饅頭……果然與食物有關系的東西都不行嗎」

。

按照原本的設定，食堂裡面當然不用說，連房間裡也應該有各式各樣的日式點心。不過無論在哪裡都沒有看到。

據點裡面的「乾麵包」應該是例外吧。

。

「但消耗品倒是會自動補充，也不全是壞事嗎」

。

在那之後，魔王進行各種各樣的實驗。肥皂以及洗發水跟熱水一樣都會自動補充，已經確定是循環的構造。如此一來應該就可以無需介意成本，持續進行運營了吧。如果每次使用都要逐一進行製作，無論擁有多少SP都肯定不夠。

。

「魔王大人丶溫泉真的好厲害啊！熱水啊丶熱氣，還有水」

。

「冷靜一下，惡」

。

隔扇唦啦地一聲被打開，身穿黃色浴衣的惡沖進被褥之中。

彷彿一起睡覺就像是理所當然般一樣。

。

「今天進去泡了一下香草浴！」

。

「嗯……的確有股香草的香氣啊」

。

「真的嗎？誒嘿嘿」

。

「又要一起睡覺嗎……」

。

盡管魔王發著牢騷，不過卻沒有趕走她。

不管怎麼說，他比較溺愛。

魔王用遙控器關掉燈光，正當他想睡覺的時候，就聽到走廊傳來急忙地腳步聲。

。

「惡！今天我早就說過你要當我的抱枕了吧！」

。

露娜也穿著代表自己特色的粉色浴衣。

恐怕是悠讓她穿的吧。

。

「惡在我旁邊睡覺哦」

。

「變丶變變態！你打算對惡做什麼啊！」

。

「雖然我不知道你到底想像了什麼，不過連你的腦袋也是粉色啊」

。

或許是因為泡溫泉的緣故吧，原本就容易入眠的惡已經發出睡聲。

見狀，就連露娜也陷入沉默。

。

「真丶真沒辦法呢……那麼我今天也在這裡睡」

。

「不，你給我回到自己的房間吧」

。

「吵死了啦……我可是這村裡的領主」

。

露娜如此說道，繞到惡的反對側鉆進被褥之中。

幼女與貧乳女的三明治夾擊。

然而，魔王的表情上只是浮現出「困擾」神色。

。

「一般都會是感到困擾吧……」

。

他終於把這句話說出口。

不過露娜就是露娜，依舊是毫無顧慮的態度。

。

「我可是領主……是這裡最偉大的人呢……」

。

最高級羽毛被褥的舒適感令露娜不禁恍惚地閉上眼睛。

即便是聖女也應該沒有體驗過柔軟到這種程度的床鋪（？）吧。這也是因為這是大帝國製作的最高品質。

在舒適度以及柔軟度方面上，可謂是無論如何都要追求到病態的品質。

。

「什麼呀，這柔軟度……真不敢相信……」

。

露娜像是在享受這份柔軟度一般，無數次翻身。期間她的屁股碰到魔王的右手，立馬就發出悲鳴聲。

。

「你丶你啊！剛才摸到我的屁股了吧！」

。

「你到底要給我套上多少冤罪啊」

。

「就丶言就算你要狡辯也是沒用的哦……！這只惡劣的左手就交給我來保管！」

。

「喂喂……」

。

露娜抓住魔王的左手，把它當做是枕頭。

她打從一開始就想這麼做嗎，還是其他原因呢――

。

「還丶還真是挺堅固的嘛……血丶血管也微微抖動著……」

。

「你別老是給我說些莫名其妙的話！」

。
