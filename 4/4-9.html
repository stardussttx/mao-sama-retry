4-9 第41話 田原勇

田原　勇

一一一一一一一一一一一一一一一一

「田原喲，在吾面前顯現吧――！」

。

白與黑的巨大光芒隨著這句話出現於眼前，當光芒重合時――

便形成一個男人的形狀。

他擁有一張睡眼惺忪般的臉孔，身材高挑，體格健壯。

。

「嗯……？這不是長官殿下嗎。是說，這裡是哪裡啊！」

。

（啊浮，是田原啊……）

。

當看到自己所創造的角色行動著丶說話的樣子，便置身於一種難以言喻的感動之中。

不過我沒有放鬆警惕的機會。因為這傢伙跟悠一樣，一旦知道我不是九內伯鬥根本不知道會採取什麼行動。

。

「終於來了啊。這裡不適合談話，我們找個可以冷靜聊天的地方吧」

。

「哦丶哦……」

。

田原東張西望，撓撓頭發。

嘛，也不是不能理解。畢竟是眼前的溫泉旅館在貧寒的村莊中，這種摸不著頭腦的情景。

。

「這裡是新的「會場」啊――還是老樣子真費錢呢」

。

「包括這件事在內，我想慢慢地進行說明」

。

我們來到溫泉旅館的大廳，坐在這裡預備的沙發上。

這裡有煙灰缸，應該可以冷靜地談話吧。從大廳設置的揚聲器發出輕快的琴聲，實在是令人安心的空間。

。

「首先我想要說的是——這裡並不是大帝國存在著的世界」

。

「哈啊？？」

。

我重新敘述以前向悠說明的事情。

包括把接下來知道的事情都告訴他之後，田原的表情猶如百變臉一般發生變化，像是呆然似的嘆了口氣。

嘛，這種態度也是正常……一下子就接受事實反而比較可怕。

。

「也就是說那什麼，這裡是小時候觀看的動漫裡出現的奇幻世界之類的？？」

。

「直截了當的說，就是這樣」

。

面對我的回答，田原持續一段相當長的沉默。

他像是困得要死似的看看天花板，撓撓一下臉頰。從一旁看來就只是傻瓜一樣的舉止，不過我卻很清楚。

創造這傢伙的我――很明白。現在這傢伙恐怕在全速運作大腦，得出各種推測與結論才對。

。

「嘛，再怎麼樣也不可能連你也會開玩笑吧……」

。

「順便一提，悠也在喔」

。

「額！那個瘋狂女嗎！？別開玩笑了啦！」

。

「我想讓你和悠共同處理某些事情」

。

我故作鎮定地點火抽煙的同時，也忍不住要笑出來。

這傢伙對待悠的反應跟我很相似。

雖然她很美麗，不過卻很可怕……我到底是為了什麼才要搞那種設定啊。

。

「長官殿下啊，有件事想確認一下可以吧？」

。

「什麼事」

。

「這裡真的是不存在那個「大帝國」的世界――？」

。

「我可以斷言，絕對沒錯」

。

田原聽到這些彷彿放心似的吐了口氣，他自己也叼著香煙點上火。這傢伙也是不遜色於我的煙鬼呢。

或許是吸幾口煙冷靜下來了吧，田原徐徐地開口說道。

。

「在長官殿下面前說這些有點那個，不過如果這裡真是沒有大帝國的世界，那就沒有比這還令人開心的事情了呢。雖然我不知道……你是怎麼想的」

。

起初田原是有所顧慮，不過最終還是凝視這邊的雙眼開口說道。

如果是「九內伯鬥那倒是另當別論，不然即便對我說這種事，在我看來所謂的大帝國終究只是遊戲的「背景」；是為了運行那個生存競爭遊戲的「舞臺裝置」罷了。

。

盡管我為了令玩家投入「遊戲」中而細心地添上世界觀，不過對於玩家而言其實只是毫無關系的要素。

。

實際上即便添加一大堆小故事，也不可能會有熱心的玩家看完全部。

從正義那側的玩家角度上看，在遊戲開始時就相當於「啊浮，這遊戲的國家是這樣子呢」。

。

「也沒必要因為在我面前就有所顧慮。畢竟我們現在已經不會受到大帝國的束縛」

。

「是這樣嗎？那麼，順便再讓我問幾個問題――你究竟打算在這個世界上做什麼？最終目的是想要回到原本的世界嗎？讓我們集合起來是打算要我們做什麼事情？」

。

難以回答的提問。

這並非是怎麼回答這傢伙才會接受的問題――

而是我自己都還沒決定好最終目的以及行動方向。

取回全部的權限丶管理機能，到這裡還好。

。

調查熾天使，總有一天回到原本的世界――這種事真的有可能實現嗎。

話又說回來，即便我回到原本的世界又打算要做什麼？

又是一成不變的日常生活與工作嗎？

遲早會與別人結婚，養育子女，最終走進墳墓之中……。

。

不，在那之前――這些被我召喚的傢伙會怎麼樣？

即便要回到原本的世界，遊戲也已經不復存在了吧？

換句話說，我相當於是間接地把這些傢伙從那個世界上完全「抹殺」掉了。

當我自己的事情結束之後，又會再次受到「抹殺」而消失嗎？

。

「我現在可以說出來的事情並不多。不過呢，至少――」

。

田原的雙眼看向這邊，非常的平靜，有種不可思議的感覺。

猶如剝開著心靈屏障，讀取全部思慮的眼神。

。

「我想要走上與大帝國相反的道路」

。

「――嘿~」

。

我把以前曾經對悠說過的這句話再次說出口。實際上，我做夢也不會去想到我會走上這種血腥的道路。畢竟走上同樣的道路就代表我要藉由武力支配世界。總覺得有些悲哀，為什麼來到異世界還非得殺人，遭人怨恨呢。

。

「……嘛，算了。我欠你一份恩情。如果不是你讓我賺一筆，真奈美早就餓死在街頭上了」

。

「……關於你妹妹這件事」

。

這傢伙是重度妹控。

正因為是自己製作的設定，所以絕對不會錯。

在不能與妹妹相見的環境中，他會怎麼看待這種事呢？

。

「好了啦。像我這種做著骯臟買賣的哥哥，肯定還是不在身邊比較好。而且已經託付給創玄爺爺了，根本不需要擔心」

。

創玄……說起來，在田原的故事中是有描述過這位爺爺的角色。

他在大帝國的上層部也是有頭有臉的人物，也是具有歷史淵源的神社神主丶大地主。

有描述過這人物實在太好了……如果沒有製作過這種設定的話，估計會用槍口指著我，讓我立刻送他回去。

。

「然後呢，目前要做什麼事才好――？」

。

「首先是經營這座貧寒的村莊。設施有野戰醫院丶溫泉旅館丶澡堂這三個」

。

改變整座村莊的結構，工作人員的教育，產生利益的運營，糾紛處理，與貴族的交涉，照看小孩。

越是告訴他這些事情，田原的表情也就扭曲得越令人感到愉快。

。

「別說這種亂來的事情！我可不是藍貓型的機器人！」

。

「你是「天才」――絕對做得到。既然是我說的肯定不會錯」

。

「嗚哇！喔……嘎……」

。

「怎麼了？」

。

田原突然跳起來，表情顯得有些奇妙。

之前悠也露出過這種反應吧？

。

「剛才，身體好像有種刺痛感…………」

。

「身體狀況不好嗎？讓悠看一下吧」

。

「給我住手！要是被那傢伙看一下，身體會被搞得四分五裂！」

。

――說得真過分呢，田原。

。

「額！瘋女……不，不對，這不是悠嗎……哈哈哈……」

。

回頭一看，在那邊的是身穿浴衣，剛洗完澡的悠。

她的臉上微微泛紅，極為美麗，然而眼神好可怕。

連我都忍不住桂開目光。
