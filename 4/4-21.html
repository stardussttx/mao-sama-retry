4-21 第53話 前往北方

一一一一一一一一一一

魔王在溫泉旅館的大門前抽著煙，等待霍瓦伊特出來。

他總算是能夠在事情鬧大之前成功解決事態，不過他也是一個一旦下定決心就會貫徹到底，絕不會在緊要關頭出現差錯。

。

（就跟郊遊一樣……平安無事地讓她回家）

。

不久之後，旅館的大門處出現霍瓦伊特的身影，不過她頭上還是戴著天使之環。

。

（還戴在頭上嗎……嘛，雖然很合適）

。

對於魔王而言，他原本是打算用可愛的天使系道具作為禮物來博得她的歡心。畢竟她是聖女，這種類型應該會感到高興吧。

盡管遊戲內容是以殺伐為主，不過由於女性玩家也有不少的緣故，大多數道具都是比較可愛的。

。

其中不乏刺繡出花紋的連衣裙丶金色手鐲丶黑白相間的女僕服丶黑色以及白色的過膝長襪丶底厚長筒皮鞋丶高跟鞋，甚至還有貓狗的人偶服裝或者肉球手套和尾巴。

。

還有百褶裙和迷你短裙，以及不知為何還存在的泡泡襪和條紋內褲。在各種意義上，都可窺視到遊戲裡的玩家變態的一幕。

。

「聖女霍瓦伊特，你是乘坐馬車來到這座村莊的嗎？」

。

「不丶不是……」

。

「那麼，是獨自前來？」

。

「是的……」

。

面對魔王的詢問，霍瓦伊特說不出話來。

因為無論對像是誰，她都無法輕易地說出「奇跡」。對於霍瓦伊特而言，她唯有曖昧地回答。

。

「原來如此，那麼，由我來送你到聖城吧。」

「誒？到底要怎麼做……啊……」

。

魔王二話不說地就抱住霍瓦伊特的腰，拉進身邊。

在魔王的角度上來看，這並沒有其他意思。只是不想讓國家重要人物受傷而緊緊把抓牢而已。

。

「那丶那個！你丶你打算做什麼……」

。

「完全不需要擔心。盡管把身體交給我」

。

魔王的聲音在霍瓦伊特的耳際響起。那是深深地殘留於耳朵裡的聲音。

其聲音的力度以及強硬已經達到完全無法抵抗的領域。跟霍瓦伊特至今看過的男人相比，簡直就是不同次元的存在。

。

「那個，你把我的――啊」

「――安靜，這種時候就應當保持沉默」

。

魔王用食指輕輕地抵在霍瓦伊特那想要說些什麼的小脣上。

這個舉動也沒有其他意思。

他只是覺得如果移動過程中舌頭被咬到就糟糕了。然而總覺得霍瓦伊特面乏紅暈，身體也僵硬起來了。

。

「翅膀喲，帶飛吾身吧――《全移動：聖城》」

。

魔王為了矇混過關，隨意地說出像是魔法的詠唱（？），兩人的身影一瞬間便出現在聖城前面。映照於視野之中的――是早已見慣的聖城。

面對這不可能出現的光景，霍瓦伊特在驚愕的同時，全身也止不住顫抖。

。

這也不是沒道理。

並非毫無道理可言。

。

這與她引發的「奇跡」相同――絕對是天使的偉業。

。

「你丶你究竟是……！？」

。

「我曾經對你說過一句話。――比起聽取他人所說，不如實際觀望理解比較快」

。

「啊……」

。

這句話令霍瓦伊特的心頭一緊。

魔王的確在聖城進行對談時，說過這句話。

。

「如今你平安無事的回歸。無論事情緣由如何，這就是全部」

。

實際上，霍瓦伊特的身體並沒有受到傷害。

不如說泡到溫泉，又喝了能夠回復氣力的日本酒，她的身體已經是狀態絕佳。

更重要的是在她頭上――浮空著的「天使之環」。

。

「以防萬一先說一下――今天發生的事情別告訴其他人」

。

「好的……」

。

霍瓦伊特有氣無力地點著頭。

即使不用明說，也不可能輕易地說出這種事情吧。畢竟搞不好會在全國引發大混亂。

某人可以同樣行使熾天使遺留下來的奇跡。

某人可以把只存在於天使頭上的神聖光環賦予給他人。

。

「我再說一遍――別告訴其他人」

。

「好丶好的……！」

。

面對這道甚至可以感覺到拼命的聲音，霍瓦伊特連忙點頭。

然而，她忍不住詢問道。

。

「你到底是什麼人？我已經搞不清楚了……」

。

魔王對於這詢問難得露出思索的表情。

即便是自己毫無疑問也是不清楚。他的身體是遊戲中被稱之為魔王的角色，裡面則是有現代日本人的「大野晶」存在。

若被人詢問是什麼人，當然是難以說明。不久之後，魔王彷彿像是下定決心一般開口說道。

。

「遲早有一天，有些事情需要你的幫忙。以前我也說過了，我想要調查熾天使的事情」

「為什麼要調查熾天使大人……」

。

「――因為我是被「座天使」喚來的存在」

。

「被座天使大人！？」

。

霍瓦伊特的臉上浮現出驚愕之色。

然而，或許是理解到什麼事情了。她表情逐漸的變化著實有趣。因為如果這些話是真實的，那麼眾多無法理解的事情也可以解釋清楚。

。

那個任性妄為可卻對於智天使擁有深度信仰的露娜，為何會傾倒於他到這種程度。就連那位難以應付的瑪妲穆都離開中央，趕往他的身邊這件事也是。

。

以及跟熾天使一樣「行使奇跡」這件事。

還有實施賦予別人天使之環這種非同尋常的偉業。

再來是那座無法想像是由人類之手製作而成的不可思議的設施。霍瓦伊特的思維在腦海里全速運轉，許多的點正逐漸連成一條線。

。

「您是與熾天使大人敵對的存在嗎……？」

「至少我是沒有這個打算。只是有幾個問題想要問一下而已」

。

說完這些之後，魔王終於把手從她腰上放開。由於話題過於集中，以至於到現在才察覺到自己跟她抱在一起吧。

魔王應該是想到了什麼事，把天使之環從霍瓦伊特頭上取下，又再為她戴上，無數次重復這種行為，以認真的眼神注視著她。

「啊……那丶那個……」

。

「果然天使之環非常適合你」

。

「～～～～～！」

。

盡管霍瓦伊特想要說些什麼，不過魔王轉過身甩了甩漆黑斗篷。

從他的背影傳來一道沉著冷靜的聲音。

。

「那麼，下次再會吧――聖女霍瓦伊特」

。

只留下這句話，魔王的身影就消失無蹤。

霍瓦伊特當場呆立不動，不久後從頭上取下天使之環，緊抱在她那巨大的胸部裡。

不知是否是錯覺，她臉上泛著紅暈，似乎很高興的樣子。

。

霍瓦伊特的腦海中浮現出幾個假說――

。

像著名的天使路西法。

於遠古反抗宏偉的光，在傳承中被記載為墮天使丶魔王（撒旦）的存在。

縱使沒有流傳在伝承之中，也有其他相似的存在吧。

因為在古老的傳承裡，也有身為惡魔卻所屬於天使一側的存在。

。

（天使大人的丶光環……）

。

霍瓦伊特手中的光環絲毫沒有喪失它的光輝，釋放出燦爛奪目的神聖之光。在她看來，根本無法想像製作出這種東西的存在是罪惡至極的人物。

。

（我一定會把您當做是我原本的天使大人……）

。

霍瓦伊特邊抱著天使之環，邊高興似的閉上眼睛。

。

■□■□

。

――夜晚　拉比之村

。

（簡直就像是海盜木桶一樣啊……一旦刺錯地方腦袋就搬家了喔）

。

魔王坐在溫泉旅館的廊子上，抽著煙仰望著月亮。

他如今的表情就像是完成某種壯舉的男子漢。

盡管在別人看來他只是看見聖女的裸體，享受混浴，再令她喝日本酒，把一切都付諸於流的**行為。不過他臉上甚至還浮現出自豪。

。

「魔王大人，你在這裡啊」

。

「嗯，惡嗎……盡管過來這裡」

。

盡管魔王往旁邊一指，不過惡就這樣坐在膝蓋上。

最近惡已經沒有半分顧慮丶半分躊躇。魔王在對待惡的時候也是隨她喜歡，不過只有在這種時候會微微浮現出困惑的神色。

惡今天也穿著黃色的浴衣，她那屁股的柔軟度直接傳達到大腿上。

。

「聽好了，惡。現在或許還可以這樣做，不過跟男人要適當地保持距離。你這份警戒心就算是我也開始擔心起來了喔」

。

「我根本不會靠近魔王大人以外的男人」

。

「是嗎……？那就好……」

。

「嗯」

。

惡把後背託付於魔王，放鬆身體。

該說是完全信賴吧。

總之魔王也拉近身軀，摸摸她的頭。

。

「魔王大人從明天開始就要前往北方了吧？」

。

「話是這麼說，我有全移動啊。隨時都能夠回來的旅行根本算不上旅行吧」

。

「……嗯，不過好寂寞」

。

惡揪住魔王的襯衫，仰望著他。見狀，魔王一瞬間露出困惑的表情，不過還是以明亮的聲音回復她。

。

「如果有一些有趣的土產我會買回來的。你就期待地等著好了」

。

「……魔王大人，你要平安無事地回來哦？」

。

「哈哈，你以為我是誰啊」

。

沒錯，盡管這男人性格愛開玩笑，不過卻是名副其實的「魔王」。

其麾下的親信也同樣具備超脫常規的戰力。

只要這男人真心有「那個意圖」，豈止是這片大陸，甚至會為全世界招致非同尋常的血河吧。

。

「惡，你去哪裡了啦？，我說過你當我的抱枕了吧！」

。

「露娜真粗暴」

。

廊子對面傳來慌亂的腳步聲，露娜和託瀧從中出現。

。

「惡的話在我膝蓋上喔」

。

「你丶你這個變態！不單單是我的屁股，連惡的屁股也是！」

。

「說起來，我忘了跟你說了，你的姐姐已經送到城裡了喔」

。

「你丶你你你啊，難道說連姐姐大人你都要伸出毒牙！？」

。

「我好睏，差不多該睡覺了」

。

魔王一番話令露娜吵嚷著，而託瀧正如其名，發出睏倦的聲音離開了。

雖說離開了，也只是地方轉移到了房間而已。

。

跟以往相同的成員，引發平時常有的鬧劇，如同往常一般小孩子圍住魔王的被褥並躺上去。

右側是惡，左側是露娜，而身體之上則是被託瀧所佔據。

正所謂是魔之三角。

。

（這根本不可能睡得著吧！差不多給我適可而止了吧！）

。

如此這般，魔王在無法一眠的情況下――迎來旅行啟程的早晨。

■□■□

――翌日早晨。

。

天空仍有些陰暗，不過村莊門口早已停著一輛大馬車。

雪風和蜜康已經乘坐在上面了吧。

這是一輛能夠輕易容納八個大人的大型馬車。

。

由於魔王討厭送行，因此在這裡的只有田原和悠。實際上，若考慮到全移動的機能，根本不用勞師動眾地來送行。

。

「悠，醫院的事情就交給你了。努力提高吾等的評價」

。

「遵命，一切包在我身上。啊，長官……領帶」

。悠走到魔王身邊，以溫柔的手法把領帶糾正。

其實領帶倒也不怎麼彎曲，只是她單純想做做看而已吧。光看這幅樣子，就像是為獨自赴任的丈夫送行般的妻子。

。

「田原，村裡的事情全由你來負責。如果有緊急事態，隨時都可以用《通信》傳達給我」

。

「了～解。不過我可沒打算要犯下勞煩長官殿下的差錯」

。

實際上田原所不能處理的事情就是相當嚴重的事態。

因為天才都無法處理的事情，「大野晶」根本不可能會處理。當這位魔王開始行動時，那隻會是――伴隨著「行使實力」的時刻。

。

魔王遠遠地往整座村莊望去。

在仍沒完全迎來拂曉的天空之下，已經有幾個兔人來到農田裡開始幹活。農民的早晨比較快開始吧。

往村子裡四周一看，仍有幾處在幾小時後繼續施工的工程。

。

村莊發生變化。

正逐漸改變。

只因一個男人的來訪。

。

（既然已經走到這一步，那就只能徹底地做好了吧）

。

魔王那漆黑的眼瞳裡，映照出溫泉旅館的身影。

在那裡有幾個因遇見魔王而徹底改變命運的人。

。

應當守護的東西增加了――

。

魔王閉著眼睛，坦率地如此想道。

而考慮到為了守護他們，那必須就得變得更強。只要魔法這一弱點沒有克服，遲早會遭他人所利用。

。

魔王再次睜開眼睛時――

悠開心地糾好領帶的手便停下動作。

因為她從中窺視到具有絕對力量的「神」，它的存在。

。

――田原，再給你一個指示。

。

「若有惡意者接近這座村莊，那就消滅掉，一個不留。可以吧？」

。

「……瞭解」

。

田原簡潔地回答道。

盡管他在煎熬中表情毫無變化，不過身體止不住顫抖。因為他從魔王身上感覺到他無論如何都無法抵抗的「絕對力量」。

。

魔王翻開漆黑的斗篷，坐上大型馬車。

與此同時，馬車也勢如破竹地跑起來。田原和悠直到看不到馬車的身影之前，都只是在原地呆站不動。

不久後，田原嘀咕一句。「我啊，不知道怎麼回事……在剛才回想起懷念的往事」

。

「懷念的往事？」

。

「跟長官殿下初次見面那天的事情」

。

「……真令人興趣盎然呢」

。

田原從衣袋裡拿出香煙，點上火。或許是直接放進衣袋裡的緣故吧，香煙皺巴巴的，不過他似乎毫不在意。

。

「其實也不是什麼大不了的事情啦。只是覺得如果那時候拒絕長官殿下的邀請……――我應該會死吧」

。

或許是對田原這番話想到了什麼吧，悠也沉默不語。

兩人丶不對，在不夜城的委員會裡的成員全都是受到魔王的邀請而集中在一起。縱使他們每個人與魔王的相遇丶事情經歷有所不同，不過那位「魔王」才是一切的開始。

。

「那種怪物，即使竭盡全力也打不贏吧」

。

「這是當然，像你這種人跟長官相比就是一隻草履蟲」

。

「你啊，難道就沒有更好一點的比喻嗎？」

。

「長官……您就不能快點回來嗎」

。

「他不是才剛剛出發嗎！」

。

在兩名親信吵鬧的期間，馬車也逐漸移動著。

。

魔王乘坐的馬車所前往的方向――是戰亂漩渦，北之大地。

。

在那裡，會有嶄新的邂逅以及許多迷宮丶遺跡在等待著他吧。

。

魔王招致的混亂，終於擴散到大陸的中央處……

。

――――然而，那是在不遠的將來所發生的事情。

。
