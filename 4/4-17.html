雪風，襲來

一一一一一一一一一一

――神都

。

（最近真的是忙得不可開交啊……）

。

「我」久違地一個人走在街道上。

獨自一人挺不錯的。身心極為輕鬆，連步調也能隨自己的意向。

如果露娜或者惡在這裡，無論如何都需要配合她們的步調，畢竟我的步伐比較大――魔王的身體並不只是有能力而已，連體型也比較龐大。

。

全身如同鋼鐵般的身軀自不用說，身高也是足以跟模特相提並論的187。無論怎樣都顯得美觀。

因此――僅僅是走路也會引人注目。

。

「……叔叔大人，終於見面了」

。

「為什麼魔王在這裡！？為什麼！？」

。

轉頭一看，是以前見過幾次面的冒險者二人組。

一位是有著健康般褐色肌膚的戰士，另一位則是有著白皙通透般肌膚的魔法師。怎麼說呢，真是猶如室內與室外般極端的兩人組。

。

「喔，好久不見啊」

。

盡管我鄭重地回答道，不過我並不知道這些傢伙的名字。

又或者是聽過之後完全忘得一干二凈也說不定吧。畢竟最近經歷了許多事情，也不是不可能。

這世界也沒有交換名片的習俗啊……。

。

（嗯……名片嗎。或許會流行起來吧）

。

可能是由於最近都在考慮金錢的事情吧，腦海里終於想起了這種事。

實際上拉比村的特產僅有胡蘿卜也太寒酸了。試著考慮其他特產或許也不錯。

。

「……叔叔大人為什麼會在神都？」

。

「稍微有點事想要去趟冒險者公會」

。

「……公會？你有什麼想知道的事情？」

。

「嘛，是關於北方迷宮的事情」

。

其實我是想打聽有關抵禦魔法的道具，不過這不是可以特意向人詢問的事情。

畢竟這就是在展現出自己的弱點一樣。

。

「……國外的事情即便向職員打聽也不會有多少收獲，我來告訴你」

。

「等一下啦，雪風！桂一個人擅自決定！」

。

「……那麼，蜜康回家去，在家裡凍成冷橘子」

（颯:蜜康音譯，意思是橘子）

。

「誰是冷橘子啊！」

。

原來如此，這些傢伙分別是雪風和蜜康嗎。

該說簡單易懂嗎，還是怎麼說呢。

這對組合就簡稱為雪橘子吧。

。

「……叔叔大人，這附近有家我們經常關顧的店。算上之前的回禮，我請客」

。

「以前我就說過了。根本沒必要報恩」

。

「……嗯，說是「重要的事情」」

。

「正是如此」

。

當說完這句話的瞬間――潔白的手心便溫柔地握住我的手。

這傢伙真的是很白啊。

就好像是某種無法融化的雪一般，甚至都有股神祕感。

。

「……雖然現在還是白天，不過那邊也可以喝酒」

。

「酒嗎……」

。

有時候我也想久違地喝喝酒。

而且最近幾乎沒有時間讓我一個人獨處。

。

「……那麼，這邊走」

。

「我就跟在後面吧，又不是小孩子，根本沒必要牽著手喔」

。

「……不行，會迷路的，主要是我」

。

（原來是你啊！）

。

我差點忍不住大喊，不過總算是忍住了。這孩子，雖然表面上看會給人一種在圖書館安靜看書的穩重印像，不過其實是天然呆嗎？

。

「……蜜康，帶路」

。

「知道了啦！我去就行了吧，我也去就好了吧！」

。

看樣子我似乎不怎麼受到這位叫做蜜康的孩子歡迎。

嘛，畢竟被別人稱之為魔王什麼的，也不會有令人喜歡的因素吧。

總之，在村莊的經營走上軌道之前都得忍耐。

。

「……戀愛迷宮，Labyrinth Love」

。

（在說什麼啊，這傢伙……）

。

我被帶到的地方，是一家招牌寫著諾瑪諾瑪的酒館。

盡管時候還早，可店裡似乎有相當多的客人。豎耳傾聽裡面傳出來的聲音，能聽見男女喝著酒，情緒高漲的聲音。

。

「老闆娘，來三杯啤酒！我把客人帶過來了哦！」

。

「哦呀，這不是蜜康嗎，真稀奇呢，客人居然會是你帶過來……誒！？」

。

進去店裡面，老闆娘和店裡的客人都一起看向這邊。

真是不得了注目度。

我有做過什麼事情嗎……不對，難道說這些人就是在前陣子引發的騷亂裡的那些人？

。

「這不是傳聞中的魔王大人嗎！蜜康，幹得漂亮！」

「噢噢，你是！」

「來這邊這一桌，我請你喝一杯喔！」

「嗯？誰啊，他？」

「你傻啊！他就是之前把伽尼巴魯揍飛的老爺啊！」

「喂，給我把閑著的傢伙都叫過來！」

。

（喂喂喂喂！）

。

店裡的人連忙行動起來，越來越吵吵嚷嚷。

我明明只是想問點事情而已！

。

「……叔叔大人，坐在這裡」

。

「嗯」

。

總之我被帶到墻邊的餐桌上坐下來。

然而實在不是可以輕鬆喝酒的氣氛。老闆娘豪爽地把啤酒放在餐桌上，為前陣子的事情對我表示感謝。

。

「包括這附近的店家，大家都很感謝你。今天無論你喝多少酒，都由我們這邊來買單吧！」

。

老闆娘拍打著我的肩膀，豪爽地大笑。

瑪妲穆也好，這位老闆娘也好，都擁有不遜色於那邊男人的風度。

無論如何還是先喝杯酒吧。事情待會也可以處理。

我把手伸向啤酒杯，雪風也從旁拎住啤酒杯，還輕巧地坐在我的大腿上。

。。

「你在做什麼……？」

。

「……喂叔叔大人喝酒」

。

「抱歉，讓我自己來吧」

。

這是哪來的郊區夜總會啊。

怎麼可能會有大白天就以這種姿勢喝酒的笨蛋。大體上，喝酒若不能隨心所欲，自由地慰藉自己的話是不行的。

我讓雪風從大腿上下來，終於喝上一口啤酒。

。

「嗯，真好喝」

。

甚至都覺得這幾天的疲憊都是為了這一刻。

當我在為久違的啤酒而感動時，一旁的雪風把我手上的啤酒杯奪走，自己一飲而盡。

明明自己也有啤酒，為什麼偏要喝我的？

。

「嗯……跟叔叔大人的喉嚨連在一起……」

。

「別給我用這種奇妙的說法」

。

這些傢伙居然都全給我用這種莫名其妙地說法。

我早就說過我會被衛兵叔叔帶走的吧！

。

「……間接KISS。戀愛是Bubblicious」

。

「抱歉，給我好好說人話」

。

「小雪風還是老樣子呢。好了，我把吃的東西拿過來了哦」

。

「喔，這是――」

。

像是炒豆一樣菜色丶烤肉串丶以及炒蔬菜的料理接二連三地搬上餐桌。無論是外觀還是香氣都蠻不錯。

在亞侯之城吃過的料理並不怎麼美味，這裡又會是如何呢？

這支烤肉串表面上看起來加上大量的調味料，我將其放進口中。

。

「這還是……普通地美味啊」

。

「普通，你也會用這種失禮的說法呢」

。

「抱歉，我沒其他意思。在亞侯之城有吃過類似的事物，不過並不怎麼美味」

。

「啊，原來如此。可是把我這邊跟那種地方相提並論也實在是困擾呢」

。

老闆娘豪爽地笑著，又回到櫃臺裡面。這位老闆娘的手腕或許挺了不起的吧。

在我想著這種事情的時候，雪風把我手裡的烤肉串搶走了。

這傢伙怎麼老是把別人吃過的食物拿走？

。
