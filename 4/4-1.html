4-1 第33話 與聖女的交談

一一一一一一一一一一一一一一一一

神都如今聚集著許多民眾，正在進行修復工作。

盡管已經避免最糟糕的事態，可損傷卻不小。召集而來的大量人們把木材或者石頭都陸續搬往現場。

還能看見大街上擺置著幾個大鍋，工匠們正狼吞虎嚥地吃著飯菜。

。

「我啊，看到了！傳聞中的銀色龍人！」

「笨蛋，我可是看見了魔王喔！那真是不得了的人物啊！」

「然後呢，到底哪一邊比較強啊？」

「這種事像我們這些普通人根本不可能會知道吧！」

。

話題的中心當然是那「兩人」――雖說是同一人物。

神都到處都在討論這件熱門話題。

這也並非毫無道理。因為魔王與龍人之類的人物若不成為傳聞反倒是奇怪了。

早有耳聞的北方諸國之中，已經有國家往聖光國派遣本領高超的間諜打聽詳細的情報。

。

而傳聞中的中心人物――則在聖城的接待室與最後的聖女會面。

■□■□

。

聖城深處的某間接待室。

這裡已經徹底驅散無關人員，在做好周密的盜聽對策之後才迎接自稱魔王的人物。

房間裡就只有聖女霍瓦伊特與魔王兩人。

。

瞬間令多達500名惡魔信奉者屈服，哼著鼻歌把中級惡魔炸死的存在。

霍瓦伊特在聽取上次引發的騷動之後，深切地感覺到自己的危機意識過於天真這件事。

她確信惡魔王的復活以及消滅都與這個魔王有所關聯。

。

尤其令人懼怕的事情是這魔王能夠輕而易舉地就「進入」聖城。

甚至就連智天使所鋪設的結界都無法拒絕這位魔王。

面對這一荒唐無稽的事實，霍瓦伊特感覺到了戰慄。

。

「……初次見面，魔……要怎麼稱呼您才好呢？」

。

「魔王就好了。在我看來，這名字就像是綽號或者別名一樣」

。

魔王以沉著冷靜的舉止拿起一杯看似咖啡的飲品，喝完一口之後又從懷裡拿出香煙點上火。

其旁若無人的舉動令霍瓦伊特一瞬間皺著臉面。

。

在聖城而且可說是聖女頂端的她面前，悠然自得抽著煙之類的可謂是前所未聞。今後的未來也永遠不會出現這號人物吧。

。

或許是想追究聖光國這邊準備不周全吧，魔王特意從懷裡取出疑似煙灰缸的物品，像是展現給霍瓦伊特看到一般放在桌子上。

如果這是在「外交」，那也的確算是過失吧。既然提出會談的是霍瓦伊特，那麼本就應該做好萬全的準備以令對像感到舒適愉快。

。

「您――真的是傳承中的「魔王」嗎」

。

「無論你們怎麼猜測，請盡管隨意――既然您如此認為，那應該就是這樣吧」

。

魔王所說的話令霍瓦伊特在圓桌下不禁緊握拳頭。

這究竟是何等的態度。

他在宣言著無論如何你們就擅自想像吧，這邊被怎麼認為都毫不在乎。

。

從態度中傳達過來的情感――便是壓倒性的自信。

彷彿在訴說著只要他想，這個國家隨時都可以毀滅掉一樣。

。

「您究竟打算要把露娜怎麼樣――」

。

霍瓦伊特往詢問的話語中注入力道。

按照原本的預定本應是之後再來打聽的事情，不過她已經無法再按耐住自己了吧。

因為她是為妹妹著想的正經聖女。

。

「我與她的相遇實在頗為不幸――她應該是以為我是邪惡的存在吧。然而如今我們已經構建了「美妙的關系」喔」

。

這句「邪惡的字眼」令霍瓦伊特緊咬牙根。

究竟是使用了什麼魔法――那個宛若穿著名為任性妄為的衣服走動的露娜，莫名地親近這位魔王且不離開他的身邊。

這實在是不可能發生的事情。

。

「您是為這個國家招來禍害的存在嗎――」

。

霍瓦伊特直截了當的出言說道。

若是平時的她絕不會直接把這種事情說出口。

不僅是國家的各種活動，就連應付他國而來的使者之類的事情都全部是由她一手操辦。

。

其圓滑卻又貫徹到底的外交方式令諸國都甘拜下風。

直接向對方打聽這麼粗暴的事情，應該是頗具有自己的考慮吧。

。

「就是這一點喔，我想要說的是」

。

魔王享受著吸煙的同時，以他那銳利的眼光看向霍瓦伊特。

僅是如此，霍瓦伊特便暗中顫抖著身軀。

這究竟是什麼眼神，又是什麼威壓呢。都甚至令人感到其全身丶其一切都是為了使得人類感受到恐怖而存在。

。

「在我的國家有句話叫做「百聞不如一見」。意思是比起從他人所聽說，還不如自己實際見聞一下比較容易理解」

。

魔王滿臉笑容，以自信滿滿的態度說道。

他這樣子咋看之下令人感到友好，不過對於霍瓦伊特而言怎麼也無法相信。

。

畢竟站在她的角度上――

。

她最重要的一個妹妹已經成為了「人質」。

如果強行把露娜關在聖城裡，根本不知道魔王會以此作為藉口幹出什麼事來。

不對，恐怕他就是在「等待」這種事吧，霍瓦伊特思考著。

。

關鍵是關押露娜在物理上是不可能實現的事情。

因為無論關在什麼地方，即便找遍全世界也根本找不到可以抵禦那個「金色」魔法的建築物。。

「您只需實際觀摩我接下來要做的事情，再以此定下判斷便可。我從以前開始就是以實際行動展現意向，而不是口舌上的說明」

。

魔王過於自信的態度讓霍瓦伊特的警戒心更上一層樓。

雖然還不知道要做什麼事情，不過對這個國家的侵略行為著實在進行中，這樣。

表面上面露微笑的同時，其背地裡已經在確實地進行著什麼――

在霍瓦伊特看來，眼前的人物正所謂是「魔王本尊」。

。

「還有，我有一件事想拜託您――」

。

終於來了，如此想道的霍瓦伊特做好覺悟。

在前些天的騷動中，這位魔王姑且有做出鎮壓行動。藉由這份功績向國家提出某些要求也是在意料之中的事情。

。

「我想要調查一下關於「熾天使」的事情――如有書庫之類的地方，還請讓我自由的閱覽」

。

霍瓦伊特感覺到眼前突然一黑。

不對，事實上已經引發頭暈了。

他偏偏說出想要知道「熾天使大人」之類的事情。是意圖殺死嗎，又或是尋找弱點呢。

尤其是堂堂正正地在聖女面前說出這種話來，霍瓦伊特感受到前所未有的恐怖。

。

「書庫的閱覽……請隨意……但是，請容我拒絕我親口說出熾天使大人的事情」

。

「原來如此，這就足夠了。因為像您這麼高貴的人士親口闡述，我也只會感到不勝惶恐」

。

魔王深深地點頭，表示理解。

霍瓦伊特從中感覺到強烈的諷刺感。

應該是在譏笑這邊的垂死掙扎吧，她如此想著。

。

「那麼，我差不多也該告辭了――真是非常有意義的時間呢」

。

「嗯，對於這邊來說也是……真的是有意義的時間」

。

魔王和聖女互相握手，接著魔王便翻開漆黒的斗篷走出房間。霍瓦伊特則對著他的背影詢問最後一個問題。

。

「是您――消滅了「惡魔王」吧」

。

面對這個疑問，魔王持續一段很長的沉默。

不久後他便張口說道，那是對於霍瓦伊特而言極為恐懼的話語。

。

「那東西稱之為「王」，有點滑稽可笑呢。只不過是一事無成的廢物――」

。

連惡魔王甚至都是廢物。

絲毫沒有資格自稱為王。

魔王以他的背影雄辯地述說著，隨即離開了房間。
