3-4 第21話 冒險者們

一一一一一一一一一一一一一一一一

――聖光國　某處

。

那裡是一個無法令人想像是在地下的廣闊空間。

這裡是惡魔信奉者的根據地，並且還是幹部級的成員集中的地方。現在正針對於前些天的襲擊展開商議。

。

「沃金格，你連奈落都攜帶而出，可卻是這幅慘樣嗎？」

。

「雖說你精通指揮，不過看來還是沒有擺脫膽怯怕事的缺點」

。

交談的對話中，大多數是在辱罵沃金格。

然而他並沒有提出反論，一直在忍耐著。因為無論是誰身在當場，都會確信那已經是必敗無疑。

。

「那個龍人擁有不同尋常的力量。僅此而已」

。

面對這句彷彿無話可說的斷然說辭，其他人更加喧嘩。

然而，位於深處椅子上坐著的人物在舉起手的瞬間――房間隨即便鴉雀無聲。

惡魔信奉者的領導者——烏託邦沉穩地開口說道。

。

「比起那個男人，我更在意「魔王出現」這條傳聞」

。

誰也無法對這句話做出回答。

魔王降臨是他們的夙願，然而奔赴願意祠堂之者卻沒有任何人回來過。所以視為計劃失敗也是極為妥當。

。

目前神都已經流傳起「自稱魔王的存在」的人物畫，可那無論怎麼看都只不過是男性的人類。雖然長相有幾分可怕，不過還是無法稱得上魔族。

。

「也有這條傳聞呢，惡魔王蘇醒了――不過遭到魔王殺害」

。

烏託邦接著敘述的字句令所有人面露苦澀的表情，自己極其奢望降臨的魔王把惡魔王殺死，那是多麼荒唐無稽的事情。

。

盡管謠言說已經遭到魔王殺害，可就連惡魔王的復活都甚至只不過是傳聞罷了，只是聽在耳裡著實令他們感到不是滋味。

。

「無論如何，奈落需要更多的「祭品」與「力量」」

。

那便意味著更大的混亂以及招致更多的傷亡。

所有人聽聞至此，都深深地點頭。對於他們而言，他們認為這個國家的病魔已經達到無法痊癒的地步。

既然如此，便唯有把全部化成灰燼再來重新建立，他們便應該是如此考慮的吧。

。

「――毀滅神都，無論好壞都一同投放於奈落裡面吧」

。

從烏託邦口裡說出的話語便是破滅的宣言。

幹部們連忙從席位上站起，各自為了做好應做的準備而離開這房間。還留在房間裡的只剩烏託邦一人。

。

「古雷歐魯這愚蠢之者――究竟栽在了什麼「小石」身上了呢」

。

在誰也不在的房間裡，烏託邦獨自嘀咕著這句話。

。

■□■□

。

――神都　酒館「諾瑪諾瑪」

。

若說起神都聚集的冒險者最多的酒館，那應該便是這裡吧。

無關冒險者階級高低，冒險者從早期以來便有時常關顧的傾向。

多數人在仍未熬出頭的時候開始就已經受到店家的關照，所以應該都懷有強烈的留戀之情吧。

。

在這層意義而言，受到這家店關照的人為數不少，聞名遐邇之後也有許多人繼續關顧這家店。

原本便是小型酒館的這家店，在經歷過數次的擴展，如今已然發展到號稱神都首屈一指的名店。

店家與顧客共同在成長，可堪稱是稀有的實例。

。

「魔王丶那就是魔王，不會錯的！」

。

蜜康向著店家老闆娘吐露出抱怨之類的難以形容的話。其手上還拿著一杯啤酒，貌似喝得酩酊大醉。

看來似乎是無法忘懷前些日子遇見的魔王。

。

「魔王……那傢伙是個好男人嗎？」

。

聽完這句話，老闆娘也回了句無比豪爽的話。

她的名字伊夜。

體態姣好，擁有五十多歲的媽媽風采。由於十分擅長照顧別人，因此仰慕她的冒險者也有不少。

。

「……是位時髦的叔叔大人。真想讓他填補我的心與屁股上的空洞」

。

「啊哈哈！小雪風還是老樣子吶。男人啊，只要抓住蛋蛋就是這邊的東西啦」

。

雪風邊喝著牛奶，邊說出令人無法恭維的言論。然而伊夜對此也豪爽地大笑道。

兩人沒喝醉就已經是這副樣子的話，一旦喝醉究竟會變成什麼樣呢，真是令人懼怕。

其中可謂是常識人的蜜康，無法忍受般紅著臉大叫道。

。

「雪風，那可是魔王哦？你啊，是明知道還要這麼說嗎！？說穿了，你根本就是男人吧！」

。

「……蜜康無知。偽娘也是能夠好好懷孕的」

。

「你那杯牛奶是加了酒精嗎！？」

。

身在酒館的男人們，聽到這段對話表情都扭曲了。

當聽聞自身深愛的偶像「雪風大人」的心已經被其他人所奪走，應該無法再保持冷靜了吧。

。

「那個男人……無丶法丶原丶諒」

「什麼魔王嘛！絕對是誇大妄想的傢伙」

「在下反而倒是想被雪風殿下搞得懷孕是也」

「白痴！夢話給我到睡覺的時候再說！」

。

他們作為雪風粉絲俱樂部的一員，跨越隊伍之間的隔閡組成了同盟。製作各式各樣的周邊商品，時而也會做出一些本子。

甚至還會有人身穿印有她樣貌的斗篷。

無論身處於哪個世界，粉絲所採取的行動似乎都是一個樣。

。

「說到底呀，那可是有將近三百匹沙狼哦……？　一瞬間就全部燒死什麼的怎麼可能做得到啊！那是叫做災難的存在啦！」

。

「……蜜康，你沒有忘記自己是被叔叔大人拯救這件事吧？」

。。

「嗚……雖丶雖然丶是這樣……」

。

「……報恩應當用身體償還」

。

「這種事你就給我自己去做吧！」

。

諾瑪諾瑪的喧嘩在今天也依舊是持續不斷。

然而，其中卻有一位男子把劍抱在膝蓋上，沒有與任何一個人進行交談，一直不停地在喝著度數不低的葡萄酒。

他是這個國家的著名劍士，是位譽為「劍閃之亞路貝魯託」之稱的男人。

以一劍便把一切都斬斷的劍豪。

。

「哈哈，自稱魔王的男人嗎……出現了一位提高名聲的人物啊」

。

把傳聞中的存在砍死――究竟會得到多少功名成就與財富呢。

他僅僅是想像著，便面露無畏的笑容。

。

■□■□

。

――神都　酒館「亞露緹米絲」

。

這裡並不是冒險者的酒館，而是貴族時常關顧的加上「超」字高級店。

店裡彌漫著可謂是靜謐的氛圍，是與喧鬧相差甚遠的氣氛。

在店裡的一個角落處，兩位女性正安靜地用著餐。

。

在周圍盡是貴族的環境中，身著冒險者打扮的她們是種輕浮的存在，可沒有人會對此提出意見。

因為她們便是稱之為「Star Player」的人物，位居S階級的世界級冒險者。

她們聊天的話題並不是魔王――而是「龍人」。

。

「歐露康，你覺得真的會是龍人嗎？」

。

女人吃了一口料理，並開口說道。

她身著的衣物充滿著神聖的力量，宛若純粹的修道士打扮。甚至就連下級惡魔都難以靠近。

然而，比什麼都值得一提的是她的胸部。

衣服彷彿都要撐破般高高頂起，令旁人目睹都會覺得痛苦似地。

。

女人名叫米庫。

毫無疑問是Star Player其中一人，其美麗的藍發與巨大的胸部無不誘惑著男人的眼光。

。

「不會――我雖然想這麼說，可在這世上也不是百分百不存在」

。

被稱呼為歐露康的女人吃了一口沙拉，回答道。

她並沒有往沙拉上添加什麼調味料，似乎就直接吃進嘴裡。

其身材嬌小得有如小孩子，由於全身覆蓋著一件黑色長袍，因此不是很清楚她的容貌。

不過，以嬌小玲瓏的小嘴巴吃著沙拉那副臉頰鼓鼓的身姿卻有幾分可愛之處。

。

她是極為稀奇的混血兒――「魔人」。

由於根據國家的不同而會成為該討伐的對像，因此她隱藏真面目，一直變更著活動據點度過日子。

。

聖光國當然也不認同魔人的存在，她們只是旅途中順路來到這裡罷了。

如今縱使找遍全世界也找不到知道她是魔人的人物。因為知情者――全部都已經消失得無影無蹤。

。

「不過，真是有夢想的事情呢。憎恨邪惡的龍人什麼的」

。

「雖說如果真的存在的話，獸人國不可能會放任不管」

。

「你想說他是野生的龍人？」

。

「無聊透頂。如果真的存在，那還請務必讓我見識一下――」

。

明明同樣是混血兒，龍人受人尊敬，魔人卻遭受忌諱。

從歐露康的角度來看，龍人是令她感到毫無道理可言，也是令她感到滑稽可笑的存在。

。

「我從傳聞中聽說魔王現身了。我的「暗」也在如此述說著」

。

「你應該是修道士吧……什麼暗啊」

。

「這只右眼的疼痛……不會錯的。令世界陷入混沌的魔王已經降臨，我們要討伐成為黑暗的魔王」

。

「你到底在說什麼啊」

。

遺憾的事情是米庫患上了中二病。

無論是哪支隊伍，有一方是奇特人物或許已經算是既定規律也說不定。

。

不知這些冒險者的思緒，魔王正逐漸接近神都。

與此同時，惡魔信奉者也開始在水面之下展開行動。

在不遠的將來，於神都所引發的騷亂已然無法避免――
