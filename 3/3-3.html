3-3 第20話 神之手

一一一一一一一一一一一一一一一一

在據點之中，當所有人的自我介紹結束之後，便立即開始治療惡的腳傷。

雖然我是相信悠的能力，不過只有這一次不嘗試一下我也不清楚。

不對……別害怕啊我。

連自己創造的人物角色都無法相信是要鬧哪樣。

。

「那麼，小惡……放鬆身體」

。

「好丶好的……」

。

只見悠右手的手指頭逐漸發生改變，其手指變化為注射器與手術刀。

這就是她持有的特殊能力――神之手(God Hand)。

。

雖說在遊戲中早已司空見慣，不過一旦出現在眼前還是相當驚悚，不如說好恐怖。

然而，其能力卻是附加上「驅逐丶治療這世界上一切的病魔與傷口」這種可謂犯規的設定。

。

「那丶那個……這丶這隻手……是……」

。

不出所料，由於外觀驚悚的緣故，連惡也似乎膽怯起來了。

不過該說悠是作為天才般的醫生這種設定過活的嗎，她浮現出柔和的笑容，為了令患者放心而開口說道。

太好了……這方面在以前有好好的進行設定。

。

「沒事的，小惡……就尖頭部分而已，就尖頭部分而已啦」

。

「嗚……氣喘吁吁的，總覺得好可怕！」

。

唔喔喔喔！這傢伙到底在說些什麼！

的確惡是有種中性的感覺，不過是女孩子喔！

根本不是你所喜歡小正太啊！

。

「小惡，治療結束之後要不要試一下男裝？跟姐姐一起玩吧」

。

「我丶我不想……！」

。

「悠，別玩了，快點做」

。

「這還真是失禮了」

。

再繼續放任不管也不知道她會說些什麼鬼。

話說，這傢伙喜歡美少年的設定好像是我弄的吧……。

總覺得受到責罵的應該是我才對。

。

「那麼，長官。開始了――」

。

「嗯，那就拜託你了」

。

悠的右手——變化為各種各樣的器官觸碰著惡的腳。

「那東西」彷彿像是在觸診般運作著，在疑似注射器的物體上滿溢出液體，她把這液體注射在惡的腳內。

悠能夠在體內生成各種藥物，並注射於對像身體內部。

這當然也包括毒藥一類。

。

「等丶等一下……魔王！真的沒問題嗎！？」

。

「別擔心。你就相信我和我的部下吧」

。

「我丶我可從來沒有相信過你哦……」

。

雖然我以冠冕堂皇的態度回答道，可其實內向是宛若祈禱般的心情。

應該是醫療工具的一種吧，悠的手指連續不斷地發生變化，依次接觸著惡的右腳。

而每一次都會令人覺得惡右腳的血色增加了。彷彿就像是沒有血液流通的部分開始恢復正常一樣。

。

（拜託你了喔，悠……）

。

「好了，這樣就結束。傷痕也幫你完美消除掉了。畢竟是女孩子的腳」

。

……誒，已經治好了嗎！？

悠的手指頭變化成像是刷子一樣的東西，每當撫摸著傷口表面，傷痕便逐漸消失不見。即便含蓄地說也是驚人的光景。

我安撫住激烈動搖的情緒，故作理所當然的舉止。

。

「嗯……結束了嗎」

。

「是的，長官。沒有問題了」

。

「惡，站得起來嗎？雖說已經治好傷口了，可也得稍微走一下讓身體習慣起來」

。

「好丶好的」

。

惡站了起來，走了幾步。

動作雖說有些僵硬，不過似乎已經不會再拖著腳走路了。

。

「走丶走起來了……！我丶我的腳！」

。

「那就好。那麼，稍微到外面走一下試試看吧」

。

「那丶那個！悠大人，非常感謝您……！關丶關於這件事，該怎麼說才好呢丶那個……」

。

「好了啦，要道謝的話就向長官道謝」

。

悠笑著如此說道，可我其實又沒做過什麼事……。

總之，讓她在外面走一走稍微練習一下吧。雖說我幾乎沒有康復訓練的相關知識，不過牽著她的手一起走路這種事我還是辦得到的。

。

「我稍微出去一下。你們就呆在這裡」

。

我牽著惡的手走出據點。

外面的太陽一如既往地在迎接我們，像今天這樣值得賀喜的日子，只有萬裡晴空的天氣才配得上――

。

「魔丶魔王大人！我真的在走路……這樣好厲害啊！」

。

「我早就說過了吧――大帝國沒有不可能」

。

我隻字不提剛才內心其實在祈禱的這件事，很偉大似的回答道。

不如說，我經常持續著演技也逐漸疲憊不堪了。在親信面前雖說還不能放鬆警惕，不過跟惡兩人單獨在一起的時候，即便不用裝模作樣也可以吧。

。

「真的就好像在做夢一樣……我能夠普通地走路什麼的」

。

「不是在做夢啦。從今往後都能普通的走路丶普通的生活」

。

「魔丶魔王大人……能不能捏一下我的臉？」

。

「又是這種古典丶的――」

。

我笑著說出口的言語，在途中就噤口不言。

因為從惡的眼睛裡――流出了眼淚。

看到她這副樣子，我莫名地無法冷靜而移開了目光。不行了，這種氛圍我從以前開始就很不擅長應付啊……。

。

「嘛，就是那個……怎麼說，實在是太好了」

。

什麼鬼，這句臺詞……！

就不能再稍微說些感覺不錯的話嗎。

明明平時就能夠隨意地胡扯幾句，關鍵時刻卻是這樣子，真想把自己的嘴巴給抓一頓。

。

「……魔王大人。可以再稍微走一下嗎？」惡如此說著，並握著我的手。雖然這風景實在不適合散步，不過其實我也不討厭在這個異世界散步。

乾涸的大地處處布滿龜裂，照射著大地的太陽也無情地釋放熱量。

可是，為什麼呢……我感覺比起現實中的生活，還是這邊的生活比較有趣。

。

（或許是因為有這孩子在吧……）

。

起初我只是在哀嘆我為什麼會來到這種異世界，如果能夠治好惡的腳傷，那我自己來到這個世界或許也算有所價值了。

一想到我是為此來到這裡，這種不講理的事情我也能夠勉強接受。

。

「已經不能再讓魔王大人背著我了呢――」

。

「沒事，要是你累了我還是會背著你的啦」

。

我莫名覺得惡的表情有股成熟的氣息，不禁點上了香煙。

雖說女孩子成長很快，可真的是這樣嗎。

不巧我只是單身狗，所以關於這方面的事情也不是很清楚。當我想著這種事情的時候，感覺到牽著的手上力量加強了。

。

「魔王大人，請一直要呆在我身邊哦」

「…………。這種話，你還是為了將來會喜歡的男人說出來比較好」

。

一直。

代表要在這世界上一直活下去。

全數舍棄現實中的一切。

明明今後還要調查熾天使的事情，我實在無法說出這種不負責任的話來。

。

「因為我已經「決定」了――」

。

「我說你啊……」

。

13歲的小孩子到底在說些什麼啊……。

想起初中的時候，也有一些人喜歡上身材高挑的老師，就跟那個差不多？

。

「那麼，是時候該回去了吧」

。

結果，牽著的手――在回到據點之前都沒有分開過。

■□■□

。

「長官還真是相當關心那個孩子呢……」

。

我不禁把這句話說出口。

從以前開始，長官對於滿足特定條件的人都會展現出判若兩人的寬大態度。

。

那是――對於「遊戲優勝者」。

。

向跨越諸多苦難與絕望的勇者約定，給與莫大的財富與深受黃金點綴般的將來――因為這也是遊戲中的一個設計理念。

就這層意義上而言，我對於優勝者當然也是抱有敬意。

可那個孩子――並不是優勝者。

。

（應該是擁有某種特殊的力量吧……）

。

長官明明是隻不過是在挖掘人才，卻在這方面也擁有卓越的能力。因為守護不夜城的八位親信都全部是由長官勸誘而來的人類。

其中也不乏可稱之為小孩子的年齡。

。

（真想知道呢……關於這世界上人類的多方面事情）

。

其實我也很想更多地調查那個孩子，不過我果然還是無法對長官介紹給我的客人做出失禮的舉動。

何況只要建好醫院――機會無論怎樣都會來臨。

。

「居然治好了那種傷口……不僅僅是魔王什麼都做得到，就連部下也是如此呢」

。

長官向我介紹的名叫露娜的少女，在一旁呆然地嘀咕道。

從這邊的角度來看，這邊的人類能夠使用魔法之類的力量才更令人覺得什麼事都做得到。

人，總是奢求自己所沒有的東西。就跟這種情形相似吧？

。

「不過我也向你道謝。雖然我不會對那傢伙說出這種話，可對於你還是很感謝的」

。

這番話令我不禁笑了出來。

這孩子天真的地方或許就跟不知恐懼的「茜」有點相似也說不定。

。

「原來如此。也就是說長官很中意吧――對於你」

。

「哈……哈！？這丶這種事我也很為難！超級為難的啦！」

。

你看，這不坦率的性格也很相似。

話又說回來，這孩子也擁有可愛的臉蛋。如果把她打扮成男孩子，不也完全能行嗎？

剪掉頭發，讓她穿上短褲試試看怎麼樣呢。

。

「我丶我們回來了！」

。

「惡，你的腳怎麼樣？沒有被變態魔王做了些什麼事吧？」

。

「你還真是無禮啊……某種意義上真佩服你耶」

。

長官邊說著，邊走過來。

接著，他把手放在我肩膀上――啪地拍了一下。

。

「悠，真是漂亮的醫療處理――――叫你過來實在太好了」

。

（嗚………）

。

對於這句話丶這隻手掌，我感覺到一串電流。

而且是從頭到尾猶如遭受貫穿般的異樣興奮與麻痺感。

。

（這是丶什麼感覺……！？）

。

如今一旦放鬆身心，這幸福感似乎都要令我溢出眼淚來。

身體丶心靈丶細胞丶甚至一根毛發都在因歡喜而顫動……。

為什麼呢……這種彷彿置身於獲得「全知全能的創造主」所認可的壓倒性幸福感之中。

。

「不丶不……能起到幫助丶那是再好不過……」

。

呼吸紊亂。

明明是自己的身體，我卻不知道自己的事情……。

迄今以來我也在工作上獲得長官的稱贊，可是深陷這種感覺卻一次也沒有。明明如此……這又是為何？

。

「今後也繼續輔佐我吧。就靠你了喔？」

。

「好丶好的……！」

。

（嗚嗚嗚……話都不能順利說出口……）

。

回過神來，連心臟都發出劇烈的心跳聲。

好奇怪。這樣――好奇怪！

。

為什麼我受到稱贊，受到依賴會這麼高興呢……。令人呆然的事情是眼淚都讓視野變得模糊不清。

如果這就是所謂的「幸福」，那麼自己迄今所感覺的一切幸福都全是毫無價值。

。。

（總覺得丶不對……不是以前的丶長官？）

。

「那麼，惡的右腳也治好了。抵達神都之後就舉辦盛大的慶祝吧」

。

「贊成！不過這一次輪到你請客了哦！」

。

「我丶我只要乾麵包也……」

。

「不行。露娜，抵達神都之後你就帶我們到最好的店家」

。

「可以是可以……你啊，應該真的有帶錢吧……？」

。

聽著三人之間的交談，我暫時保留這個疑問。

被觸碰到的肩膀，如今也依然發著熱。

根本就無法正常地進行思考。

。

（首先是建好醫院吧。長官還會再稱贊我嗎……）

。

。

惡的生存技能已經追加了《女主角》。

生存技能　――　女主角

遊戲中存在的稀有技能。

發動幾率雖說只有極其低下的3%，然而一切往技能持有者發動的攻擊都全部由九內使其無效化。

。

在遊戲中也有過持有此技能玩家實現大逆轉的案例，在堪稱是關鍵時刻所發動的必殺一擊慘遭無效化。從持有這個技能的角度上來看，九內便是救贖之神，從敵方的角度上看就等同於是地獄惡魔般的存在。
