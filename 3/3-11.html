3-11 第28話 惡魔VS魔王

惡魔　VS　魔王

一一一一一一一一一一一一一一一一

「給與這詛咒的土地……降下災難……」

。

馬基嘀咕完的瞬間，手中的逆十字架便長出滿滿的荊刺。

這些荊刺貫穿馬基的手掌，濺散出鮮血。

。

「老爸……對不起了……」

。

馬基思念著故鄉的父親。

雖說他是手藝不錯的鞋匠，不過受到不景氣的影響，導致生意急劇減少，最終被逼迫到失業的境地。

接著父親便整天沉浸在酒醉之中，在家裡施展暴力，馬基也每次都慘遭暴行。

。

年紀尚輕的馬基無法忍受這種生活，遂而刺殺父親離開了村莊。

在猶如人生跌落萬丈深淵般的落魄之中，撿到馬基的便是惡魔信奉者集團。

這種事在這個時代中無論在哪裡都會發生，也有常有的事情。

。

從逆十字架裊裊地冒出赤紅色的煙霧，包裹住馬基的全身。

這是將自己作為祭品召喚出惡魔的儀式。赤紅色煙霧不僅是包裹住馬基，就連他身後蹲伏著的500名惡魔信奉者的身體亦是如此。

。

像人類這種小祭品召喚出來的應該是下級惡魔。

然而，烏託邦交給他的逆十字架卻能引誘出階級更高的怪物。而且還是在中級惡魔之中也是極為強大的存在。

。

「惡魔召喚(Summon Devil)――――」

。

血霧化為一個形狀――把中級惡魔「伽尼巴魯」(Carnival)召喚出來了。

(颯:Carnival是歐美節日，狂歡節)

見狀，冒險者們發出悲鳴之聲。不可能會發生這種事。

這種怪物居然出現在人類的城鎮中，而且還是在擊退惡魔的「神都」！

。

「哎呀……居然被人類之類的給召喚出來，到底有何事？」

。

頭上長著兩支角，漆黑的頭發，宛若燒焦般的膚色，集齊醜陋而形成的臉孔。

光是看見便會削弱體力般的存在。

明明如此，卻只有身上的衣服誇張到不合時宜。金色的衣服上到處都刺繡著無數個閃閃發光的顆粒。

後背還背著一種奇妙的樂器。

。

「嘛，算了。畢竟好像有許多好吃的東西――《美獄之地》」

。

伽尼巴魯發動惡魔特有的技能，使周圍產生不可視的屏障。

悪魔喜歡先是封住退路，再來進行玩樂。

無論是殺死丶啃食丶亦或是玩得碎屍萬段，一旦逃跑感覺實在是不愉快。特別是像伽尼巴魯這種以人的悲鳴為歌聲，彈奏樂器的惡魔而言，等同於是「歌手」消失了。

。

冒險者一看見伽尼巴魯便驚慌失措地想要逃跑，然而已經為時已晚。

惡魔已然製作出固有的空間，平凡的力量根本無法擊破。

回過神來不僅僅是冒險者而已，就連商業地區的一角——多達幾千人都被困在伽尼巴魯所製作出來的空間裡。

。

「開玩笑的吧，為什麼伽尼巴魯會在這種地方！」

「給丶給我等一下……我們只不過是冒險者！」

「沒丶沒錯！我們可不是教會的人啊！」

「我不要……我不要被惡魔吃掉呀啊浮浮浮！」

。

冒險者們彈奏的音色令伽尼巴魯面露恍惚神色，瞇著眼睛。

對於「他」而言，這應該稱得上是愉悅的時間吧。

商業地區的人們也躲在家裡瑟瑟發抖。無以計數的「觀眾」對於伽尼巴魯而言亦即是最美妙的舞臺。

。

「這麼受歡迎真令人高興啊。那麼，哪隻小貓咪會為我歌唱呢――――噫啊浮浮浮浮浮浮！」

。

充滿愉悅的話語在途中就遭到阻礙。

不知從哪而來的小石頭把伽尼巴魯的角給打斷了。

過於激烈的疼痛令惡魔抱著頭尖叫起來。用來誇耀作為惡魔的榮耀和叡智的角被「小石頭」給擊碎了。

。

「是丶是誰乾的啊浮浮浮浮浮！你這個螻蟻之輩啊浮浮浮浮！」

。

伽尼巴魯拋棄迄今的語調，以赤裸裸的聲音大叫著。

這道聲音令冒險者們蜷縮著身體，不過投擲石頭的本人，聲音倒是安穩平靜。彷彿就像是在講述「明天會下雨嗎」一樣的聲色。

。

「比起人妖的語調――還是這種比較適合你啊」

。

■□■□

「是你這傢伙嗎啊浮浮浮！你這個狗屎混蛋啊浮浮浮！」

。

「……該怎麼說呢，沒氣質丶臉也醜丶聲音也汙濁。就連身上穿的衣服都好糟糕。根本沒一處可見人的，雖說作為藝人而言倒也不壞」

。

聽到「魔王」這一席話，伽尼巴魯的表情越來越扭曲。

他不會給這只螻蟻歌唱的「奢侈」待遇。而是應該立即把他的腦袋都捏個粉碎。因此他踏出步伐。

。

可腳卻停了下來。

。

那是因為他是優秀的惡魔，所以感覺到了本能上或者其他什麼有所察覺。

魔王靜靜地開口說道。

從中組織的語言與技能的發動――無論是什麼惡魔都會落荒而逃。

。

「實際上，跟像是你這種惡魔相遇已經是第二次了吧」

――生存技能「鬥爭心」發動

（使用者的攻擊與防禦+10%）

。

「雖說上一次的「那位」輕易地就死翹翹了」

。

――戰鬥技能「脫力」發動

（敵對者的攻擊-10%）

。

「可你又會是怎樣呢？」

。

――戰鬥技能「威圧」發動

（敵對者的防禦-10%)

。

「為了作為將來的參考，這次就讓我呈現出原本的姿態吧」

。

――戰鬥技能「無雙」發動

（使用者的攻擊與防禦+30%）。

「是叫做伽尼巴魯吧――「準備」好了嗎？」

。

說話的本人應該是沒有注意到吧。

從他的身體散發出異樣的靈氣，甚至都引起伽尼巴魯的身體越來越嬌小的錯覺。

。

反之他本人身體充滿著僅僅是觸碰就令人爆炸般的危險事物，原本就已經具有壓倒性的屬性值，藉由技能補正又得到爆發性的提高。

。

正所謂是單純的普通攻擊――――就連神也可弒殺的狀態。

。

「等丶等等……等一下！美妙的先生！是丶是我錯……」

。

「哎呀，忘了一件重要的事情」

。

魔王這句話令伽尼巴魯找到一縷希望。

這種猶如惡夢一樣的狀況本來是不可能發生的。因為作為強大的惡魔而聞名遐邇的伽尼巴魯――被「單單的人類」所壓制。

。

「就丶就讓我來幫你那件重要的事情吧！無論是什麼事情我都做！」

。

「基本態勢丶變更――――《戰鬥態勢》」

「應戰態勢丶變更――――《迎擊態勢》」

。

或許是沒聽見伽尼巴魯說的話吧，魔王變更了基本態勢。

這種態勢雖會令發現敵方玩家的幾率大幅下降，不過卻給與全屬性值的補正。在確信對手身在這個地區，並且不想讓對手逃跑時才能發揮最大的效果。

。

相對的，應戰態勢也變更為完全型迎擊態勢。

這邊也同樣是給與攻擊與防禦的補正，不過最大的特徵是極容易遭遇敵方玩家這一點。

打造出必殺的態勢，做好準備一擊獵殺敵人。這就是魔王原本的戰鬥姿態。

。

可伽尼巴魯動彈不了。

也不可能動得了。

到底有誰能對「這種對手」做出攻擊行為呢。

。

「不要……居然在這種地方……被人類殺掉什麼的！」

。

「你也是殺過許多人走過來的吧？那自己也體驗一次死亡的經驗吧」

。

魔王毫無取巧地接近伽尼巴魯，抓住他的頭。

隨即如同拋飛棒球一般，往高高的空中丟出去。

巨大的惡魔以可怕的速度飛到天空中，這實在無法認為是現實中的光景。

商業地區的人們也吞了口沫注視著事態，不過在眼前發生更冷人驚訝的光景。

。

魔王手中散發出白色閃光，釋放而出的索多瑪之火在刺中伽尼巴魯的瞬間便產生震撼大氣的劇烈沖擊，地獄的爆炎彷彿要染紅夜空般在神都的上空擴散開來。

。

令人還覺得是伽尼巴魯的肉片與碎渣在周邊傾注而下。

一時之間，誰也無法動彈。

光景過於驚愕，以至於連發出聲音都感到忌諱。

。

「正如所料――――骯臟的煙花」

。

魔王咧嘴一笑的瞬間――從凶殘的惡魔手中解放的冒險者們也爆發出喜悅了。

被困在商業地區的人們彷彿也受到他們的大歡聲所引誘，都陸續走到大街上互相擁抱著，慶祝彼此平安無事。

還以為肯定會慘遭殺害的人也流出淚水，互相以拳頭相擊。

。

「魔王大人萬歲！」

「你啊，真是不得了啊！」

「居然把那個伽尼巴魯當做小孩子耍……在故鄉有英勇事跡可以說咯！」

「老爺，今天就讓我盡情奢侈一把吧！」

。

聽到周圍的歡呼聲，魔王難得狼狽不堪地環顧四周。

應該是沒有預料到這種展開吧。然而他們是從壓倒性的恐怖之中解放出來，所以做出這種反應也是理所當然圧倒的。

。

「嘛丶嘛……只要由吾來做，那種程度的事情簡單至極」

。

魔王心神不定地為香煙點上火，仰望著夜空。

這時候，他應該是回想起自己趁著興頭跟惡魔戰斗的事情而感到羞恥了吧。然而周圍人卻不會如此認為。

他們誤以為真，連說話聲都甚至可以掩蓋的歡呼聲響遍四周。

。

「那丶那麼……吾要走了。畢竟其他地方似乎也引發騷亂了」

。

魔王立馬打算要逃跑，不過卻有一位魔法師站在他面前。是白色魔法師雪風。

。

「你好像是……」

。

「……這已經是第二次被叔叔大人拯救性命。我想報恩」

。

「沒這必要。對於吾而言（賺取SP）就是最重要的事情」

。

「……叔叔大人」

。

這番話令雪風面泛紅暈，身體也扭扭捏捏的。

雖然感覺好像漏了些什麼重要的句子，不過魔王只是想盡早離開此地。

然而雪風的少女心一直呈現上升趨勢。雖說是男人。

。

「……因為我的錯，令叔叔大人兩次陷入危險的境地」

。

「這構不成猶豫的理由。這也就是說，（SP）是多麼重要的事情」

。

一樣的回答已經完全把雪風攻略完畢。

原本白皙通透的臉蛋，如今也隱約變為櫻花色。成為一位完美無瑕的美少女。雖說是男人。

。

「……我希望叔叔大人把名字告訴我」

。

「叔叔……吾丶吾名為九內伯鬥」

。

無數次重復這種「叔叔大人」的稱呼，這一次令魔王心情低落。

雖然他拼命地小聲嘀咕著「裡面的我還很年輕啊……」，不過並沒有傳到爆發出喜悅的周圍。

。

（話是這麼說，從這種小孩子來看，三十歲也已經算是大叔了嗎……）

。

魔王邊想著這種苦惱的事情，邊仰望著月亮。

這種略帶憂鬱的表情，不禁令雪風加快心跳。

這位魔王雖說可怕，可表情卻絕不壞。那是一流的惡黨才會展現出的，可說是一種「色情」吧。

。

「九內！　九內！　九內！　九內！　九內！」

。。

回過神來，周圍的冒險者已經在連續呼喚魔王的名字。

該說是吊橋效應嗎，又或者只是情緒高漲嗎。周圍人遠離死亡的喜悅之情再加上喝酒使得呼喚聲逐漸更大。

連街道上酒店的老闆都拿出酒作為慶祝。

。

（這是什麼鬼！這可不是開玩笑的吧……！）

。

雖然是想消除惡劣的評價，不過達到這種程度已經是意料之外。不如說這種狀況幾乎已經是懲罰遊戲。

魔王跳上屋頂，彷彿逃跑一樣往別的方向飛奔而去。

。

「那就是魔王丶嗎………」

。

蜜康無力地嘀咕道，原地癱坐在地。

一瞬間就令500名左右的惡魔信奉者屈服，甚至連中級惡魔也能哼著鼻歌把他炸死的存在。

這已經不是人類能夠應付的層次。

。

「……我跟叔叔大人已經結成紅線了」

「等丶等一下！雪風……你是被欺騙了啦，一定！」

。

「……蜜康，嫉妒？」

。

「誰會嫉妒啊！我的喜好呢，是像傳聞中銀色龍人一樣深愛正義的人啦！」

。

「……真下流。喜歡深愛性技的人什麼的」

(颯:日語正義與性技同音)

。

「感覺話裡的意思有點微妙的不同啊ーーー！」

。

如此這般，商業地區平安無事地祛除了威脅。不過其他地區依然還在暴亂中。

並且有一位小丑在接近「亞露緹米絲」。

在不知道那邊有一位將忠誠奉獻給魔王的「恐怖魔女」情況下。
