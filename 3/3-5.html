無情的侵略

一一一一一一一一一一一一一一一一

（有種已經接近都市的感覺啊……）

。

「我」悠閑地欣賞著景色，邊想著這件事。

隨著離神都的距離越來越近，眼中可見的乾涸大地也逐漸變少，綠意盎然的植物則是漸漸增多。

道路寬度也變得更長，時而與過往的馬車相互交錯。走在於道路上的人們也越來越多，沿著大道建設而起的村莊似乎也相當熱鬧的樣子。

。

―――然而，唯獨那座村莊截然不同。

。

整座村子由巨大的柵欄所包圍，可卻幾乎沒看見任何人的身影，彌漫著一種異樣的氣氛。

應該說是廢村吧？

村子規模頗為巨大，可這反倒是令悽涼的樣子更加引人注目。

。

「露娜，那是什麼村子？總感覺真沒有活力」

。

「…………我的村子」

。

「哈？」

。

「所以說，這裡是我的領地！」

。

喂喂，聖女在國家的立場上不是很偉大嗎。

這麼荒涼的廢村居然是領地……。

。

「看起來還真是冷清，這樣可以？」

。

「我又沒有興趣治理領土。而且教會也派遣人員過來管理了，所以根本就沒有我出場的餘地」

。

嘛，實際上我也不覺得露娜能治理好領土。

如果實施離譜的重稅，結果就會是引發叛亂吧。我很容易就想像得到她在處刑臺之上慘遭打屁股。據我來看，這傢伙雖說有魔法的才能，不過單純只是所謂的「神輿」吧。

(颯:神與是指日本祭祀活動時，由眾人抬舉的神橋)

。

她雖然受到眾人的抬舉，可卻完全被疏遠於權益受惠以及有利可圖的土地不是嗎？

本人也對這方面不感興趣，可謂是容易操控的人偶。

我把想到的事情都藉由《通信》傳遞給悠，進行探討。

。

《我是這麼認為，你覺得呢？》

。

《我也有同感。恐怕她被貴族以及教會的上層部巧妙地利用，把難以處理的土地贈與給她》

。

《哼。既然如此――縱使舍棄的土地被吾等所回收，他們也無法提出意見》

。

《要利用這座村莊？》

。

《既然是聖女的領土，也沒多少人敢從旁干涉吧。正好方便》

。

雖然我說著這種話，可其實最重要的理由是已經沒錢了。

大金幣似乎擁有相當高的價值，不過我不認為連土地都能夠買下。而神都又是首都，物價也會較為昂貴吧。

反正建設據點丶拆掉據點都只是一瞬間的事情，不行的話再到其他地方建設就好了。

。

「露娜，我想借用這座村子的一部分土地」

。

「誒？？」

。

■□■□

。

走進村子裡，一位外貌有意思的人物便映入眼簾。

頭上附帶著疑似兔兒的東西。起初我還以為是在做什麼打扮，不過聽露娜所說這似乎是名叫《兔人》的種族。

。

「嘛，兔兒的人類吧……」

。

「聽說這個種族在以前就受到智天使大人的喜愛。所以即便人數不多也聚在這裡生活著」

。

「喜歡丶呢……雖然在我看來就好像是受到隔離哦？」

。

「因為聖光國基本上……都討厭人類以外的種族」

。

我感覺露娜的神情上似乎夾雜著一點寂寞之情，不過現在不是打聽這件事的時候。

而是在聖女的村莊裡建造起醫院，多多治病令我的評價轉為好評。

就讓我盡情地利用露娜的名聲以及立場幹。

。

「魔王大人，您打算要在這裡做什麼？」

。

惡一步一步地往這邊走過來。看到她這副樣子，我心中有種無法言喻的喜悅湧現而出。

惡――她已經可以獨自走路了。

雖然總覺得自己有種成為長腿爸爸的心情……。

(颯:「長腿爸爸」出自於美國1912年女性作家所出版的兒童文學作品，有興趣的童鞋可以百度百科)

。

「也沒啥大不了的，只是想著要在這裡效仿醫生的工作」

。

「啊，是悠大人吧！如果悠大人成為醫生的話，我覺得全國各地都會有人過來！」

。

旅客集中便意味著「消費金錢」。不如干脆在醫院的隔壁建造一間《溫泉旅館》吧？

大致上，病人是以老人居多，所以應該會喜歡溫泉才對。

畢竟從惡的樣子來看，浸泡在熱水裡是相當奢侈的事情。

。

「長官，這附近一帶您覺得怎麼樣呢？」

。

「地方有點小幹。我還想要旁邊建造溫泉旅館來著」

。

「溫泉嗎，那真是美妙的事情呢……」

。

沒錯，基本上女人也喜歡溫泉。

事到如今還是以醫院及溫泉這種雙收入為目標比較好。如果搭配香皂販賣出去，或許還可以期待這個不錯的銷路吧？

無論是我還是露娜，名聲都會變好起來。而使用者也會展露笑容，可以說是誰也不會有損失的完美計劃吧。

。

「這丶這是露娜大人……歡迎您勞駕此地。請問，這些大人是……啊！？那丶那位大人是通緝令上的……！」

。

「對，他是魔王哦。如今是我的「協力者」――」

。

露娜挺起飛機場，很偉大似地說道。說是協力者其實是苦肉計。

看樣子我的長相已經作為通緝令傳遍於神都之中。雖然正確的來講，似乎是「自稱魔王的男人」……。

。

（得想辦法讓傳聞的方向改變……）

。

比如他雖然擁有魔王的力量，可其實是位親切的人。

比如他盡管是魔王，卻是個好人。

最終再來個他是位經營溫泉和醫院的大富豪之類的。。

（雖說連我自己都覺得非常困難……）

。

不過這邊在實際上也的確是有建造醫院以及溫泉。

一個一個的事實堆積起來，遲早會解開人們的誤解。成為受萬人歡迎的魔王才對……喂，成為魔王我是要搞什麼鬼。

。

順便一提，《野戰病院》是呆在裡面就會提高氣力回復速度的據點，而《溫泉旅館》則是具有提高體力的回復速度。

這裡遲早會成為這世界上獨一無二的地方。我為了確保地方，立即與教會派遣而來的代理進行交涉。

。

如果可以我也想把教會的人都趕出去。露娜姑且不論，我完全不覺得什麼聖堂教會能夠成為我們的友方。

要是他們提出遞交幾成的銷售額之類的話來，也實在麻煩死了。

。

「你是教會派遣過來的人吧？今後這座村子由我和露娜來治理。你就給我回到那什麼教會，把我們的意思傳達給他們」

。

首先第一步是從上級開始――

。

「什……可丶可這件事……如果不跟上級商量一下的話……」

。

「還上級呢……這在道理上行不通吧。身為這塊土地本來領主的露娜說要發揮才幹。你和教會的上級這些亂七八糟的人物想要主張自己才是這座村子的正當領主嗎？」

。

就這樣，一氣呵成地施加連珠炮的言辭――

。

「不丶不是！沒有這種事……這座村子是露娜大人的所有物……」

。

「哼――剛才你口中似乎就已經說出「答案」了。那麼，你就趕緊給我動身吧」

。

男人慌慌張張地走進家裡，騎著馬飛奔而出。不如說是逃跑了。

可惡……就那麼可怕嗎，我的臉……。

算了，這樣礙事的人就消失了。這下子就能盡情地施行這邊的計劃。

看完這一連串對話的露娜，面露驚呆的表情嘟囔道。

。

「你在這種時候還真是油嘴滑舌呢」

。

「別擔心。只要交給我來辦，這座村子絕對會發展起來」

。
