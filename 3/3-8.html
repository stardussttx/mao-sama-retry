晩餐會

一一一一一一一一一一一一一一一一

――神都　夜晚

。

（那麼，旅館房間也訂好了……）

。

「我」使用我所持有的大金幣在神都訂好了房間。

這間高級旅館似乎相當有名。

如果考慮到貧匱的所持金，還是便宜的旅館比較好。可是若要大張旗鼓地宣傳醫院和溫泉，對外展現出大富豪這種形像感覺比較好。

因為即便是同一句話，窮人所說的跟有錢人所說的說服力是截然不同。

。

「然後呢，店名是叫做亞露緹米絲嗎」

。

露娜介紹給我的店以這國家而言還真是挺正經的店名。雖說光是聽到正經的店名就會吃驚也是蠻過分的。

。

「是啊，那是像我這種高貴的人才可以關顧的店」

。

「還高貴呢……這件事先不提了，為什麼連你也要住在旅館啊」

。

「也丶也無所謂吧……我要住在哪裡都隨我意吧！」

。

隨你妹！居住費根本是由我來付吧！

我不禁想要捏住那張嘴，不過又不能在大街上引來奇妙的眼光。本來我們就已經引人注目了。

聖女丶自稱魔王的男人丶身穿白衣的美女丶身裹白色禮服的公主殿下。

這到底是什麼一行人，光是走在一起就已經會引發騷動了。

。

《長官，很懷念呢……這種氛圍》

。

《……是啊》

。

不，抱歉。

你到底在指什麼事情，我完全搞不明白。

。

《就跟臣服於吾等武威之下佔領地區的可悲民眾十分相似。那時候靜把丟小石頭過來的小孩子做成了「不倒翁」，這件事就好像昨天才剛發生過一樣》

。

《哈丶哈哈……》

。

說起來，在遊戲中的小故事好像是有寫上這種事來著……。

從親信的角度來看，那也是成為了「現實」吧。靜在這方面的事情比較多，所以真的是危險的人物角色。

明明就是為了演繹出大帝國的毫無慈悲而寫上的小故事，可卻當做是現實中存在的事情，想想身體都顫抖。

。

「魔王大人……像我這樣的人去那麼厲害的店也沒問題嗎？」

。

「什麼都不用擔心。現在你看起來就像是公主殿下哦」

。

「公主殿下什麼的……」

。

看見面乏紅暈丶羞赧的惡，我也冷靜下來。

這孩子逐漸成為我的精神安定劑了吧……？

。

（話又說回來，這就好像是夢一樣的都市呢……）

。

街道上遍佈的路燈猶如在點綴著夜晚一般，都散發出形形色色的光芒。

規模可說是夜燈美景。

大路旁也設有噴水池，其周圍能看見多對情侶在相互傾述著愛意。

。

（就連路人的打扮也完全不同）

。

路人的服裝整潔是理所當然，大部分都具有打扮意識。發型應該也有講究吧，從中能感覺到花在這方面上的心思。

白天的時候還沒有這種感覺，一到晚上就跟涉谷晚上的感覺差不多吧。

。

路旁懸掛著藉由使用各種魔石而呈現出五花八門的招牌，盛裝打扮的女人站在酒店前向過往的路人打招呼。

以這種狀況來看，彷彿就像是什麼歡樂街一樣。

。

（白天是神都，晚上又是別樣的樣子嗎……）

。

我覺得挺有意思。

看來這裡似乎不只是表面冠冕堂皇而已。

我不討厭像是寺廟那種靜謐的氛圍，可也絕不討厭像這種精神奕奕的熱鬧城鎮。

。

「露娜，這城鎮真有趣」

。

「是丶是嗎？你會稱贊什麼的還真是意外呢……」

。

或許是沒想到吧，露娜面露驚訝的神色。

這不是什麼諷刺嘲諷，而是掌握住某種什麼的感覺。

。

「這比起天使還是什麼所說的還要更加簡單易懂。人類的慾望如實地展露出來――這裡就是這國家的原點吧」

。

通過努力，磨練自己，終於得到力量與財富。

於是這種人所迎來的場所便是這裡。

比起——在死後的世界過得幸福之類的可疑事情更具有現實感。

。

「慾望丶你啊……啊，到了呢。這裡就是亞露緹米絲哦」

。

「哼――那麼，我們進去裡面吧」

。

■□■□

。

一走進店裡，那又是不一樣的世界。

從顧客層次上便一目瞭然。

身穿似乎昂貴的衣服，把葡萄酒和肉送進嘴裡，優雅地談笑著。感覺有點像是社交界。

。

「露娜大人，歡迎您今日的到來――」

。

「嗯，今天也拜託你準備不錯的料理」

。

疑似管理員的人不問是否進座，直接過來打招呼了。不愧是聖女，看來頭銜真不是吃鈣的。

這頭銜在今後似乎也能為我起到巨大的作用。

菜單由對方決定，我們各自點了自己的飲料。我當然是酒。

。

「那麼，為了慶祝惡的腳治好了，我們乾杯吧――」

。

面對陸續搬運到餐桌上的料理，惡已經目瞪口呆，我也緊張得心跳不已。

然而「九內伯鬥」的身體應該是熟知這種餐桌上的禮節吧，以宛如自小廣是貴族般的嫻熟動作進行用餐。

。

「惡，不用在意什麼禮節，想怎麼吃都可以」

。

「好丶好的！」

。

下一次應該去能夠放輕鬆吃飯的店，而不是這種形式多多的店。

我也放鬆肩膀，想要心情愉快地吃飯。

。

《雖然料理不錯，不過跟我們的世界相比，還是劣等太多了呢――》

。

《嘛，這也沒辦法》

。

遊戲中提供的食物大多是來自於現代的日本。如果替換成現實的事物，那當然是很美味吧。

。

提到日本，那可為了製作快餐都要對味道拘泥到從地球另一邊運輸材料的國家。

無論是冷凍技術丶加工技術丶調理技術都遠遠領駕馭這個世界，跟我們的世界相提並論都有些過分。

。

（那麼，說到慶祝……果然還是需要糕點吧）

。

遊戲中當然也存在所謂的糕點，不過卻不是能直接獲得的道具。它需要作為加工道具的《食材》。

根據食材的使用，可以製作出四個種類的回復道具。

。

。

如果再加上持有生存技能《料理》，選擇項目也會更多。

。

。

就是這樣。

遺憾的是九內沒有料理的技能。這一點只能讓技能獲得的權限恢復了，至於召喚持有這技能的親信這點就先保留。

。

「上級道具製作――《食材》」

。

我在餐桌下製作食材。

雖說是食材，可也不過是個漂亮的白色的球。正因為不知道要變化成什麼樣，所以才會是這種形狀嗎吧。

畢竟是在這種地方，稍微掩飾一下吧？

。

「道具加工――《草莓果餡餅》」

。

雖說會令人覺得反感，不過我還是用左手打響指，進行宛若魔法般的演出。

食材逐漸發生變化，草莓餡餅也隨著光輝而現身。

從一旁的角度來看，這才真是「魔法」吧……。

。

「哇啊浮浮浮！　魔王大人，那是點心嗎！？」

。

「這丶這是什麼啊！非常可愛呢！」

。

無論是哪個世界的女孩子果然都喜歡糕點吧。

消耗20點SP也算是有所價值了。其實也不是我親手製作的，不過卻體驗到一種成為一流糕點師傅的心情。

。

「惡，這是我給你的禮物――悠，適當地切分吧」

。

「好的，長官」

。

畢竟能回復50氣力，長途旅行積累的疲勞也會消失不見。正所謂是一石二鳥。

在每個人面前放著切成小塊的果餡餅，她們都畏畏縮縮地吃進嘴裡。

對於味道我雖有自信，不過我不怎麼喜歡甜的食物……。

。

「好甜！好好吃！好可愛！」

。

「不行了呀啊浮浮！好美味！臉都忍不住鬆弛了！你是使用什麼魔法了嗎！？」

。

「吃進嘴裡都變得幸福起來了呢，長官……呼……」

。

「是丶是嗎……那就好」

。

這些傢伙會不會太吵了？

再怎麼說女孩子喜歡甜品也太……而且，連悠你也是這樣……。

。

「還真是相當高興呢――小露娜」

。

「誒，瑪妲穆！？」

(颯:瑪妲穆=英語中的夫人)

。

往那邊一看，身穿一身碧色的禮服――看起來就富裕的老闆娘過來跟露娜聊天。其手指頭上戴著令人感到堅牢的戒指。

而且還是十個戒指，每根手指頭各一個。

應該是露娜的熟人吧，態度還真是相當熟稔。反觀露娜那邊或許是不擅長應付吧，難得眼神會漂移不定。

。

（誰啊，這超有錢的女人……全身可都是在發光喔）

。

這老闆娘到底是什麼人――我覺得應該要暗中試探，因此行動起來。

明明接下來要開始創辦「事業」了，絕不能在莫名其妙的地方栽跟頭。

我從席位上站起來，優雅地行了一禮。

。

「您就是瑪妲穆。初次拜會――本人名叫九內伯鬥」

。

(颯:第一詞為英文的炸蝦，第二個為蝴蝶)

。

（一發藝人嗎！）

(颯:指單憑一個動作就能令全場鬨笑的搞笑藝人)

。

這名字與其說令人噴出來，不如說是怒氣逐漸湧起。

父母到底是有什麼想法才會取這種名字啊！？

。

《露娜，這個瑪妲穆是什麼來頭？》

。

《呀！？頭腦裡響起了魔王下流的聲音！？》

。

《下流這句話是多餘的。趕緊給我回答》

。

《她是貴族夫人的中心人物哦……。因為在貴族之間臉面非常廣，影響力也很巨大，所以是個恐怖的人……》

。

正所謂是富有餘閑的夫人們的女王。

如果能從中巧妙地進行推銷，或許可以期待無法想像的效果。畢竟是沒有網絡存在的社會，影響力巨大的人說話的份量也應該很大吧。

。

「夫人――您若不介意，能否與我共處一席」

。

「等丶等一下啦，魔王！」

。

「哎呀，居然受到「傳聞中的魔王大人」的邀請……真是非常刺激呢」

。

這女人不簡單啊。

明明就知道我的事情，卻擺出佯裝不知的嘴臉接觸這邊。

雖說是多餘的話，她的能力的確能掌握住貴族的夫人們。年齡恐怕超過五十歲了吧，雖然不知道具體多少。

從她具有猶如小山一樣的巨大體格，由此可見沒有過多地做過運動。

。

（來接觸傳聞中的本尊嗎？你，反而會被我吃掉喔――）

。

如此這般，表面上和諧的談笑已然開始。

可同時，其暗地裡。

惡魔信奉者已溶入黑暗之中，一直等候著暗號響起之時――

。

一部分情報已經公開了。

。

除了加工道具以外還有其他道具存在，舉例而言以《武器素材》和《非合法物》等最具代表性。

前者是使用完便會產生武器攻擊力+2效果的道具，後者則是能令各種屬性值提高。

由於雙方都具有嚴重影響遊戲平衡性的可能性，因此每種需擁有對應的《秀才》以及《非合法物》這類技能，這些技能的獲取難度並不低。

。

其他還有像《変化道具》這種一使用就會隨機產生的效果以及主題內容的道具。

《御神簽》和《占卜雜志》都存在善惡的結果，一使用搞不好有屬性值下降效果，因此使用時需十分謹慎。

《福袋》《小功箱》《豬型儲錢罐》《聖誕老人的袋子》等道具則大多會出現貴重物品，不過偶爾也會出現垃圾物品，所以經常會令玩家感到失望。
