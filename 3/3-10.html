魔王的冰山一角

一一一一一一一一一一一一一一一一

神都分為四大塊區域。

第一個當然就是聖城。

這座具有歷史性質丶周邊張開幾個神聖結界的城堡使得惡魔這種存在完全無法靠近。即便是實力相當高強的魔族也難以闖入其中吧。

。

接著是聖堂騎士團的本部和眾多貴族宅邸歸布的高級地區，以及由聖堂教會的本部和一般市民的住宅所構成的一般地區。

最後則是聚集多數商會和路攤店的商業地區。順便一提，商業地區裡面也有冒險者們關顧的公會和歡樂街。

。

惡魔信奉者除了聖城之外，對其餘的地區都同時開始發動攻擊。他們花費漫長的時間挖掘地下道，潛入神都的最下方。

於各個地區掘開大洞，從中同時湧現出身穿同一種裝束的惡魔信奉者。

。

在神都掘開的三個巨洞――

宛如某處的「奈落」令人感到毛骨悚然。

■□■□

。

「原來如此，同時攻擊嗎……還挺能幹的嘛」

。

「我」離開店裡之後便立馬跳上屋頂，從高處環視整座神都。

在我自己所在的地區也回響著悲鳴與怒號，抬頭凝視遠方，另外兩個地區也開始行動起來了。

。

應該是打算把能夠應對這場供亂的人都分散開來吧。

同時多發恐怖襲擊——腦海里浮現出這一詞語。雖說古典，可也是有效的手段。

我回到店裡立即對悠下達指示。其實也沒必要這麼急，可我想盡可能的把SP……不對，不想讓敵人逃跑而已。

。

「悠，你就保護好這裡的人――我要去鎮壓這場供亂」

。

「瞭解，長官」

。

「等丶等一下啦，別自己擅自決定！我也要去！」

。

「露娜，這場供亂的目的有可能就是你。所以你跑來跑去的想保護你也保護不了吧」

。

「保丶保護……就丶就算不用你保護我也……」

。

露娜紅著臉陷入沉默。

這傢伙……真的好容易應付，這樣沒問題嗎？看到她對於這種毫無免疫力的性格，就算是我也擔心她了。

言歸正傳，如果露娜用魔法打倒敵人的話，我就不能賺到SP了。

。

「惡你只要待在悠的身邊就不用擔心。慢慢享受晚餐就可以了」

。

「好丶好的……可丶可是丶魔王大人沒問題嗎？」

。

惡抬著頭以擔心似地表情仰望著我，還握住我的手。

總覺得有種爸爸被小孩子擔心的心情……我的年紀可沒那麼大，而且還是單身喔？

。

「我早說過是餘興了吧――這就跟飯後的運動差不多」

。

實際上我滿腦子只想著賺取SP。

如果對手實在不妙，到時候還真是隻要逃跑就沒問題。從戰鬥中逃脫的手段多得是。

。

「瑪妲穆也請務必繼續用餐」

。

「您真的是位厲害的自信家呢……不，若要借用您的話來講，應該就是將「全部化為現實」吧？」

。

「您所言極是――對於那幫傢伙而言，最大的不幸便是我身在此處這件事吧」

。

我只說完這些，便走出店外。

明明對手不妙我就會想全力逃跑，還真虧我能想說什麼就說什麼。連我自己都感覺心臟已經長出毛了喔。

。

總之在走之前先囑咐一下悠……那傢伙要是放任不管，都不知道到底會做出什麼事情來。

像是我一回來，所有襲擊者都已經遭到解剖什麼的，感覺還蠻有可能的。

。

《悠，吾等為了知曉――必須得處於容易行動的立場之上不可》

。

《意思是殘虐的行為要有所節制，獲得正面評價？》

。

《依舊還是理解迅速啊。那就拜託你了喔，悠――》

。

《好丶好滴……》

。

嗯？

感覺最後好像咬到舌頭了吧？

不不，只有悠是不可能發生這種事的吧……。

。

「那麼，首先要到哪邊去呢」

。

看向街道，人們不知逃向何方的情景映入眼簾 。

這是――TV裡面經常可以看見的光景。

。

因為我在起居室宛如「置身度外」地看過和平的城市突如發生恐怖襲擊，滿身是血的人們被搬運到救護車之類的情形。

不對，實際上就是置身度外。

如果是家裡附近發生的事情那倒另當別論，可如果是隔著大海丶距離遙遠的外國，無論發生規模多麼巨大的恐怖襲擊，我都沒有一種現實感。

。

（不過在這裡就不能再置身度外了吧……）

。

露娜還是具備保護自己的手段，不過惡即便是一塊小碎片飛過來都應該會身受重傷吧。

明明難得治好了腳，如果還在這種暴亂中受傷的話，都不知道我到底是為了什麼才走到今天。

。

（而且，又是你們嗎……）

。

遠處可見熟悉的黑服裝。其手上各自都握有凶器，其中也有人高舉著法杖。

那幫傢伙才是把自己召喚到這世界上的元凶。

我不知不覺中往緊握住的拳頭施加力氣。

。

「到處來回亂串的。如果真這麼想要遇見「魔王」的話，那我就讓你們見識一下」

。

我避開人群雜亂無章的街道――

從屋頂飛躍到另外的屋頂，開始行動起來。

。

■□■□

。

――商業地區

。

馬基率領著這地區的集團，如今難掩焦躁神色。

盡管起初實行突襲，不過卻有許多人從冒險者公會出來迎擊。當然，對於馬基等人而言他們也是計劃中早已預料到的，然而其中就屬兩位實力極為突出，甚至都打亂了計劃的進行。

。

「哈啊浮浮浮浮浮――！《剛擊》」

。

擁有褐色肌膚的女劍士如今也揮下那把需抬頭仰望的大劍，把三個惡魔信奉者給砍飛了。

其表情上滿是疲憊之色，令人覺得她遲早會用盡力氣。然而再繼續無動於衷，我方受到的損傷也不可輕視。

。

「還沒完還沒完！《剛元武》」

。

女戰士讓大劍旋轉一週，周邊隨即迸發出強烈的沖擊。

黑衣人的身體如同紙張般慘遭撕毀，惡魔信奉者的戰力也逐漸減少。馬基終於也按耐不住，下定決心前往前線。

對他而言，在這裡浪費氣力也是他所始料未及的事情。然而戰局已經容不得他再繼續坐視不管了。

。

「低賤的冒險者們……！《冰刃/Ice Slash！》」

。

馬基所釋放的刀刃是「水」的上位「冰」。

他若是使用「水」魔法，最高能使用第三魔法，算是具有相當實力。雖說其上位的「冰」達到第二魔法就已經是極限，可即便如此也能說是與常人分隔一線的存在。

。

「汝，冷凍蜜康也――――《雪之戀人/Snow Kiss》」

。

然而，淡白色的魔法師像是在保護女戰士一樣詠唱著防禦魔法。

。

「幫大忙了，雪風！」

。

「……勝利的救助。嘴上的戀人」

。

魔法師嘀咕著什麼意義不明的話，不過女戰士無視她更加沖進敵陣。

被稱之為雪風的少女也再一次為了釋放魔法而開始詠唱。

一般來說，施展魔法之後直到下次施展之前都需要相當長的間隔，不過她的力量應該是相當優秀吧。

。

――連續詠唱

。

「……以海濱捕獲《冰之手/Ice Hand》」

。

「你的這裡，硬梆梆……《冰槍衾/Ice Splash》」

。

雪風最初釋放的魔法令地面產生出無數的冰之手，抓住惡魔信奉者的腳裸。

在他們無法動彈之時，冰之槍立馬殺到。惡魔信奉者在可謂是毫無慈悲的連續詠唱下，都陸續鮮血四濺地倒在地面上。

雖擁有一副可愛的臉蛋，可做的事情卻殘酷無比。

。

「別害怕！數量上是這邊佔據優勢……直到氣力用完為止都給我繼續進攻！」

。

馬基的命令讓後方的惡魔信奉者都蜂擁而至。

接下來便是完全的亂戰。

冒險者與惡魔信奉者展開交戰，雙方的損傷也飛躍性地提高。

。

藉由偷襲的惡魔信奉者這邊果然優勢比較大，而戰局則形成冒險者遭到壓迫的展開。

從冒險者的角度來看他們的行動終究只是源自於「自我保護」，已然是敗意逐漸湧起的狀況，而反觀惡魔信奉者則是「敢死隊」。

。

他們在今天已經做好死在這裡的覺悟。

對於冒險者而言，他們根本沒有義務為了守護神都而殉命。

因為那應該是聖堂騎士團和聖堂教會的職責。冒險者之中已經有人想要逃跑，也有人環顧著四周意圖逃之夭夭。

。

「有點不妙呢……怎麼辦，雪風」

。

「抱有處女死掉會死不瞑目」

。

「在這種狀況還真虧你能說出這種話呢……而且你根本是男的吧」

。

「蜜康無知。偽娘是――」

。

在這番話說完之前，一位惡魔信奉者便用劍對準雪風的臉上揮下去。

然而，在揮下去之前就響起一陣奇妙的聲音。

。

那是一股切開大氣的聲音。隨即是響起一聲宛若干枯的樹枝折斷般，與戰場毫不相稱的聲響。

明明是在怒號與武器相擊的聲響之中，那道聲音卻毛骨悚然地傳到所有人的耳朵裡。

。

那支打算要揮下的手腕，應該是被某種沖擊給折斷了吧。

拿著劍的手腕以可怕的速度扭曲到緊貼背上，連同身體也猶如陀螺般以驚人的速度旋轉起來。

突然出現於眼前的「人型陀螺」令所有人停下動作，目瞪口呆。

當陀螺停止下來時——已經翻著白眼趴在地面上了。

。

「――――這不是為吾展現出一幕與小丑相稱，精彩絕倫的表演嗎」

。

抬頭仰望，全身包裹在漆黑外套裡的「魔王」便在屋頂之上。

其手上握著幾顆小石頭，並且丟向空中把玩著。剛才的表演也是因為投擲石頭而產生的吧。

僅僅是被小石頭砸到，人類的身軀就猶如陀螺旋轉起來。

。

實在過於荒唐無稽。

這樣的存在――才正是所謂的「級別不同」吧。

。

「叔叔大人……又再次相會了，好美妙」

「魔王……！為什麼在神都！？」

。

魔王並沒有回應這兩人，彷彿這麼做是理所當然般從屋頂上跳下來。毫無任何躊躇，立馬展開下一次行動。

襲擊這塊地區的惡魔信奉者總共高達500名的規模，然而在魔王面前未免也太無防備了。

。

「究竟是站在誰的眼前？跪下――――《霸者》」

。

魔王的全身散發出赤紅色的靈氣，在揮下他的右手時，便吹起一股無屬性的暴風。

這是遊戲的戰鬥技能之一，名為霸者。給予使用者攻擊數值的三分之一傷害。

席捲而來的「全體攻擊」令惡魔信奉者迸發出鮮血。

。

然而，魔王的攻擊並沒有結束。在遊戲中戰斗的基本是收集技能，通過各自的喜好組合技能，進而形成「連招」。

。

通常攻擊達到一定的熟練度便可施展連擊，接著是屬性技能丶無屬性技能丶遺恨技能丶特殊能力。根據組合能夠造成天文數字的傷害，如果是知名人士，聽說連「神也可弒殺」。

。

「低賤的正確姿勢――――《開眼》」

。

接著是以左手呈水平面揮過，藍色靈氣便以扇型擴展開來，穿透惡魔信奉者的身體。

這是與霸者相反，給與使用者防禦數值的三分之一傷害。

惡魔信奉者接連不斷地跪倒在地，從口中吐出大量的鮮血。

。

「如今，便讓汝等償還――――《粉碎》」

。

從魔王的身體迸發出一陣黃色氣息，並化為巨大的錘子。當錘子從天空揮下來時，多達500名惡魔信奉者的身體便全身感受到眩暈般的沖擊。

。

這是給予對像現有體力十分之一傷害的戰鬥技能，一定幾率造成「骨折」異常狀態。惡魔信奉者遭受到「粉碎」，腰部和大腿都陸續骨折，彷彿要以頭著地一樣趴在地面上。

。

「不錯的姿勢――為了不再遺忘，盡管刻印在腦海里吧」

。

三種無屬性攻擊肆虐全場――

舉目所見之處，惡魔信奉者都悽慘地倒在地面上。其全身到處都骨折了吧。

其中既有擊潰成宛如青蛙的人，也有許多人手腳彎向奇怪的方向。

。

「原來如此――汝等的體力頂多也就60～80的程度嗎？」

。

魔王如此嘀咕道，可在場應該沒有人能夠理解這句話的意思吧。

若說到唯獨能夠理解的事情，那便是這男人並不是「自稱魔王的男人」――

而是任誰也無計可施的，「真正的魔王」。

。

「即便是「我」――也有對你們發火的正當權利吧」

。

這聲嘟嚷小聲到誰也無法聽見，可裡面卻蘊含著深刻的實感。

。

「雖說現在，多少有點「感謝」――――」

。

魔王嘀咕到這裡的瞬間，滿身是血的馬基舉起逆十字架，吐露出細若蚊絲的可疑詛咒。這是惡魔信奉者實施的血腥術式。

並不只是把自己作為祭品奉獻出生命，而是奉獻出許多祭品的――禁斷儀式。
