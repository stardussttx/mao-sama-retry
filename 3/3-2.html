3-2 第19話 桐野悠

一一一一一一一一一一一一一一一一

「悠喲，在吾面前顯現吧――」

。

白與黑的巨大光芒伴隨著這句話出現於前方，當光芒重合時――

便形成一個女性的輪廓。

出現的是一位身穿白衣，足以用美麗加以形容的美女。

她留有一頭長而筆直的黑發，擁有完美比例的身材，作為女人而言可稱得上是理想般的姿態吧。

。

「您在呼喚我嗎，長官」

。

對於這道聲音，「我」感到某種難以言喻的懷念之情。

九內伯鬥在大帝國的正式頭銜是「國民幸福管理委員會」的長官。所以要以符合這頭銜的態度進行交談。

邊緩慢地丶慎重地丶逐漸地刺探對方的情況，邊行動吧。

。

「嗯，真虧你過來了――」

。

「若是長官的呼喚，隨時隨地便可。……請問，這裡是新的「遊戲會場」嗎？」

。

悠看著陌生的景色，面露困惑般的表情。

宛若冰雪般的女人露出困惑神色的模樣，莫名有種可愛之處。然而這邊的內向已經是因緊張而臨近崩潰。

。

（她在說話呢……果然具備自我意識）

。

目睹自己創作的人物角色在眼前活動著丶說話著的模樣，不禁感慨萬千。可我已經無法再從容地沉浸於感動之中。

到底要向她敘述多少真實，又到底從何說起呢？

。

（不過嘛……真是不得了的美女啊……）

。

她在遊戲中深受抖M玩家的歡迎，我覺得我隱約能體會他們的心情。

順便一提，悠和靜在女性玩家之中也是頗具人氣。

是因為強大的女性即便是在女孩子眼中也算很不錯嗎？？

。

「接下來，要從何開始說明呢……首先，這裡並不是大帝國。雖說如此，也並不是在他國」

。

恐怕親信們並沒有想過大帝國其實是電子空間裡的產物。他們會認為原本的世界正是「地球」丶「現實」才對。

即便說明原本的世界，以及個人角色都是由某人所創造，無論怎麼樣都無法令其相信吧。

。

我慎重地選取言語，向她說明迄今以來的事情緣由。

突如身在大森林一事丶遭受疑似惡魔襲擊一事丶墮落天使與這次事件似乎有所關聯一事，以及如今正前往神都等等。

。

無論再怎樣慎重地闡述言論，聽起來都是難以置信的話題。

可悠卻極為認真的，時而面露深刻思考般的深色聽取我的話。

。

「聽完事情經過，我倒覺得是某種強制移動的攻擊……原來如此，這片景色的確與我們所在的世界有所不同的樣子」

。

「理解迅速實在省事。然後是關於今後的行動方針――」

。

我告訴她關於調查熾天使的事情，得到SP就可以解放權限等等。

悠聽完之後，一時半會地把手置於下巴，隨即便從嘴裡回答出明確的解答。

。

「長官，無論事情經由如何，「不夜城」都將是必要――」

。

「……也是」

。

我意有所指地回答，可說實話我覺得沒這個必要。

如果是居住地這意義上，據點就已經十分足夠。而且前往城鎮也會有旅館。若在這幻想世界中建造近代化的大要塞，那才真正稱之為「魔王城」吧。

雖然我一時是想趁著興頭建造出來，不過現在還是持保留意見。

。

（畢竟似乎會立起最終被「勇者」之類的討伐致死的Flag啊……）

。

當然，無論是什麼來頭，我絲毫沒有打算要束手就擒。

因為如今縱使聖女過來，我也以波濤洶湧般的體罰做出對應了。所以即便是傳說中的勇者，又或者是光之什麼的存在來臨，我也只能以類似的方式來應付。

。

「長官――這不就是「天意」嗎？」

「嗯……就「轉移」一事，吾也如此認為」

。

「――――果然是這樣嗎」

。

悠浮現出妖艷的笑容，我不禁心頭一緊。

這傢伙絕對是大叔角色吧。

不對，從女孩子之間也具有高人氣這點來看，應該是魔性的女人這一系的？

。

「在這塊土地上，我們已經能看見應當要成就的事情了吧」

。

「離抵達目的還為時過早。即便說是千里之路的一步也不為過不是嗎」

。

不妙，我已經不知道這傢伙到底在說些什麼了。

於是我委婉地試著夾雜「話別說那麼快，給我好好說明」的意思，然而悠逐漸露出笑容，表情似乎很高興。

。

「不愧是長官――縱使在此等事態中也感到「愉悅」呢」

。

「……無論何事都應盡情享受。吾是這麼認為的喔」

。

嘛，如果把這當作是旅行考慮的話，應該不會有這麼豪華的旅行吧。

因為自己所製造丶遊玩的遊戲中，各種各樣的事物都具體化之類的，按照常識來考慮是不可能發生的事情。

。

「那麼，長官。請問「最初的第一步」要做什麼呢？」

。

「首先我打算進化據點，打造出《野戰病院》」

。

「………！長官總是在實現我的心願呢」

。

「當然――可愛部下的心願也即是我的心願」

。

真虧我能隨意地瞎說一些令人呆然的謊話。

總而言之，既然已經召喚了悠，那麼我想建造醫院把魔王之類的傳聞都清除掉。

。

只要以便宜的價格進行治療，惡劣的傳聞也就自然而然地消逝而去了吧。

畢竟這世界的醫療水平較為落後，魔法使用者也為數不多。

如果能夠順利運營，或許還能得到一大筆錢也說不定。

。。

「當然，嚴禁殺人。首先，吾等「非得知曉」不可」

。

不用說，我沒有忘記要鄭重地叮囑她一遍。

殘餘的熾天使丶這個國家的事情丶社會構造丶魔法丶冒險者，以及魔物和惡魔等諸如此類，非得知曉的事情無以計數。

如果流傳起殺人醫院的傳聞，那就血本無歸了。

。

「是的，若不知曉便一切無從談起……「知曉」是最重要的事情」

。

悠深深地點著頭，見狀我也感到放心了。

至少對於這邊並沒有忤逆之意，也會聽從我的命令。照這個樣子，繼續召喚親信應該也沒有問題吧。

人數一旦增加，行動範圍也得到擴充，那麼就能得到更多的情報。

。

「那麼，吾就把吾所知道的情報告訴給你」

。

我把這個異世界上的知識以及許多需要傳達的內容都告訴她。悠就猶如她的頭腦般，如同吸水的海綿立即做出反應。

這傢伙無疑比我還要聰明。無論是科學亦或是醫學，我都是一竅不通。所以得注意不要露出破綻……。

如果讓她知道我其實不是「九內伯鬥」――連我也不知道悠到底會做出什麼事情來。

。

「接下來，悠。我來介紹一下旅行的同行者――你會跟她們和睦相處吧？」

。

「是的，長官」

。

「在這之前，吾等的立場就決定以「從遠國而來的人」。今後你的言行舉動都得基於這一點」

。

「我明白了」

。

如此這般，我把悠介紹給兩人，不過當然是引起了大騷動。

嘛，理所當然。

突然之間就出現了一個人，這完全就像是魔法吧。

。

「非丶非常漂亮的人……！」

。

惡如此大喊道――

。

「難丶難道是愛人？到底從哪裡找來的！？」

。

露娜如此大喊道――

。

「我把「部下」召喚過來了――你們大家和睦相處吧」

。

■□■□

。

（事情變得有趣了呢……）

。

悠在聽取九內敘說事情的同時，內心微微歡悅。

雖說她們自己本身被賦予的權限十分巨大，可終究只不過是僅限於會場之中。

一旦脫離會場一步，大帝國的上層部便會化為權利鬥爭的大風暴，她們自己的立場是建立於極其危險的平衡之上。

。

（在這裡就不會受到上層部的拘束……我們自身的考量就能決定一切）

。

即便只是犯了一個錯誤，全員的腦袋隨時慘遭砍頭也不足為奇。

因為直至昨日依舊是友方的同伴，今天卻一同舉起槍支對準這邊之類的事情，在那世界上是家常便飯。

。

對於自己而言，實話說這裡是什麼世界其實都無所謂。

從世界本身就是「大帝國」解放一事――

這不就是在真正意義上，代表我們「活著」的意思嗎。

。

（而且是跟我們世界不同的人類，何況還擁有魔法……！？）

。

非得知曉不可。

非得知曉一切不可。

知曉這世界上的人類。

知曉肉丶皮丶臟器丶大腦裡丶心臟丶細胞丶遺傳子。

。

收集一切的情報，必須解開奧祕不可。

其實我現在都已經想立刻付諸行動。

不過，我壓抑住急躁的心靈，拼命地調整呼吸――

。

（話又說回來，長官依舊是位令人敬畏的大人……）

。

絲毫不把這等異常事態放在眼裡，似乎在優雅愉悅地享受著。

甚至還把來到異世界一事當做是「天意」。

我也一致同意這個觀點。

彷彿有人在述說——在這塊土地上建立起「只屬於我們的帝國」。

。

由脫離厭煩至極的上層部老頭子們的我們所建立的國家――那將會是理想的國家吧。無論是對於長官而言，亦或是對我而言。

雖說此話乃逾越之言，然而長官對於我而言既是上司的同時，亦是同志。

。

我是「僅僅」殺死了800名人類的微薄之身，而長官卻是立於不止400萬名人類血流之上，所誕生的魔王。

。

在這個世界。

如果我能呆在長官身邊，輔佐著他。

那到底……。

會有多麼的。

。

（呵呵……呵呵呵……）

。

不行了，笑意逐漸湧起。我已經忍不住了。

我想朝著世界，以最高的音量吶喊道。

。

「我們已經解放了」―――

。
