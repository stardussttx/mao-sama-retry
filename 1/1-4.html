1-4 第4話 願望祠堂

一一一一一一一一一一一一一一一一

長發男子正背著年幼的小孩子行走著。

如果只聽到這番話，應該會覺得這是一對父子，不過男人的外貌實屬異常，擁有這世界上較為罕見的黑發，而且其長度宛若女性一般。

何況全身上下都是一片黑的衣物，越看越像是死神或者惡魔。

。

「你好像說的是願望祠堂吧？是那種地方嗎，投點香錢然後祭拜的？」

。

「我也不知道詳細情形，不過我聽說智天使大人就是藉由這間祠堂的力量封印住惡魔王」

。

聽到這句話，魔王小聲地嘀咕道「這是啥神鬼片啊」。

他還以為是神社或者其他設施的同類，不過似乎是千錯萬錯。

即便原本在話題中就已經說過智天使丶惡魔什麼的，不過應該還是不會想到這些吧。

。

其內在可謂是大野晶的男人對於宗教毫無興趣，近似於無神論者。

若換作是九內伯鬥，如果被詢問神究竟乃為何物――他應該會當機立斷地回答是「自己」吧。

。

無論怎麼想這種人都不適合在幻想世界中生存，如果有異端審問團，肯定會是先被執以火刑的目標。

。

「我總覺得有種去了也沒意義的感覺……」

「沒丶沒有這回事！傳說中聽說會實現來訪者的願望」

。

惡如此說著，又往抱住我脖子的手上加強力氣。

好像是畏懼著魔王的奔跑速度，為了不被甩落而用力抱緊。不過從態度上看，似乎不僅僅是因為如此。

。

「跟丶跟別人丶像這樣接觸什麼的，那個丶是第一次來著」

。

惡的臉緊貼在背上，開口說道，聽到這番話的魔王則是蹙起半邊眉頭。

。

（對於這傢伙的性別，我到現在依舊是不明不白……）

。

「魔丶魔王大人要是前往祠堂的話，肯定是想許願得到征服世界的力量吧！」

。

「這種力量我才不需要！」

。

雖然魔王作為成年人的態度正逐漸奔潰，不過他們終於抵達目的地。這裡是杳無人跡的森林之中更加人影稀疏的場所。

那便是巨大的山墻上鑿開的――洞窟。

■□■□

（喂喂，這氣味不就是……）

。

一接近洞窟，一股異樣的臭味便撲鼻而來，魔王不悅地蹙起眉頭。

。

「惡，你在這裡等著。裡面好像很危險」

。

「好丶好的！」

。

氣味的根源顯而易見。洞窟深處盡是人類的屍體。

既有似乎被巨大的尖爪切傷的，也有身體只剩一部分的焦炭。從這些屍體裡流出的大量血液與內臟濺灑而出的尿糞相互結合，產生出異樣般的臭味。

。

（這是在搞什麼黑魔術的儀式嗎）

。

一座石像坐鎮於此，彷彿在俯瞰著這些屍體。

只見石像上索繞著邪氣，這邪氣足以令人覺得光憑濃度就可以殺死人。願望祠堂這一名字與這幅光景實在相差甚遠。

。

當魔王往前邁出步伐的瞬間，石像的眼睛散發出赤紅之光。

魔王見狀，把右手伸向小刀。不過石像紋絲不動，彷彿在用那雙眼睛仔細端詳著入侵者的全身。

毫無行動跡像的石像開口說道，嘟嚷著什麼。

。

「原來如此――這的確是「魔王」」

「……嗯？」

「雖然吾已經實現了諸多願望，不過這恐怕將會是生平最後一次了吧」

「等丶等一下……你知道些什麼嗎？難道說，是你把我叫過來的？」

。

石像對於魔王的質問保持沉默，不久後開口說道。其回答對於魔王而言是極為重要的事情，然而魔像卻若無其事地說了出來。

。

「並非是吾――亦或是說這群人類吧。要讓「魔王降臨「呢」

。

「這些傢伙嗎……！？剛才的怪物也是這些傢伙叫出來的？」

。

「它是以一己之力復活的――拜此所賜，吾之力也已經消失殆盡」

。

「消失殆盡……那麼，在力量消失之前先讓我回到原本的世界」

。

石像的回答實在過於簡潔――「無法辦到」。

過於殘酷的斷定語氣，令魔王不禁撓破腦袋。

。

「為什麼辦不到啊。想要香錢嗎？應該不會是像這群傢伙一樣，要我獻上祭品吧」

。

「這裡的死人都是古雷歐魯乾的好事。而且吾無法實現「違背願望」的願望――」

。

簡而言之，令魔王降臨的願望已經實現了――所以能夠令這個願望作廢的願望都無法實現吧。某種意義上真是循規蹈矩。

。

「雖說如此，汝亦即是最終的來訪者――就把這個賦予給汝」

。

從石像身上出現了不詳的戒指，並強制性戴在魔王的手指上。看到這枚戒指的惡心設計，魔王拼命想要摘下來，無奈戒指紋絲不動。

。

「別給我開玩笑了，你這傢伙！隨身攜帶這枚戒指是要搞什麼懲罰遊戲嗎！」

。

「吾祈禱著――汝之願實現之時――」

。

「你這個混蛋邪神……給我等一下！」

。

「吾本應乃為清廉之身――只因常年累月中，人類的邪惡之願才化為此身」

。

石像宛如說完遺言一般，身體便粉身碎骨，遂而化為粉末，從臺座上流落而盡。而看著這幅景像的魔王卻無能為力，只能呆然地看著石像崩毀。

。

「哈啊……就算得到這枚戒指，又要怎麼辦才好啊」

。

魔王看見右手指上戴著的戒指，不禁垂頭喪氣。

無論用上多少力氣，做什麼事都完全摘不下戒指。戒指樣式也是，完全就是負面道具。

。「管理者權限――《道具鑒定》」

。

魔王發著牢騷進行道具鑒定，結果更是垂頭喪氣。

如果這是擁有「詛咒屬性」的道具，還可以使用解咒道具摘下戒指，不過期待似乎落空了。

。

「為什麼是以普通的道具對待啊……而且連鑒定都要使用SP麼……」

。

魔王不停地嘆氣。

這下子根本就不是願望祠堂，而是嘆息祠堂。

當出去洞窟外面，惡就滿臉笑容地搭話說「怎麼樣？」，不過魔王依舊板著一張臉。

。

「果然是征服世界吧？還是酒池肉林之類的呢？」

。

魔王聽到這句話，一聲不吭地封住惡的脖子。

接著，緩慢地用拳頭輕輕的往惡頭上招呼。

。

鏗鏘！鏗鏘！

。

「好痛好痛！請住手吧，魔王大人！」

■□■□

（接下來要怎麼辦呢……要去其他城市收集情報嗎？）

。

「我」把長發往背後一甩，沉浸在思考之中。

不知道的事情，不清楚的事情實在太多。

如果還像這樣對事情不明所以，遲早會陷入意想不到的圈套中。是時候從這偏僻的森林脫身了吧。

。

（結果也沒從那些屍體中找到什麼呢……）

。

那副光景僅僅只是回想起來，就有股想吐的沖動。

雖說遊戲中經常出現大量屍體，不過跟在現實中看見屍體完全就是兩碼事。如果說那副光景是剛才的惡魔造成的，那麼消滅掉它果然是正確的選擇。

只要那隻惡魔還活著，就只會產生人類的屍體而已。

。

「惡，這附近有什麼大城市嗎？」

。

「有的……可是在這之前，能不能去一趟村裡？　雖然我的行李很少，不過我想帶走……」

。

「嗯？你還打算跟著來？」

「不丶不行嗎……那個，我明明就已經作為祭品貢獻出去了，又不可能再回去過日子……嗚嗚……」

。

聽到惡這番話，我不禁想抱著頭，不過重新想一想，這或許也不錯。

不管怎麼說，我對於這個世界實在不甚瞭解。如果身邊有位這個世界的居民，也算是一劑強心劑吧。

而且，從迄今聽到的遭遇來看，惡即便繼續呆在那村子裡也只會遭受到嚴酷的對待吧。

。

「我知道了，那就先去你們的村子吧。近嗎？」

。

「非常感謝！以魔王大人的速度很快就到了！」

。

就這樣，魔王又再次背著小孩子奔跑起來。

雖然魔王嘟嚷著「又不是帶子雄狼……」，不過惡的表情鬆弛著，總覺得似乎很高興的樣子。

(颯:《帶子雄狼》（日文：子連れ狼）是日本的一部以江戶時代為背景的時代劇漫畫，描述遭柳生一族陷害的高強劍客拜一刀，為了避難與伺機報仇，帶著幼子大五郎浪跡天涯的故事。by百度百科)

■□■□

（這種時候，在抵達村子之前還是先打聽一些事情吧……）

。

路途中，我向惡提出各種疑問與詢問。

事到如今，已經知道日語能夠溝通，而記在本子上的日語丶數字以及羅馬音，惡都能順利地讀出來。

是因為這世界原本就是能用日語溝通的嗎，還是說有某種過濾器在運作呢，這方面根本無從得知。

。

「嘛，既然日語可以相通那是再好不過……這種歲數還要學習異世界語言什麼的，老子才不幹」

「……？這字體非常漂亮，也容易看懂哦？」

。

對，寫在本子上的文字漂亮到我自己都怕。

這恐怕不是我的字跡，而是九內的吧。真不愧是設定上的高官。

自己的字跡比較醜這一點，莫名感覺到敗北感。

。

「不過祭品還真是古時代的習俗呢……這附近也有這種習俗嗎？」

。

「由於惡魔王在幾年前復活，這一帶都遭受波及，因此附近的村莊也陸續有了獻上祭品的習俗……」

。

「雖然我不怎麼清楚……不過你曾經說過的「國家」就沒有做過什麼措施嗎？無論是討伐惡魔又或者是什麼不也可以」

。

「因為這一帶是遠離神都的土地……」

。

換句話說，從「都會」的角度來看，這裡是怎樣都好的土地？

在日本也有邊境的集落，離島之類的。偶爾會在新聞上看到這些信息，差不多是這種感覺吧。

。

「那個丶魔王大人……究竟是從什麼地方過來的呢？」

。

我一瞬間猶豫著要怎麼回答。

畢竟就算說出日本也不可能知道，況且大帝國之類的更是意義不明。因為那隻不過是以我的妄想製作的遊戲世界，根本不存在於現實之中。

。

「嘛丶嘛……該說是從很遠的地方而來吧」

。

我曖昧不清地說著，加快步伐。畢竟實話實說誰也不會相信我，而且若被人認為是一位好瘋子，那也是個悲劇。

。

「啊丶魔王大人，那片柵欄對面就是我們村子！」

。

「那裡嗎……」

。

雖然我對自己逐漸習慣被人稱呼「魔王」這件事感到可怕，不過柵欄對面的光景才更加可怕。眼前闊展的村莊與其說是無人居住，不如說是鬼怪故事裡出現的寂靜小村。　□　□　■　■　□　■　■　□　□
