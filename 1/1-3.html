1-3 第3話 魔王與惡

一一一一一一一一一一一一一一一一

某座森林已然許久杳無人跡，然而今日卻有兩位人類在此。

一位是形跡可疑的長發男子，另一位則是個小孩子。

。

「我丶我的名字是……九內。我可不是什麼可疑人物喔？跟魔王更是毫無緣分。我有幾件事想要跟你打聽一下，可以嗎？」

。

男子……不對，「魔王」把自身長發往後一撩，邊開口說道。

雖說他苦惱著要自報什麼名字，不過最終還是選擇九內。姑且是在小孩子面前，留意著自己的語調像是個成年人，不過這張假面似乎隨時都會剝落似的。

。

「好丶好的……」

。

魔王為了不被懷疑，盡可能面露笑容地說話。然而小孩子卻睜大眼睛，顫抖著身體。

這也是無可厚非，眼前這位可是把最上位的惡魔給秒殺的人物。

。

在這世界上，只要知道「那」是怎麼樣的存在，這反應也可以說是理所當然。連最上位惡魔都無法匹敵的存在――除了「魔王」之外，並無他人不是嗎。

外貌――也是與魔王與之對應的恐怖容貌。

。

「在這之前，我先問一下你的名字吧。如果可以，能不能告訴我呢？」

。

「我丶我的名字是惡……」

。

聽到這回答，魔王「噗啊！」地噴出來，激烈地咳嗽著。嘛，也不是不明白他的心情。

「魔王」與「惡」――這到底是哪裡的謎語啊。

。

「真丶真是不錯的名字呢。然後呢，你知道日本這個國家嗎？或者紐約也行」

「非丶非常抱歉……我沒聽說過……」

。

魔王邊輕聲地嘟囔道「我想也是」，邊點上煙，吸起了煙。

就他看來，惡身上的穿著無論怎麼想都不會是近代的衣物。以簡陋的毛布製成的衣物上，只有少量裁縫的藍色花紋，跟時髦大相徑庭。

。

下裝的褲子長度則是不到膝蓋，與骯臟的臉頰相互結合，難以明白性別是男是女。

發型雖說是短發，可只有劉海那邊異樣地長，都完全覆蓋住左邊的臉部。

。

魔王不禁想要直接詢問性別，不過以作為大人的矜持硬是忍住不問。他應該是覺得對於初次見面的小孩子詢問「你是男是女？」而感到羞恥吧。

如果他是「九內伯鬥」，肯定會不顧一切手段也要把情報都毫無保留地吐出來。內在是「別人」這件事，對於惡而言實屬幸事。

。

魔王的提問連綿不斷――

這世界的名字是？這座森林是哪裡？剛才的惡魔是什麼？

相較於魔王的詢問，惡回答得語無倫次，不得要領。

。

不如說惡的知識量並不那麼豐富。

關於這個世界的常識都毫無問題，不過似乎沒有更進一步的知識。

。

從衣物來看，單純是村民而已。

村民A跟一般人A在一起商討，根本得不到任何收獲。

。

「我還有想要打聽的事情……不過在這之前，那邊有片湖泊。先把身上的泥巴洗幹凈怎麼樣？」

。

「可丶可以嗎！？」

。

「嗯……？嘛，不是很好嗎。也可以順便把身體洗幹凈」

。

「非丶非常感謝！」

。

惡臉上浮現的笑容，一瞬間讓魔王啞口無言。

對於他而言，這只不過是看到骯臟的臉頰與衣服而脫口的無心之言罷了。

對，他不知道――在這塊土地上，所謂的「水」是多麼珍貴的事物。

。

僅僅是用於飲用與烹飪就已經勉為其難。在此之上還想奢求水源，當然不會是免費。

只有一部分人才有權利用水洗凈身體。

在這塊偏僻的地區，水是重要的財源，領主才有權受惠。

■□■□

（哈啊……怎麼辦）

。

目送惡以興高采烈的步伐走向湖泊的身影，「我」嘆了口氣。

難得我找到了情報源，不過惡卻是平淡無奇的村民，得到的情報寥寥無幾。

現在我知道的信息就只有關於這個國家的事情，以及惡居住的村子。

。

（叫做聖光國來著？在地球不可能有這種國家吧……）

。

這個國家似乎是信仰封印住那隻惡魔的智天使而形成的。旗下有三位稱之為聖女侍奉著智天使的人物，以及聖堂騎士團，聖堂教會等等。一堆只能令人呆然傻笑的詞語。

這裡就是極致的幻想世界。

。

如果是讓年輕旺盛的高中生轉移到這裡那倒還好，可明明被召喚的是這種大叔卻還要把上班族都叫過來是鬧哪樣啊。

想到這裡，某種討厭的預感在大腦中一閃而過――被召喚過來的，並不是「自己」不是嗎。

。

（被召喚過來的，不如說是這具身體吧……？）

。

一般來說，如果連續體驗到這些不可思議的事情，常識都會逐漸崩壞。曾經不也有一位玩家「炸藥☆死國」笑著說「無論是誰都會轉移到異世界喔」。

無論遊戲角色，還是佛像，又或者是戰國武將，這世上有什麼被召喚到異世界都已經不足為奇。

。

如果用我這已經腦殘的大腦仔細思考一下，也能夠理解這具「身體」會被召喚的理由。

雖然我不知道有什麼目的――不過若是「九內伯鬥」，應該擁有多種用途吧。只不過本應被召喚的角色剛好是我在「登陸」而已。

。

（換句話說，並不是我被」卷進事件」了！）

。

一想到這裡，怒氣便猛然湧現。

就因為這該死的老頭子，搞得我自己被丟在莫名其妙的世界之中。絕對是這樣，不會錯的。不如說，如果我不這麼想就無法再堅持下來。

。

（得立即找到回去的方法）

。

雙親早已經逝世，而我又是單身狗，只有這一點該說是幸運嗎？然而，無論怎麼想公司都完蛋了。

如果在這種歲數還擅自曠工的話，我會被怎麼說呢……。

。

（既然已經下定決心，那麼應該要前往哪座城市呢？還是說要在這座森林裡尋找線索？）

。

我緩慢地環顧四周，森林之中只有令人感到毛骨悚然的寂靜圍繞於此。

看來此地似乎不宜久留。

。

（話又說回來，惡那傢伙好慢啊……不慢？）

■□■□

（還活著真是太好了……！）

。

惡如今欣喜若狂。

浸泡於大量幹凈通透的湖泊之中，充分地把身體與衣物洗幹凈。甚至連村長都無法享受到這種奢侈。

因為要洗澡和洗衣服的時候，充其量只能等待天空下雨而已。

。

而且必須是傾盆大雨――而不是綿綿細雨。這塊土地曾經是惡魔橫行霸道之地。拜此所賜，大地也完全乾枯，古井幾乎都被毒所侵蝕，甚至都難以確保最低限度的水源。

。

更重要的是――原本自己就是「汙穢之身」。絕對不會放過這次機會。

。

（話說回來，那個人到底是何方神聖呢……）

。

惡一絲不掛地泡在湖水裡，只露出頭部想著事情。僅僅一擊就把那位最上位惡魔完全消滅的人物。果然是如今遺留的神都傳承中――被稱之為魔王的存在吧。

光從全身那套漆黑色的服裝來看，根本想像不到是天使大人。

。

原本獨自一人呆在這座森林本身就實屬異常。因為這座森林裡有傳承中流傳下來的神聖祠堂，是連走近都會有所忌諱的地方。

雖然常聽說這森林裡有個美麗的湖泊，不過誰都不敢來到這裡不僅僅是出自於對傳承的顧慮，也是因為還有那位惡魔在此。

。

（難道說，要讓我身體洗幹凈之後再打算吃掉我……？）

。

身體不禁泛起惡寒。

雖說現在就想立馬逃跑，不過衣服已經洗幹凈放在樹枝上晒幹了。
