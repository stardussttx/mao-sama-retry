1-5 第5話 魔王之戒

明明現在仍是白天，可卻沒有從那村子裡感覺到任何「人類的活力」。

腦袋裡浮現出深山內部裡封閉式村莊的印像。

。

「這村子給人的感覺就像是無法相信外地人呢」

。

「是丶是的……我們村子是有排斥外來者的傾向」

。

還真的是這樣啊……雖然我只是說出我的偏見而已。

我還想著如果有機會就分點食物或者旅費呢，不過期待落空了。

。

「我的家在這邊。這裡或許會讓您留下不好的回憶，務必請寬容大量……稍等一下，我馬上就回來」

。

惡從我背上下來，拖著右腳般走了過去。

看到惡的背影，心中不禁感到悲哀。或許惡都無法正經地看過醫生說不定。

我邊跟在惡的身後，邊謹慎地觀察村子裡面。該說果真如此嗎，完全看不出是近代文明的感覺。

。

（房屋是以裸露的木頭以及凝固的土塊做成……屋頂也是鋪上一層層稻草）

。

在日本可謂是理所當然的空調丶暖氣丶以及電線等都完全看不到一絲痕跡。

我再次認清這裡是與日本截然不同的世界。

當我還在端詳著周遭的事物時，遠方到處都看得到村民的身影。

是因為遊戲的影響嗎，我立即隱藏身姿――由於在遊戲被其他玩家發現就會遭受到殘酷的對待，所以這反射動作幾乎已經算是職業病。

。

「應戰態勢變更――――《隱祕態勢》」

。

在這句話脫口而出的瞬間，我的身體彷彿與周圍的景色同化一般，輪廓變得模糊不清。

這在遊戲中擁有急劇降低被對像發現的效果，不過卻有大幅度降低攻擊力與防禦力的缺點。

。

當看見自己的身體消失了，我才安心地摸了摸胸口。

雖說心裡還是有點緊張，不過從周圍的反應來看，似乎並沒有看到我。

。

連這些系統都跟遊戲如出一轍，莫名感到有些可怕。

這世界――是否能夠把遊戲中存在的都全部具體化呢？

那麼，嘗試一下其他事情吧。

。

「啊咧……魔王大人？」

。

果然似乎就連惡也看不到我。

在這堪稱絕妙的時機上，我有件務必想要嘗試的事情。

。

在遊戲中幾乎很少使用的《通訊》變成什麼樣了呢？這功能主要是在玩家之間傳達信息。

在遊戲最初運營的時候，當然不存在什麼智能手機。那時候的通訊可謂是風靡一時，不過在近年來早已經是過時的機能。

。

《通信「惡」――聽得見嗎？》

《……誒，大腦裡響起了魔王大人不詳的聲音！？》

。

《不詳這句話是多餘的。我的事情就不用在意了，反正就在附近》

《好丶好的》

。

我順利地確認完通訊機能的運作狀態。

果然，其他的事情也嘗試看看吧。如果我孤零零地呆在那片森林裡，肯定還會是不明不白的狀態之中吧。

或許我該向惡表示感謝也說不定呢……。

。

（不過我神經也太大條了吧……照一般情況來說，現在我就算是哭天喊地也不奇怪）

。

難道說連我的意識都向著身體的持有者「九內本人」靠攏了？

。

（別想了，別想了……這是什麼恐怖片啊）

。

自己想到的――是非常可怕的想像。

我連忙搖搖頭，把無意義的幻想拋之腦後。

。

「喂，垃圾――為什麼你會在這裡！」

。

我聽見不悅的聲音，便朝著聲音源頭的方向一看。只見幾位像是村民的人，手指著惡怒吼道。

不用想也知道，這就是那幫欺凌惡的人吧。不對，或許是整座村莊也說不定。

。

「垃圾，你……應該不會是逃跑了吧！」

。

「別給我開玩笑！要是那隻惡魔來到村裡，你打算怎麼辦啊！」

。

「你到底知不知道作為祭品的意義啊！？」

。

聽到村民們說的這些話，我頭都大了。

這些傢伙到底知不知道自己在跟小孩子說些什麼？

雖說如此，並不是這世界上居民的我，也難以插手這世界的事情(？)。

。

在這個世界上，向惡魔獻上祭品或許是理所當然的事情。現代中也經常出現外國獨特的習俗或者「在日本是難以想像的事情，不過在某些國家卻是常識」等情形。

。

（話是這麼說丶呢……）

。

果然，大人斥責小孩子並不是令人愉快的事情。

如果保持隱祕態勢的狀態下直接帶走的話，應該可以平安了事吧？

。

『若是感到不悅，「肅清」即可――――』

。

在大腦裡回響的聲音，令我背脊發寒。

我立馬明白這究竟是「誰」。接著，右手指產生一股――難以忍受的苦痛。

我不禁蹲下來按住右手。

疼痛過於激烈，我完全無法站起來……！

。

『處分已審判的無法者――此「權限」便在「吾」身』

。

別開玩笑……那是僅限於遊戲中發生的事情。

對於現實中生存的人類進行肅清丶處分，這種事情我怎麼可能做得出來。

。

『汝說了莫名其妙的話。製作這樣的權限，這樣的遊戲，這樣的系統，這麼瘋狂的國家――不全都「汝」嗎』

。

聽到這番話，我不禁陷入沉默。

。

『汝才是――「諸惡的根源」。我若是魔王，那汝亦即是為全世界招來破滅的――』

。

我緊握著戒指，以彷彿會握碎戒指的力度強行讓聲音消失。這番話只不過是我的幻聽罷了――我如此相信著，深深地閉上眼睛。發生這麼多事情，肯定是我累了吧。

。

《惡，別管這些傢伙，去把行李拿回來就好了》

。

等了一會兒，並沒有答復我。

我睜開眼睛，惡那仰空倒在地面上的身影便映入眼簾。村民舉起手，似乎在用什麼骯臟的話語怒罵著。

我無法再繼續袖手旁觀，解除了隱祕態勢。

。

瞬間――村民們吵鬧不已。這也是沒辦法的事情，畢竟突然有人憑空出現。不過他們那副狼狽的樣子真使人發笑。

。

「惡，拿回行李就趕緊過來」

「我丶我……知道了」

。

我看著惡拐著腳往房屋裡走進去之後，慢慢點上香煙。這時候村民們依然沒有停止吵鬧，人數還反而還越來越多。

原本我應該是要向他們打聽許多事情的，不過我沒興趣跟這些傢伙聊天。

。

「你丶你是惡魔嗎……？是那位大人的手下嗎……？」

。

「別對我們村子出手！我們已經獻上祭品才對！」

。

「跟約好的不一樣！說到惡魔應該都是會遵守約定的吧！」

。

我邊吐出煙氣，邊思考著村民說的話。

跟惡魔約定丶契約還真是引人趣味的事情。不過我倒是比較擔心「對方」是否有打算要遵守約定的意思。

倒不如說，這應該是對方先提出契約，再借此把對像侷限在封閉的環境之中吧？

。

「讓您久等了，魔王大人！」

「居丶居然是魔王！？」

「魔魔魔王！」

。

（喂喂，別說些多餘的事情啊……！）

。

惡的無心之言，讓村民們的喧吵聲更上一層樓。

雖說如此，我也無心解開這幫傢伙對我的誤會。把小孩子叫做垃圾，當做祭品貢獻，再加上整座村莊都在施虐。

我有必要為這些傢伙努力嗎――？不，沒有。

。

「我丶我先去通報領主大人吧！」

。

說完，撂倒惡的男子跑了起來。

其表情上浮現出令人生厭的笑容。

或許是想到什麼事情吧，男人先是回家一趟，出來的時候就已經拿著個揹包。應該是為了得到獎賞而準備放進去的吧。

。

我――無意間蹙起眉頭。

。

戒指彷彿呼應著我的感情，釋放出妖艷的光芒――只不過這一次我並沒有打算阻止。

右手伸進外套裡面，毫無取巧地往男人家裡投擲一把小刀。

小刀準確無誤地刺進房屋上，迸發出漆黑火焰。用木頭製作而成的房屋一瞬間起火了――包裹在裊裊升起的黑煙之中。

。

「我丶我的……我的家啊浮浮浮浮浮！」

『――哈哈哈！無論何時，一看見火彷彿心靈都受到洗禮！』

。

突然，我仰天說出這番話來。

隨即慌慌張張地背上惡――離開了這裡。如果放任自己委身於這枚戒指，似乎會釀成大禍。

。

（這不就像是個縱火魔一樣嗎！）

。

我擦拭冷汗的同時，全速逃跑著。景色宛若飛奔一般地流逝，彷彿只有自己是在快進中。

。

「魔丶魔王大人――沒問題嗎，那種事！」

。

「你丶你別誤會喔，我只是想取暖而已。就算說是我的溫柔也不為過。嗯，我有這種感覺！」

。

踏出的步伐已然無法停止。

隨意胡說的事情也已然無法停止。

。

「不丶不過……！稍微有一點點丶神清氣爽了！」

。

惡如此說完，便露出笑容。

或許這是我和惡相遇以來，最初看到的「笑容」。

。

「對丶對吧！人類的溫柔果然是要傳播出去的吧！」

。

隨意的言語，令我自己都忍不住噴出來。

注意到的時候，兩人早已經爆笑不停。

抬頭望天，明亮的太陽已經沉浸，夜幕即將要降臨。如果是這具不知疲倦的身體，我感覺無論是天涯海角我都能奔騰而至。

。

「魔王大人，我們要去哪裡！？」

。

在宛如吹拂著身體的風中，惡大叫道。

迄今為止，惡的表情上總有某種難以掩飾的陰影，不過如今卻綻放出與年齡相應的光彩。我――也不禁忘記年齡，大聲回答道。

。

「我們將前往都會――那個叫「神都」的噢！」

。

如此這般，化身成為小小縱火魔的魔王大人，與明亮的「惡」的旅行，就此開始。

兩人引發的無數次騷動，把聖光國搞得是天翻地覆……

。

――――不過，那是在不遠的將來所發生的事情。

一部分情報已經解放。

「Status」

攻擊與防禦的「1」，為人類擁有的平均數值。

這數值無論是小孩子亦或是大人都不會產生變化。

換言之，若數值為「2」，那就是人類平均數值的兩倍。

而「3」就是三倍的數值，如果是4或者5……

。

若硬要用現代的觀念來衡量，可以說是近似於超一流運動員或是奧運會選手。

在這世界上1與2有相當大的差距，每提升一數值都要面臨巨大的墻壁。或許可以稱之為「才能之壁」。

。

「氣力」

氣力擁有形形色色的用途，魔法也是以此為基準而釋放的。

基本上在指數完全耗盡之前都可以全力行動。

若是超一流的戰士，都具備50點屬性。

這代表兩小時之內能夠全力行動的指數。

。

九內的氣力是600。

他在遊戲上是二十四小時――全時段都能夠全力行動。

既是強大無敵的「怪物」，也是一位「魔王」
