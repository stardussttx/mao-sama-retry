2-2 第7話 悄悄接近之影

悄然接近之影

一一一一一一一一一一一一一一一一

――聖光國　前往神都途中

。

（這就是洗澡……！）

。

惡從走進據點開始，就不停地感到震驚。

最先感覺到與自己家裡根本性的不同――就是「堅固性」。譬如說，縱使熊和豬沖進來，據點是不是依然穩如泰山呢？

即便對像換做是凶惡的魔物，感覺也不可能沖破這裡。

。

據點之中還配備有床鋪和廚房，終於有種自己完全就是井底之蛙的感覺。

在家裡頂多也就是用簡陋的毛布鋪在稻草上，然後睡在上面而已。

與其相比，這裡簡直就像是天國。

。

（值得一提的，就是這個鐵桶浴――！）

。

聽魔王大人所說，這似乎是叫做「鐵桶澡盆」。

令人吃驚的事情是，這裡面放進了「熱水」！奢侈到無法相信的地步。即使我已經泡在裡面，身體也止不住顫抖。

。

（不過，魔王大人為什麼要道歉……）

。

正確的來講，應該是「抱歉，現在這座據點就已經是極限了」來著？

老實說，我完全聽不懂這是什麼意思。

如果是想說這已經是「極限」的話，那是理所當然。因為我根本就想像不出還有什麼比這更加奢侈的事情。

。

（魔王大人究竟是來自哪裡呢？雖然我也想過是魔界之類的地方……）

。

有點在意魔王大人喝醉的時候說過的「大帝國」。

那裡有可能是魔王大人的故鄉也說不定。

當我想著這些事情，就聽到魔王大人的聲音從門對面傳過來。

。

「抱歉，地方太小了。不過你給我等著――遲早我要設置「溫泉旅館」」

。

魔王大人僅僅說完這些，便發出腳步聲離開了。

魔王大人說的話非常艱澀難懂……如果我更加聰明，或許就能知道是什麼意思，不過沒有的東西強求也無濟於事。

。

（話說回來，什麼是文圈驢廣？）

。

即使惡浮現出單純的疑問，鐵桶浴依然讓惡感到十分滿足――

這是非常和諧的光景。

。

另一邊――

。

魔王側身躺在床上，眺望著窗戶外面的夜空。

。

（這個世界也有月亮啊……）

。

像這樣呆一會兒，應該會逐漸醒酒吧。

來到這個世界已經過了相當長的時間，不過並沒有發生回到原本世界的自然跡像。

。

（如果權限全部都恢復了，真的就能回去了嗎……？）

。

至少管理者權限是遊戲中的相關系統，所以當然不存在「回到原本的世界」的指令。

如果我是由那座像是邪神的石像召喚過來的，那就代表我只能尋找類似的人物，拜託它送我回去嗎？

。

（最大的問題應該是，就算我回到原本的世界，時間段是否會是一樣吧？）

。

如果能回到那一天那一刻，那倒沒什麼問題。

可如果在這裡度過的時間跟原本世界的時間是同樣在運作的話，那將是致命性。

。

假如說我一個月之後回到原本世界，本應作為失蹤案件處理的我，肯定會引起巨大騷動。「我去了趟異世界，嘿嘿」這句話也未必行得通。

到時候，我應該會被送進精神病院，長期被關在付有鐵欄的病房裡。

。

（現在應該先以恢復全部權限為目標嗎？）

。

不過為此需要什麼，我仍然不清楚。

只要單純積累SP就好了嗎，還是說賺錢就可以了，又或者是收集道具呢。

無論如何，繼續漫無目的下去也不會有任何改變。那還不如想著可以旅行了，抱著期待過活還比較有建設性。

。

（就算回去了，也是日復一日的工作麼……）

。

「魔王大人~！鐵桶浴實在太幸福了呢！」

。

聽到這句話，我噴了出來。

又不是戰爭時期，就算你對我說這種洗澡很幸福，我也不知道該露出什麼樣的表情。

雖說因為某些因果關系而一起旅行……不過管理機能要是恢復了，做些讓惡大吃一驚的事情或許也蠻不錯的。

畢竟惡直到現在，似乎都沒有度過幸福的日子。

。

（溫泉旅館丶嗎……既然要做，乾脆也把「不夜城」復活吧？）

。

不夜城――是遊戲中以「九內伯鬥」為首的全部親信所坐鎮的最終領域。

即便是在世界上的大富豪之中，也只有特別的人類才能受到招待，實時觀賞大帝國的「遊戲」。這就是不夜城的設定

。

鐵壁般的最終防禦據點不僅僅只有親信在保衛著，還有兩千多名部隊。

如果讓惡見識到集結了大帝國全部科學力量的不夜城――究竟會露出什麼樣的反應呢？

。

不夜城度過了十年以上的漫長歲月，無數次擊退了玩家，也因此是個沾滿鮮血的地方。不過只要有不夜城，就能夠完全確保人身安全。

因為不夜城在過去從未陷落過。

何況其他親信如果也來到這個世界，行動範圍也會大幅擴大吧。

。

「魔王大人，今天我也用肥皂把身體洗幹凈了」

。

「嗯……的確，有股香味呢」

。

「真的嗎？誒嘿嘿……」

。

惡很高興似的笑著，接著鉆進被窩裡。

因為惡的頭發是金發，就像是毛色不錯的小貓，有種令人懷唸的心情。

。

「你啊，打算跟我一起睡嗎」

。

「不行嗎？」

。

「你好像是把我這種大叔當做是父親一樣，不過我有件非常重要的事情要告訴你――聽好了，真正的我其實是「哥哥」。我再說一遍，我還不是大叔」

。。

「等一下魔王大人，我聽不懂您在說什麼」

。

如此這般，魔王做著無謂的掙扎，就這樣度過一晚。

■□■□

幾天後――聖光國　山中

。

山中有將近四十名「山賊」在蠢蠢欲動著。

他們聽聞聖女乘坐的馬車將會通過這條山麓。所以山賊們不會放過這絕妙的機會，做好襲擊的準備。

。

這一帶有被稱之為土龍――遭受人們忌諱的一幫人。

。

他們不會選擇獵物。

無論對像是多麼厲害的強者，亦或是多麼柔弱的弱者，他們都會一視同仁地進行襲擊。

時而也因此受到慘痛的反擊，不過其總人數比起減少的數量，反倒是增加的人數比較多。這也著實證明，這國家是多麼混亂不堪。

。

首領率領著這個不懼生死的集團，坐在樹樁上傲然盯著山麓。

他從十歲起就幹著搶劫的勾當，如今已然是馳名於附近一帶的山賊首領。雖說年齡高達51歲，不過體格依然健壯如牛，與年輕時期別無二致。

正當他緩緩傾斜著酒瓶之時，便看到山麓裡的人影。

。

――不是那些獵物。

。

他立即得出判斷，不過又想到其他的事情。

兩個人馬車也不坐，就在這種堪稱荒野的偏僻地區上走著。

。

――陷阱嗎。

。

如果我們盯上這份人數不多的美味誘餌，背後卻有本隊進攻過來――

回過神來已經被團團包圍，只能連滾帶爬地逃之夭夭。這種事也是常有的。

這時候，原本無法掌握聖女目的的他，終於也明白過來了。聖女是打算引誘我們出現，再借此機會完全殲滅。

。

「……呲，還真是被小看了」

。

「首領，那兩個人要怎麼辦？」

。

「殺掉――在背後的本隊趕來之前撤退」

。

正因為首領在這條路上生存已久，所以才要迅速地下定決斷。

在緊急關頭根本無暇感到迷茫，因為如果不做出行動，立即就會喪失性命。已經有許多同伴就因為關鍵時刻不知所措，導致失去生命。

手下們靜悄悄地行動起來，把弓箭搭在弓弦。

全員同時釋放弓箭――那是對於殺人絲毫不感到躊躇的連串動作。

■□■□

。

（真是粗劣的攻擊……）

。

魔王看見飛過來的弓箭，嘆了口氣。

氣息既沒能完全隱藏起來，弓箭也是普普通通。

。

――什麼力量都沒有。

。

如果是在遊戲中，屬性技能《雨箭連擊》就能輕鬆地向我發射出幾十支既死級的弓箭吧。從魔王的角度來看，這只不過是朝著過往的路人隨意射出弓箭的難看攻擊。

。

（要用身體硬抗下來嗎――？）

。

魔王下定決心，決定要扛下來。

他想要嘗試一下，這究竟會對自己造成多大的傷害。自從與那隻惡魔遭遇之後，由於沒有任何敵對者，也因此沒辦法正經地進行實驗。

這是一次絕妙的機會。

。

以防萬一，魔王已經取消了自動迎擊和自動反擊的效果。

既然不知道這究竟是基於什麼緣由觸發效果，那麼也就沒辦法加以信任。

。

搞不好會對於毫無問題的行為進行可謂是致命性的反擊。如果反擊的對像是怪物那倒還好，可如果是人類，麻煩就大了。

。

「惡，退下」

。

「……誒？」

。

飛過來的弓箭，在魔王眼裡彷彿就像是停止不動一般。

這也是當然――因為他在遊戲中曾經就躲過突擊步槍發射的子彈，甚至就連散彈槍發射出來的散彈都躲開了。

飛過來的弓箭終於迫近到魔王眼前，不過在刺進身體之前就微微響起電子聲，弓箭也無力地掉落在地面上。

。

「哼，不滿LV30嗎……真是垃圾」

。

只有等級達到系統等級上限的30級玩家，才能對於身為遊戲中最終BOSS的魔王造成傷害。除此以外的玩家，甚至連與魔王對峙的資格都沒有。

。

說到底，即便有等級達到上限的玩家，也無法跟魔王進行真正的戰鬥吧――他就像是RPG遊戲的最終BOSS一般，如果不使用各類道具都無法給與一定程度上的傷害，而且其體力也實屬異於常人。

。

「明明打算要殺人，卻還要居高臨下嗎――？」

。

聽到魔王這番話，山賊們終於現身了。

它們的表情上無不是驚訝與迷惑不解。

。

「你這傢伙，到底做了什麼……魔法師嗎！？」

。

「別給我太得意忘形喔。魔法使一旦用完氣力，就只不過是條靶子！」

。

「謔，靶子呢……還真是聽到一件不錯的事情」

。

明明是四面楚歌的狀況，魔王的表情卻鬆弛著。

雖然從惡那裡聽說過「魔法」一詞，不過惡並不清楚詳細的內容，所以至今沒辦法理解魔法具體是什麼。

。

「如果你們之中有哪位是魔法師的話，請務必來一發讓我見識一下」

。

聽到這句話，山賊們轟然吵鬧著。

這已經不是被人小看的級別――而是某種以昆蟲為對像進行各種實驗般的對待方式。

首領終於按耐不住了嗎，他晃著鬍子現身了。

。

「喂，大叔。你還挺能吹的嘛」

「誰是大叔啊――！」

。

魔王情不自禁地大叫道。

然而無論怎麼看，他的外表即便委婉地說也稱不上是青年。首領那邊則是不知所措似地沉默不語，山中流動著微妙的氛圍。

。

。

。

「……然後呢，自殺點找我有什麼事？」

。

「誰是自殺點啊――！　你這混蛋，耳朵是進水了嗎！」

。

兩個大叔正醜惡地爭吵著。

山賊們也彷彿呼應著首領的憤怒，接二連三地從山上下來，包圍住兩人。

。

「魔丶魔王大人！太危險了……我們逃跑吧！」

。

惡大叫道。這幾天在惡泡過鐵桶浴之後，總覺得肌膚都變得水潤潔白。

正所謂是連水都能彈開的年齡段*――可以說是在周圍的骯臟大叔之中截然不同的存在。

「啊？魔王……？大叔，你還讓小鬼用這種誇張的名字叫你？」

。

一個山賊大笑道，周圍的男人們也拍著手笑個不停。

雖然魔王的太陽穴浮現出憤怒符號，不過山賊們會笑也不是沒道理。

從他們的角度來看，無非就是「這把年紀了，還玩魔王遊戲嗎」。可如果真要舉出一個過失――應該是這個男人就是「真正的魔王」這一點吧。

。

「很好，你們……不陪我玩個遊戲嗎――」

。

伴隨著這句話，魔王對著最初嘲笑他的男人「彈指頭」。

男人彷彿被狂風吹走一般，直接飛到後面去，無數次彈跳起來之後，難看地躺在地面上。

當他停下來的時候，身體已經斷續地痙攣著，完全昏迷了。

。

「來來來，接下來是你的回合喔――哦呀，起不來了？遺憾，是我不戰而勝了」

。

魔王面露狂妄笑容，環視著山賊們。

山賊們彷彿倒吸一口氣一般，現場一片寂靜，不久後似乎是回過神了，人人議論紛紛。見狀，這一次是魔王笑了。

。

「不愧是「自殺點」率領的山賊。連飛進的方向都是自家的守球門？」
