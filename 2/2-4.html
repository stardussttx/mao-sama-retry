2-4 第9話 異世界的城鎮

異世界的城鎮

一一一一一一一一一一一一一一一一

――聖光國　交易之城「亞侯」

。

「這個國家的命名品味糟透了」

。

「是這樣嗎？」

。

熙來攘往的人群，其熱鬧程度的確不負交易之城之名。

雖說這座城鎮並沒有所謂的特產，不過卻有一個幾條大道相互交錯的地方，籠絡來自全國各地的物品。

。

既有綁上頭巾的人，也有體格健壯的戰士，更有裸露大片肌膚，像是舞女的女人。

大路邊的占卜屋則有位可疑的老婆婆，甚至還有乘坐在蜥蜴上的商人。

。

「真虧他們敢坐在那種生物上面呢……」

。

「那種生物就叫做沙蜥蜴，是老實溫順的動物哦」

。

惡像是開心似的說著。

由於平時自己老是被嚇一跳，所以能夠教魔王一些知識令惡感到高興。

雖說如此，惡對於城鎮也幾乎是一無所知。當時從村裡出去的時候雙親還健在，不過那已經是遙遠的記憶了。

。

往大路望去，有許多家店鋪――每家店鋪的屋頂顏色不同，給人一種彷彿現在就是在過節日的感覺。聖光國的偏僻地區大多是貧瘠的土地，相反，繁華熱鬧的地方無論何時都還是會繁華熱鬧。

。

（所謂的格差社會嗎……？）

。

魔王腦海里浮現出這種曖昧不清的想法，這個國家的階級對待的確較為嚴酷，不過那不僅僅是地域問題，還涉及到個人因素。

往昔跟隨智天使與惡魔王戰斗的人，其後裔便成為了貴族，立於人民之上享受著快活的生活。

。

多虧國內是信奉智天使的宗教，以國家而言也得到一筆財富，不過內部卻紛爭不斷。既有人抗議說廢除貴族制度，也有對社會感到絕望，遂而信奉惡魔的一派。

。

這些全都是因為本地距離魔族領甚遠才會造成內部糾紛。

畢竟在魔族橫行霸道的地區，根本無法這麼「奢侈」。

。

「錢也有了，我們適當買點什麼東西吧？」

。

「那可是聖女大人的錢哦……」

。

「這種見人就殺的聖女怎麼可能會有啊……而且她被人打感覺還很高興的樣子。應該是變態吧」

。

魔王走近一家店鋪，看了看商品。

似乎是在燒烤著某種肉，有種微妙的胡椒香味。

這時，魔王的肚子響起聲音來。

至今以來，他們吃的食物頂多就是據點常備的《乾麵包》而已。雖然惡吃得蠻開心的，不過對於魔王而已，這只不過是拷問罷了。

。

「這要多少錢？」

。

「五枚銅幣三串」

「惡，哪枚是銅幣？」

。

「就是這個！浮，好厲害……連大銅幣都有。誒，這是傳聞中的銀幣！？」

。

魔王側眼看著惡驚訝的樣子，直接拿起鍛造得漂亮的硬幣交給店主。

明明是別人的財產，卻毫不猶豫地在浪費錢。

。

「惡，這次我來請客――你就盡管吃吧」

「那個，雖然已經說過了……這是聖女大人的錢吧……」

。

「這是正當防禦的結果。可以說這已經是我溫情的處置了」

。

的確，魔王所說的事情毫無疑問是正倫。無論有什麼理由，突然想要殺死人，就算反過來被殺死也是無可厚非。

不過，這次的情況――

對像是聖女這件事，在各種意義上都做過頭。

。

「如果這是工口遊戲，標題就是《被囚禁的聖女～跟你生孩子什麼的我才不要！～》直達性奴隸路線喔？考慮到這些，我都想為我自己的溫柔感動到哭」

。

「什麼是工口遊戲？」

。

「喂喂，你啊……穿著沒見過的衣服呢，應該是喲嗖的吧。這國家可是禁止奴隸的喔？就算要開玩笑也別說出這句話」

。

店主聽到魔王所說，便無法淡然地插嘴說道。

聖光國的階級對待雖說是殘酷，不過卻不認同任何奴隸制度。一旦觸犯，無論位居什麼地位都會立即受到裁決。

。

「你這說法簡直就像是喲嗖這個國家就有奴隸呢」

。

「北方和南方那邊吧……畢竟那裡比較野蠻喔」

。

（野蠻呢……雖然我覺得這個國家的頂端人士也足夠野蠻了）

。

魔王在心中暗自想著這些事，不過並沒說出口，作為代替又往嘴裡塞一支串燒。

■□■□

――旅店「古古雷」

。

「真的有好多奇怪的名字啊」

。

「是這樣嗎……？雖然我覺得蠻可愛的」

。

雖然名字莫名其妙，不過聽說這裡是城鎮亞侯裡首屈一指的最高級旅店。

是庶民難以跨越的一道門檻。

魔王走近櫃臺，向店長搭話。明明是初次來到城鎮，言行舉止卻沒有絲毫怯懦。

。

――可謂是不怕死。

已經完全是船到橋頭自然直的態度了吧。

。

「店長，拜託你來一間最好的房間」

。

店長聽到魔王這句話，一瞬間蹙起眉頭，不過看到他的打扮又重新改觀。

雖然是沒見過的衣服，不過在已經習慣經營服務業的店長眼中，一眼就看穿這些衣服是由上等的布料製作而成。

。

「如果要最好的房間，需要一枚金幣……」

。

「誒誒誒誒，不行啊！魔王大人！」

。

「魔丶魔王……？」

。

惡在為過高的價格而大叫著，不過店長對於魔王一詞露出訝異的眼神。

。

。

「是丶是這樣啊……」。

雖說店長面露微妙的表情，不過並沒有繼續再追究下去。因為從外表上看就知道是外國人士，而且根據國家也會有各種各樣的名字與習俗。

魔王斜眼看著店長這副樣子，開始和惡在《通信》舉行會議。

。

《我說過不要在別人面前叫我做魔王吧！給我叫「哥哥」》

《誒~……這叫法好像有點勉強的感覺……》

。

《那麼，不是還有其他的嗎，比如大哥丶歐尼醬丶前輩等等》

。

《爸丶爸爸之類的丶怎麼樣……？》

。

《別開玩笑！我還是生龍活虎的單身漢！》

。

在說出生龍活虎這種過時詞語的時點上，這個男人就已經沒有資格稱之為哥哥。

。

「總丶總之……是一枚金幣麼，這樣就可以了吧」

。

魔王從皮袋裡拿出金色的硬幣，並遞給店長。

惡被他堵住嘴巴拼命掙扎著，得到金錢的店長則難掩喜色地把兩人帶往房間裡。

■□■□

。

「……真是不錯的遠景呢」

。

魔王來到最高級的房間，很偉大似的眺望窗外風景。

雖然他在心裡為這房間的豪華裝潢感到驚訝，不過在惡的面前依然是裝腔作勢。

。

「好丶好厲害……簡直就像是貴族大人的宅邸一樣！」

。

「哼……遲早我要建造與我相對應的「城堡」讓你見識一下」

。

魔王雖這麼說大話，不過實際當建造完工時――會給世界帶來怎麼樣的影響呢？這個男人並沒有考慮到這些事情。

不對，應該說對於「世界」這種模糊不清的事物，這個男人並沒有考慮到負起責任吧。

。

到底會有誰在日常生活中頻頻考慮到整個世界而做出行動呢。誰都會為自己而工作，誰都會為自己而飲食，誰都為自己而休息。與此相同，這男人也只是為了自己而打算取回全部的權限吧。

。

其結果或許會造成大量的傷亡也說不定，或許也有可能招來救贖。

未來在如今這時點上，任誰也全然不知將會變成什麼樣――

。

「話說回來，魔王大人這種叫法實在不妙。就沒有其他稱呼嗎？」

。

「九丶九內大人……怎麼樣？」

。

「嗯~……有種違和感啊」

。

要求真多的男人。

這種類型的人，往往都會難以決定合適的約會地點或者餐館。

。

「……果然還是歐尼醬比較好吧？」

。

「不可能」

。

魔王還在固執已見，不過惡立馬回絕了。

惡與當初相比已經成長了。照這樣子成長下去，一定會成為不錯的制止器，制止因無知而會引發暴走的魔王。

。

「話說回來，剛才的肉非常好吃呢」

。

「嗯……？嘛，也是」

。

魔王聽到惡這句天真無邪的一番話，顯得欲言又止。

就魔王個人而言，那些肉很難說得上好吃。應該是血沒有去除幹凈吧，肉本身有些硬，還有一股腥味。

。

所以才要用胡椒之類的調味料，硬是用香味蓋過這些味道。

正因為有這些經歷，魔王才特意來到最高級的旅店。起碼一定要在傍晚吃到正經的食物。

。

「啊，不過我也非常喜歡吃《乾麵包》哦。乾麵包那微帶的甜味真令我感到幸福」

。

「幸福丶呢……」

。

惡表情鬆弛著說，魔王見狀則一瞬間露出陰霾之色。

乾麵包是遇到災害時所提供的應急食品。

在遊戲設定裡，大帝國對於應急食品的講究也堪稱是變態般，連味道上都花費不少功夫。不過在魔王的心境上實在不覺得是在吃飯。

。

「那麼，今天就去吃豪華大餐，尋求下一個「幸福」吧」

。

惡聽到這些，走到魔王身邊握住他的手。

一瞬間，魔王望向那隻令人心疼的右腳。不過他什麼都沒有說，只是抱起惡騎在自己頭上。

。

「在這之前，先解決你的衣服問題吧。雖然不知道這個世界在正式場合用餐有什麼講究，不過衣服也很重要」

。

「衣服嗎？？」

。

在魔王的意識之中，聖女的財產之類的事情已經完全拋諸腦後，徹徹底底地當做是自己錢財來對待。

他的眼神裡沒有絲毫陰霾。

若是讓自殺點看到他這副樣子，肯定會大叫說「這傢伙才是山賊吧」。

。

如此這般，「魔王大人與惡」的購物之旅就此開始――

「魔法」

魔法是操控這個世界上的元素，進而施展的力量。

最基本的四大元素便是――「土」「水」「火」「風」

以及理外的――「光」「魔」總共六種。

。

其中還有上位的元素，不過這是隻有跨越巨墻者才能驅使的領域。

天賦異稟者能夠結合元素，以此操控混合元素。

根據魔法名的文字數，可分為第一魔法～第十魔法，甚至連擁有龐大魔力的惡魔王，使用第六魔法都已經是極限。

。

順便一提，身為聖女的露娜所驅使的是――「金」

「光素」的上位是「聖素」，她所使用的「金」是從「光」獨立開來，也稱之為獨特元素。

自殺點承受到這一擊並沒有死亡，是因為他擁有特殊技能。
