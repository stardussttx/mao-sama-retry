2-3 第8話 聖女

聖女

一一一一一一一一一一一一一一一一

「過來討伐魔王，沒想到還遇見了骯臟的山賊呢」

。

聖女――露娜面露笑容。

那是十分討人喜歡的笑容，不過行為舉止卻是冷酷無情。

她所釋放的金色之光，一瞬間就把五個山賊的身體切開了。

。

「嘖……小的們，要撤退了噢！」

。

「你們是笨蛋嗎？我怎麼可能會讓你們逃跑」

。

露娜舉起那支付有羽毛翅膀的法杖，開始詠唱咒語。

轉眼間，金色之光聚集在她的身邊。

。

那是這個世界上的元素(ELEMENT)――

亦同樣是她控制自如的，最為珍貴的根源。

。

「被金色所切碎吧――《金刃/黃金切刃》(Gold Slash)！」

。

突然，無數個金色之刃從露娜的法杖釋放而出，又把六個人的身體切得是慘不忍睹。魔王看到這幅光景，額頭上也流下冷汗。

。

（真的假的，這傢伙……）

。

既有初次看見魔法，感到驚訝的緣故。

也有對那威力，感到恐怖的緣故。

然而，最令人驚訝的事情是――這樣的少女在毫不留情地殺人。

。

（要逃跑嗎……？）

。

這種想法一瞬間浮現出腦海之中，不過立馬就打消了。

少女並不是獨自前來，還有無數的士兵跟隨在她身邊。

當確認到這些，他也已經不再迷茫。立即從持有品取出據點來，把惡丟進裡面。

。

「誒丶魔……！？」

。

「蹲在最角落的地方。可以吧？」

。

意料之中，使用魔法的少女凝視著這邊。

這也是理所當然。畢竟是在什麼都沒有的地方，突然出現一座建築物。

。

「這是丶什麼……你丶是魔法師？還是說，這是某種魔法道具？」

。

我沒有回答，而是從懷裡拿出香煙點上火。

老實說，如果不這樣做，我根本不能保持冷靜。

即便受到這種攻擊，自己是否也能平安無事呢？　無論如何，以這位少女為對手就絕對不能表現出任何動搖之舉――我叮囑著自己。

雖說她怎麼看都只是個小孩子，不過這傢伙可是毫不猶豫地就把人殺死了。

。

少女很在意據點嗎，走近據點用手隨意亂摸，興味盎然似地偷窺著裡面。

她這些好奇心滿滿的舉止，擁有與年齡對應的可愛之處……

不過如今卻對這份「天真可愛」感到可怕。

。

「沒見過的材質呢……你，把這個獻給我吧。這樣的話，要我放你一馬也可以」

。

「這是在跟我談交易嗎，大小姐？你會遵守約定的保證呢？」

。

「哈——？你知道你在跟誰說話嗎？」

。

「很不巧，我還沒聽聞殿下的尊貴大名呢」

。

我邊吐著煙，邊沉著冷靜地回答道。

不過內心深處卻與外表相反，心臟劇烈跳動著。雖然全力逃跑我有自信逃得掉，不過對手的能力完全是未知數。

如果想要通過談話平安收場，這種情況這麼做可以嗎――？

。

「看來這裡還有鄉下人呢……不對，既然是骯臟的山賊那也是沒辦法的嗎」

。

我不知不覺地就被當做是山賊的同伴，對此感到有點不爽。

為什麼我非得悲哀地被當做是自殺點的手下不可啊。

這名字可是什麼時候被背後攻擊也不奇怪。

。

「聖女大人啊……這傢伙才不是我們的同伴喔？」

。

自殺點呲牙咧嘴地笑著說道。

對對對，給我繼續說。是說，想要互相傷害你們自己去幹個夠吧！

為什麼毫無關聯的我非得卷進去啊。

。

（恩？聖女……？）

。

記得好像是這個國家的偉人。惡好像是說過三位聖女什麼的吧？

。

「聖女大人啊……這個大叔可是「魔王大人」喔！浮哈哈哈哈！」

。

「你說什麼！？」

。

這大叔居然給我說些多餘的事情！

別開玩笑了，大叔！

你想死嗎，大叔！

打死你喔，大叔！

。

「自殺點，你就這麼想變成自殺點嗎？那麼，你這混蛋的人生終點就決定是在機場夠人用未熟的辛辣辣椒塞進嘴裡塞到死為止！」

。

。

――吵死人了吵死人了！

。

少女發著脾氣大叫道，金色之光從那支法杖迸發而出――！

暗中準備好「那個」的我，立馬躲在據點的陰影裡。當然，跟惡蹲著的地方完全相反。

金色之刃撞到據點，威力的銳減是一目瞭然。然而，威力沒能完全抵消，迫近到身體眼前。

。

「好……痛……」

。

這就是丶魔法――雖然我立馬用雙手護著頭部，不過雙手非常痛。

不過相對的，我已經知道了幾件事。

如果有據點存在，魔法的傷害的確會減少。相反，最重要的防具——攻擊抵禦並沒有觸發效果。

。

（這小孩子的等級是30嗎，除此之外也想不到其他的可能吧）

。

達到頂端的玩家都具有一種獨特的氣場，不過從這位少女身上卻完全感覺不出來。

話雖如此，她可能還會有更加厲害的魔法。

。

――魔法果然很危險。

。

往旁邊一看，自殺點和附近的幾個人都躺在地面上。該說是果然如此嗎――自殺點應該是把手下拿來當做盾牌吧。

雖然他的身體傷痕累累，不過還有呼吸。在附近的山賊應該是逃跑了吧，已經看不到任何人影。

。

「居然還活著……你，真的是魔王？」

。

「鬼知道。比起這件事來，突然就往這邊攻擊――不解釋一下？」

。

「解釋？聖女討伐邪惡的存在不是理所當然的事情嗎」

。

「原來如此，那我就把這句話當做是你的「遺言」記住了」

■□■□

「你說遺言？」

。

聽到這句話，露娜不禁啞然失笑。

這個男人是搞錯什麼事情了吧，從剛才開始一點態度都沒有。看不出有一丁點對待聖女的虔誠。

。

如果是山賊那倒也罷， 不過男人的打扮具有一定水準。

雖說是從未見過的衣服，不過每一件衣服都高級到可謂是不合場所。

。

行為舉止也絕不低劣。

其自然而然的動作中，具有文雅的風度。最初我還以為是沒落貴族。

不過從他對於聖女的知識欠缺這麼多來看，這也是不可能的事情。換句話說，他是外國人。

。

――說起來，我還沒自報姓名呢。

。

。

我不知道這男人究竟是何方神聖。

雖然不覺得魔王具有人類形態，不過毫無疑問也是危險的存在吧。

無論是承受住自己的魔法，又或者是突然讓建築物出現――

總而言之，這男人來歷不明。

。

男人不顧這邊自報姓名，只是露出苦澀般的表情。豈止如此，甚至還感覺到剛才絲毫未覺的沉穩怒氣。

這個男人有可能是他國派來的特工。

。

「突然就對別人釋放魔法的小鬼算得上優雅？先不說這些骯臟的大叔……你，要是把惡搞受傷了要怎麼辦？」

。

「哈？？明明就是個惡人，你在說什麼啊？」

。

護衛應該是察覺到男人釋放的氣息吧，都站在我的身邊。如果是平時我都會說礙事，不過這個男人有點深不見底。

在包圍住他的時候，對準弱點釋放魔法解決掉比較好。

。

「你們快把這呆子抓住！」

。

乘坐在馬匹上的騎兵揮舞著槍，往男人使出一記突刺。

然而，那把槍並沒有接觸到男人的身體。
