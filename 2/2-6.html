金色之露娜

一一一一一一一一一一一一一一一一

「――我找到你了！魔王！」

。

自己的叫喊聲，令店裡一陣嘩然。

這或許有點大事不妙……因為我擅自從城堡裡溜出來，也沒有通知姐姐大人關於討伐魔王的事情。

如果引起一些騷動，有可能傳到神都的耳目之中。如此一來，要怎麼敷衍了事呢……。

。

「哼，在這個國家――踢破大門是該有的禮儀嗎？」

。

聽到魔王的言辭，我已經怒不可遏。

坐在餐桌上的兩人與前陣子的打扮不盡相同，不過衣服卻是極為昂貴之物。難道說，我的錢財不見了是因為……。

。

「你……居然還把我的錢給偷走了！」

。

「在別人用餐之際引發騷動，結果還要把他人稱之為小偷嗎？哎呀，我著實懷疑這個國家的見解」

。

「別開玩笑了！那可是我積累起來的零花錢啊！」

。

「這又不是你付出勞動得到的金錢吧？國家的頂端人物都這樣，治安錯亂也並非毫無道理。稍微傾聽一下民眾的意見如何？」

。

這魔王居然打算要岔開話題――把偷人錢財一事化為烏有！

面對如此狡猾的話術，我緊握的拳頭止不住抖動。

如今魔王的視線從我身上移開，看向其他的方向。

。

「把錢還給我……！還有，去死吧！」

。

「找碴之後就是語言暴力嗎？世上還真有這麼可怕的聖女」

。

「吵死了吵死了！箍蛋！逛態！小偷！給我去死！」

。

「別這麼大呼小叫的，先坐下來如何。你應該沒有忘記這裡是用餐的場所吧？」

。

魔王此話一出，總覺得從周圍的氛圍中感覺到同意的意向。

為什麼，為什麼，為什麼我感覺這邊的形勢比較不利……。

這魔王――意圖拉攏周圍人壓制我！

。

「誰要照你的話乖乖――」

。

「你――還想要再受到體罰嗎？」

。

自己的全身，都遭到魔王那銳利眼神的威懾。

當他的右手緩慢舉起之時，屁股泛起一絲陣痛，胸口莫名其妙地小鹿亂撞。

。

（不會吧……這是什麼……）

。

臉頰猶如蘋果般火紅，呼吸也急促難熬。

結果連站都無法站好，遂而虛脫無力地癱坐在地。

。

「你能明白那是再好不過。各位，引起了騷亂實在萬分抱歉――為了表達我的歉意，這是一點心意」

。

魔王優雅地鞠躬，為每桌餐桌追加一瓶葡萄酒。這一席話讓店裡人流露出歡喜之色。

這家店裡的葡萄酒絕不是便宜貨色。

面露柔和微笑的魔王舉手回應周圍的人們，接著抱起癱坐的我回到自己的餐桌上。

。

為什麼丶事情會發展成這樣……？

回過神來，我已經被抱在魔王已經下單的料理前，晚餐的席位上。

。

「這次我請客――盡管吃個夠吧」

。

「笨蛋！不如說是我請客吧！」

。

「哼，算了――也可以這麼說」

。

魔王明朗的笑著，我不由得――覺得這表情挺孩子氣而心頭一緊。

這個變態魔王，是用了某種奇妙的魔法吧……！

■□■□

。

「那個，聖女大人，非常抱歉！不過魔王大人絕不是一位壞人！」

。

「你是笨蛋？怎麼可能會有善良的魔王！」

。

即便露娜如此大喊，依然是連續不斷地把肉和色拉都塞進嘴裡。

肚子似乎蠻饑腸轆轆的。

看著她這幅德性，「我」重新思考著――

。

（話說回來……「聖女」到底是什麼？智天使又是什麼？）

。

為什麼在這個世界上也能使用遊戲中的技能丶道具以及管理機能等等。

有太多的事情我實在不清楚。

單純只是因為――「九內伯鬥」嗎？

又或者自己是遊戲的管理者呢？我思考著接下來要詢問聖女的事情，在餐桌下取出道具。

。

我把手伸向漆黑的空間，從中取出小型的機器。

這機器叫做《隱私系統》，通過屏蔽音掩蓋住不願被他人聽見的談話與情報。

在遊戲中，這裝置只是用於妨礙其他玩家的《通信》，不過隨著網絡技術的發達，最終從遊戲中徹底廢棄了。

。

（以這個環境來看……「No19.家庭餐廳的嘈雜」比較好吧）

。

在裝置設置完畢的同時，在周圍產生一道連我們的聲音也會融入嘈雜聲的音之壁。道具順利地發揮效果，我不由得鬆弛臉頰。

總而言之，這樣就做好準備了――

。

「你是叫做露娜吧。趁這次機會我有幾件事情想問問你」

。

「什丶什麼事啊……」

。

為什麼這傢伙要對我這麼戒備啊。不對，考慮到我自己的所作所為也是理所當然的嗎？

所屬於國家中樞部的人居然在這裡，雖說是意料之外的事情，不過這也是能夠打聽情報的絕妙機會。把我心中各種疑問都試著說出來吧。

。

「露娜，你對於日本或者美國這些國家有沒有印像？」

。

「哈？什麼國家啊那是？是說，別擅自把敬稱去掉」

。

她以粉色的眼瞳瞪視著這邊，威嚇著我。

總覺得就好像是一隻毛色不錯的吉娃娃犬呢。

。

「那麼，遊戲丶大帝國丶網絡等名稱有沒有聽說過？」

。

「從剛才開始你到底在說些什麼呀？你是笨蛋嗎？現在立刻給我去死」

。

（這傢伙，毒舌到底是有嚴重啊……）

。

惡貌似也終於靜下心來，慢慢地品嘗著料理，一副幸福的表情。見狀，我的心情也清爽多了。清爽到完全不覺得眼前的吉娃娃跟她是同一種生物的程度。

。

「淫亂粉毛――智天使究竟是什麼？」

。

「誰丶誰誰誰淫亂了啊！你啊，到底把我當做是什麼人了！？」

。

「吉娃娃，快點告訴我。我跟你不一樣，時間寶貴得很」

。

「我丶我的時間也很寶貴呀！而且，吉娃娃是什麼啊！」

。

露娜邊用餐，邊一點一點地告訴我。

關於智天使的事情，跟從惡聽來的情報沒兩樣。

只不過似乎還有其他兩位天使。

智天使封印住惡魔王之後便逝世了，雖說還有座天使以及熾天使，不過已然許久沒嶄露頭角。

。

（三位天使嗎……原來如此，完全搞不懂）

。

應該說歐美這方面的領域對於疏於研究的我而言，實在是一問三不知。

借這次機會，也問問看那座石像的事情吧？如果是這傢伙，有可能知道些什麼。

。

「擺設在願望祠堂的「石像」――你知道嗎？」

。

面對這一詢問，露娜的表情微微一蹙。

這傢伙果然知道些什麼。

。

「那裡面是行為不端的座天使大人，……所以遲早絕對會降下神罰」

。

「那就是座天使……？」

。

的確，我記得它好像是說過――原本是清廉之身來著。

話雖如此，我也無能為力。總之就是墮落的天使之類的吧？

如果是遊戲，感覺標題應該就是「凌辱座天使　～被肉棒所玷汙～」。

。

「你們所信仰的智天使教義是怎麼樣子的？」

。

「啊啦，明明是魔王卻想要打聽智天使大人的教義嗎？」

。

露娜流露出一點笑容，得意洋洋地說道。

雖說如此，這教義的內容與其說是宗教，不如說像是某種自我研究。通過努力提高自身的實力，戰勝困難。天使會對努力之人露出微笑，給予強大的力量及加護――大致上來講就是這些。

嘛，雖說內容倒也不是相當奇怪……。

。

「簡而言之，意思就是說無論是人亦或是地區，有階級之差是理所當然嗎」

。

通過努力亦將獲得改變。

那麼努力之者會成就財富，懶惰之人就會落得一貧如洗，差不多就是這個意思。不過看到惡，我感覺又不能單純地斷定就是如此。

。

「如果努力的差距也稱之為階級的差距的話，也算是吧。畢竟我就是努力過來的」

。

「我丶我知道這件事……聖女大人是從孤兒院中，基於才能選肝而出……」

。

「……是很久以前的事情了啦」

。

我對於惡在途中插嘴的那番話，稍微湧起點興趣。

如果這件事是真實的，那麼這傢伙的確是獨力爬上來的。正因為如此，才會傾心於這教義也說不定。

不對，她是幸運地體驗到了「實感」――以為這教義是正確的。

。

「原來如此――所以野心和功名心才這麼強」

。

「什麼嘛，這種意味深長的說法」

。

粉毛睥睨地看向這邊。

假如說這傢伙想要獨自討伐名為魔王的存在。不過不在其他兩位聖女的陪同下，而是僅帶著二十多名輕步兵。

這就是想要名留青史的人，所特有的――功名心與搶先立功吧。

。

「據我推測――「其他兩人」應該很礙事吧」

。

「什……你丶你這麼說有什麼根據呀」

。

「――別隱瞞了，我都懂」

。

雖然我說得自信滿滿，可其實並沒有任何意義。

我只不過是盡可能以「大人般」的語氣，適當地胡說八道而已。與此相對，對手只會破綻百出而已。

。

這位聖女，應該沒有正經地與他人接觸的經驗吧？

嘛，從她這偉大的態度與毒舌來看，連朋友都沒有吧。

。

「――――「剩女」」

。

「嗚……！突丶突然說些什麼啊……」

。

「不，那啥。我只是覺得是時候從店裡離開了吧……」

(颯:是時候=ぼっちぼち，既關西腔的ぼちぼち，與上文「剩女」有一部分同音)

。

「說丶說說說的也是……我也是很忙的！」

。

聖女的眼神遊移不定，慌張地從椅子上站起來。

多麼簡單易懂的傢伙。

。

「我就住在一家名叫古古雷的旅店裡――如果有什麼事情盡管找我」

。

「我怎麼可能有事找你啊！箍蛋！」

。

聖女――露娜飛奔般從店裡奪門而出，周遭又回歸寂靜。

我們也差不多該回到房間休息了吧。

我買完單，帶著惡走出店。雖說本應是優雅的晚餐變成意料之外的展開，不過收獲也不少。

。

「……魔王大人既有溫柔的時候，也有欺負人的時候呢」

。

「我無論何時都是紳士。只不過――我會根據選擇的對像罷了」

。

「聖女大人可是非常偉大的哦？魔王大人――哇哇！」

。

我說完這些，就以公主抱的形式抱著惡，回到旅店。

畢竟又不能以騎脖子丶背著的形式帶她回家。雖說從周圍看過來的視線有點刺痛，不過旅行發生的羞恥事即便不在意也沒事。

。

「我丶我還是收回那句話……果然魔王大人很溫柔……」

。

「嗯？今天應該能久違地在舒適的床鋪上睡覺了吧――」

。

兩人面露笑容，度過了安穩的一夜――並沒有發生。

因為一回到房間裡，露娜就拼死拼活地敲著房門。一打開房門，便看到滿眼淚水的露娜站在那邊。

。

「笨蛋！因為你的錯，害我連訂房的錢都沒有了！」

。雖然對於她這幅樣子感到有些同情，不過讓這麼吵鬧的傢伙住在這裡也請恕我敬謝不敏。或許是我偏見，總覺得這傢伙連夢話都會吵死人。

。

「給我去露營吧。順便患上感冒，還有發燒」

。

「別開玩笑！不就是因為你害的，我才會變成這副樣子嗎！」

。

「魔丶魔王大人……讓聖女大人患上感冒就大事不妙了」

。

「真沒辦法……給你，錢還給你了，自己找個旅館住幹」

。

再被她糾纏不清也麻煩得要死，所以我把裝進金錢的皮袋還回去。

反正也該是時候獨力尋找賺錢的方法了。這剛剛好是一次不錯的機會。

。

（嗯……？）

。

明明我已經把錢還回去了，不過露娜在房門前一動不動的。

豈止如此，還越來越涕淚俱下。

怎麼回事？皮袋裡還剩下很多錢才對……。

。

「真丶真真是沒辦法，我也住在這裡吧！」

。

「哈？？？？你啊，腦袋沒問題嗎？」

。

這傢伙究竟在說些什麼？

我已經完全搞不懂了啊……。

。

「這丶這裡不是用我的錢訂好的房間嗎！既然如此，我住在這裡不就是理所當然的事情！」

。

「……跟剩女糾纏不清就會變成這樣嗎」

。

「誰是剩女啊！給我讓開，我要睡最好的床鋪！」

。

如此這般，「聖女」與「魔王」與「惡」組成了一個奇妙的組合。

夜晚的吵鬧仍在持續不停。
