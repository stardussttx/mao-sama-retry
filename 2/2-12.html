前往神都

一一一一一一一一一一一一一一一一

「雪風，準備好了嗎？」

(颯:名字跟某驅逐艦一模一樣，外加下文提到名字，故以雪風為名，還有作者你命名方式不按套路來的)

。

女人如此說道，並拔出掛在背上的劍。

那是一把半吊子的男人難以舉起的大劍。

她的劍上鑲嵌了數顆魔石，這種利用方式並不是為了減輕重量，反而像是在增加重量一般――既為了提高攻擊力。

。

她身披覆蓋全身的茶色斗篷，從中窺見得到的膚色正是與炎熱國家相襯的褐色。

或許是想要行動更加方便吧，她留著一頭短發，一見之下有著活潑開朗的印像。

無論是發色亦或是眼瞳――都是燃燒般的紅色。

。

她的名字是蜜康。

雖有著銳利無比的眼神，不過其容貌應該足以列入美女的範疇之內吧。

在聖光國是無人不知無人不曉的著名冒險者。

。

「……隨時都可以上。不如說，可以上過頭了」

。

被叫做雪風的楚楚可憐的魔法師，答復了一句莫名其妙的話。

發色與眼瞳正如其名，都是清澈透明般的白色。

她身穿黑色斗篷，頭戴一般魔法師愛用的三角帽子，不過眼神卻有種睡眼惺忪的感覺。

。

「你啊，算我拜託你了給我認真點幹活啊……」

。

「……我一直都是在認真地幹」

。

「我總覺得你話裡的意思有微妙的不同」

。

「沒有這種事。這是聽者的問題。欲求不滿」

。

雪風的聲音雖說極為平穩，有種想睡的感覺，不過卻可愛得猶如鈴音一般。

她在聖光國具有的人氣非同一般，幾乎稱得上是偶像。

年齡也僅有年幼的15歲，其楚楚可憐的容貌深深地抓住男人們的心靈。

。

――――然而，是偽娘。

。

「雖然聽說這附近一帶有沙狼(Sand Wolf)「群居」……」

。

「……太熱了，胸部附近好悶熱」

。

「你根本就沒有胸部吧……啊浮浮，完全聊不上來！」

。

「蜜康有眼無珠。我的胸部只有心靈美麗的人才能看見」

。

蜜康饒了饒頭，無視了這句話。

兩人平時之間的對話都是這幅樣子。

。

順便一提，沙狼雖說也是獨來獨往的魔物，不過一旦群聚，其數量便會爆發性增加，極具威脅性。

若吃光食物，它們會殘暴到滿不在乎地襲擊城墻守護下的城鎮，若是毫無防備的村莊遭受群體襲擊，一瞬間便會毀之一旦。

。

正因為如此，群聚的沙狼非得在「短時間」內處理不可。

當兩人終於抵達別人看見狼群的地點之後，便目瞪口呆。

在那裡的並不是一群沙狼――而是「一大群」沙狼。

■□■□

。

「不行不行！這種數量根本不可能解決！」

。

「……數量這麼多，根本承受不住。我的身體只有一個。洞也只有一個」

。

「都這種時候了你還在說什麼啦！？」

。

「我」看著吵鬧的兩人，不禁笑了起來。

然而，我越看見追逐在她們身後的狼群數量，臉色也越加蒼白。

。

（那是什麼鬼！數量未免也太多了吧！）

。

直至剛才都不屑一顧的表情上也流下了冷汗。

拿在手裡的香煙也掉落在地，甚至身體都顫抖起來。

雖說自己受到強大的防護罩所保護，然而這種數量無不令人感覺到原始的恐怖。

。

（到底是有幾百匹狼啊……是說，這不就是「火車狀態」嗎！）

。

在MMO是時而可見的一種情況。

指的是一種從敵人手上逃跑，遭到多數敵人的追逐，最終連毫無關聯的他人都捲入其中的狀況。

無論是否有意無意，一旦發生就會慘遭不幸。

。

「那邊的人快點逃！」

。

「……那邊的時髦大叔，接下來就拜託了。遺骨我不會撿的」

。

（少給我開玩笑了，這些傢伙！給我負起責任讓它們吃掉吧！）

。

我情不自禁地想要大喊，不過在脫口之前總算是忍住了。

如果自己在這時候逃跑了，連惡都會有危險。露娜就……嘛，這傢伙怎麼樣都好。

。

（沒丶沒事的……我的話做得到。因為我是要幹就幹得出來的人！）

。

我拼命地施加自我暗示，振奮著自己。

我有防護罩所以沒問題的沒問題的沒問題的……應該丶沒問題的吧？

可是在遊戲中當然只能對人類發動效果……。

不，真的……算我求你了喔！？

。

我大吸了一口氣，令情緒冷靜下來。回想起記憶中「九內伯鬥」的角色，並完美地模仿著他開口說道。

。

「愚蠢至極――以這種野獸為對手，究竟有何必要逃跑？」

。

我看向逃跑而來的兩人，彷彿在說給自己聽一般放出狠話。

當我說出口，即便自知是謊言，勇氣也逐漸湧現而出，人類的身體真是不可思議的東西。

。

「惡丶露娜你們兩人都呆在馬車裡別動――」

。

我佇立在大批沙狼前來的面前，做好覺悟。

最先一匹狼飛奔襲擊過來時，我不禁閉上眼睛，緊咬牙關――然而，在獠牙碰到我之前便響起一陣電子聲，狼就在眼前彈飛了。

。

喜悅之情漸漸滲透於全身之中――

。

（贏了……第一關，結束！）

。

我不禁想要擺出勝利的姿勢來，不過卻絲毫沒有表露在外，僅僅是以堂堂正正地態度點著煙。呆在馬車裡的兩人或許有在看我也說不定，我想盡可能地裝帥扮酷――這也是男人悲哀的本性。

。

「居然對吾展露獠牙――縱使是野獸也無法饒恕」

。

這也是不錯的機會，就拿這些傢伙來作為技能使用的練習對像吧。

畢竟我幾乎都沒經歷過正經的戰鬥。

。

「來吧，盡管在吾的手腕中絕息便可――――《FIRST SKILL:突擊》」

。

我以無法目視的速度投擲出必殺的小刀，直接貫穿十多匹狼。突然，身體也往前方高速地――飛翔。

。

（好厲害……！）

。

景色猶如視頻快進般倏然而逝，彷彿在靜止不動的世界中，感覺只有自己能夠行動。

我情緒突然高漲，不禁起了興頭。

。

「下賤之眼豈有資格目睹吾的尊顏――――！《SECOND SKILL:目封》」

。

右手的動作猶如遊隼，刻畫出橫向一直線。

呆在周圍的狼群眼睛陸續慘遭切傷，我再以左手撒發出微小顆粒。

這是擁有屏蔽幾十秒視野的沙型大帝國制道具，狼群發出混亂般的嘶吼聲，遂而背向這邊逃跑了。

。

――你們不知道嗎？

。

「無人能從」魔王」手上逃過一劫――――《THIRD SKILL:迅雷》」

。

右手拿著小刀施展出亂砍，每一次都把數不清的狼給砍成兩半。最後卯足全力投擲出小刀時――刺中狼群中央處的小刀產生出沖擊波，同時還颳起一股狂風肆虐般的巨大火焰。

。

無以計數的狼捲入火焰之中，一瞬間便化為灰炭。

回過神來，將近三百匹狼群輕而易舉地就消失了。殘餘的五匹狼或許是想要掩飾膽怯吧，拼命地發出嘶吼聲。

。

「終究只不過野獸――我連一滴汗水都不會流」

。

我叼著煙頭悠然自得地吐了口煙，很偉大似地放話說。

雖說從中途開始就得意忘形地玩起了魔王角色扮演，不過昨天因暴走族混蛋受到的心理壓力也完全消除，現在神清氣爽。

。

「深知已身的弱小之處了吧――？那麼，離開吧」

。

這句話令狼以驚慌失措的動作逃跑了。

我現在非常心曠神怡，其實也沒有必要再追殺吧。

若虐殺過頭，被人說是虐待動物我也不怎麼願意。

。

■□■□

。

「那就是丶傳聞中的魔王嗎……！」

。

蜜康對於如今在眼前展開的單方面虐殺，感到全身都止不住顫抖。

對――那並不是戰鬥，而是在虐殺。

不知為何，沙狼的尖牙彷彿被墻壁所阻攔一般，甚至都沒有碰到魔王的身軀。著實頗為異常。

重點是本人都自稱「魔王」不是嗎！

。

「叔叔大人，穩重老成……屁股都熱起來了」

。

雪風對於如今在眼前展開的單方面虐殺，感到屁股都熱起來了。

對――那並不是戰鬥，而是在凌辱。

雖說沙狼的尖牙彷彿被一座無法看見的墻壁所阻攔而無法碰觸到大叔，不過如果是我，無論是什麼墻壁都能跨越而過讓你見識一下。

即便是，性別的墻壁。

。

「那是危險的存在呢……！」

。

「的確，很危險。彷彿連我的貞操都會奪走一樣」

。

「我得前往神都通知大家！」

。

「我得前往神都美化全身」

。

兩人的對話微妙地不一致，不過那也是常有的事。

另一方面，在馬車觀看戰斗的惡也發出興奮的聲音。

。

「魔王大人好厲害！總覺得胸口小鹿亂撞的……」

。

惡把手置於胸前，恍惚地閉上眼睛。

他是拯救了縱使一如往常生存下去，也只會終末一生的自己。

他也是一直在保護自己的人。

無論這個人是多麼邪惡的存在，自己也已然無法從這個人身邊離開。

這才真正算是即便以「世界」為敵――

。

「――你們兩人有老老實實地呆著嗎？特別是露娜」

。

魔王回到馬車裡，悠閑地坐了下來。

露娜已經換好了衣服，其外貌著實楚楚可憐。單憑外觀可說是大小姐。

應該是很在意裙子吧，露娜紅著臉發出高亢的聲音。

。

「這丶這不是挺努力的嘛……就照這樣子保護我哦」

。

「雖說我並不是在保護你……」

。

「比丶比起這件事來，你應該有要說的事情吧！　在這種時候！」

。

這番話令魔王在心中嘆氣。

他姑且算是一位大人。

十分清楚對於穿上新衣服的女孩子應該要說什麼話才好。

。

「嘛，我覺得很適合你喔。不，老實說跟你非常相襯」

。

「這丶這是當然的吧……因為像我這種淑女無論穿什麼衣服都合適！」

。

露娜挺著飛機場，高興似地說著。

然而，魔王的耳朵並沒有聽漏她那句「誒嘿嘿……我做到了」細微的嘀咕聲。他的聽力異常的優秀，根本不可能聽漏細小的聲音。

看來這男人是很難成為聽漏系的主人公了。

(颯:就是那種吧，後宮女主們老是小聲嘀咕，男主跟耳聾一樣次次沒聽見也不在意)

。

■□■□

。

「啊~，難得找來的汪醬都消失掉了」

。

一位少女看見行駛的馬車，小聲地嘟嚷道。

她的頭上長有獸耳，手上也戴著一套可謂是可愛老虎的腳底般的手套。

獸人――而且還是等級相當高的存在。

。

「不過，我找到了比起汪醬還要更厲害的東西了呢~」。

少女臉上也塗上臨摹著老虎般的顏料，雖說臉蛋極為可愛，不過其身體能力完全就是怪物。

如今在她身後被稱之為「盾構獅虎」的怪物是沙狼所遠遠無法比擬的，它正低下頭臣服於少女。

僅僅就它一頭狂暴起來，聖光國就非得令騎士團出動不可吧。

。

「那個人類真的是魔王嗎？既然如此，應該就是魔族的王吧？那為什麼會和人類在一起呢？又為什麼跟聖女在一起呢？」

。

她把思考的事情都脫口而出，這是與她外在的稚嫩極為相襯的行為。

然而對於她而言――不，對於獸人族而言是非常重要的事情。因為獸人族與魔族領的領土相接，戰爭已經持續了一段極為漫長的歲月。

。

「無論怎麼樣，讓他們前往魔族領就糟糕了呢」

。

少女的思考方式凌磨兩可。

雖然她擅長運動身體，不過就有如外表所示，不善於考慮事情。

。

「可如果是那個人類，或許可以打贏過「那些傢伙」也說不定……」

。

少女小聲地嘀咕著，坐到背後待命的怪物背上。

即便不發出任何指示，怪物也奔跑起來，一瞬間它的身姿便變成顆粒般的大小。

。

時間或長或短，一行人終於臨近神都。

。

在那裡又會有嶄新的邂逅與風波在等候著吧。

。

魔王與悪與聖女。

。

僅有傳聞在持續擴大的銀之龍人，以及惡魔信奉者。

。

魔王引發的騷動，終於發展到足以動搖一國的事態……

。

然而――――那是在不遠的將來所發生的事情。

。
